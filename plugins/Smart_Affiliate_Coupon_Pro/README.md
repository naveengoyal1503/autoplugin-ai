## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Configure in **Settings > SAC Pro** (API key optional for basic use).
4. Add shortcode to any post/page: `[sac_coupon]`.

## Setup

1. Go to **Settings > SAC Pro**.
2. Enter your affiliate network API key (optional).
3. Set default discount percentage (e.g., 20%).
4. Use shortcode with attributes:
   - `affiliate`: Network name.
   - `product`: Product name.
   - `link`: Your affiliate URL.
   - `code`: Custom code (auto-generates if blank).

Example: `[sac_coupon product="WordPress Hosting" link="https://your-affiliate-link.com"]`.
