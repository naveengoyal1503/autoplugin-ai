# AffiliateGeoTargeter

AffiliateGeoTargeter is a WordPress plugin that automatically detects your visitor's country and displays region-specific affiliate offers to increase conversions and revenue.

## Features

- Automatic visitor geolocation detection
- Dynamic display of affiliate offers based on country
- Default global offers if no country-specific match found
- Simple shortcode `[affiliate_offers]` to display offers anywhere
- Lightweight and fast, minimal styling included

## Installation

1. Upload the `affiliategeotargeter.php` file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Insert the shortcode `[affiliate_offers]` into any page, post, or widget where you want affiliate offers to appear.

## Setup

- The plugin contains a predefined array of offers keyed by country codes.
- Customize affiliate links and texts by editing the plugin file directly or extend it for more advanced management.

## Usage

- Use the shortcode `[affiliate_offers]` in your content.
- Visitors will see affiliate offers tailored to their country.
- If visitor's country is not listed, a default global offer is shown.
