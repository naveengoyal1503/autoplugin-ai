# Smart Affiliate Coupon Hub

## Description
Smart Affiliate Coupon Hub helps bloggers, affiliate marketers, and e-commerce websites easily create, manage, and display exclusive affiliate coupons and deals in a user-friendly way to boost conversions and revenue.

## Features
- Add unlimited affiliate coupons with codes, descriptions, affiliate URLs, and optional expiry dates.
- Admin interface with list management: add, delete coupons.
- Frontend shortcode `[sach_coupons]` to display active coupons anywhere on your site.
- Coupons auto-hide once expired.
- Simple, clean design with basic styling built-in.
- No configuration required; ready to use after activation.

## Installation
1. Upload the `smart-affiliate-coupon-hub.php` file to your WP plugins directory (`/wp-content/plugins/`).
2. Activate the plugin through the WordPress admin dashboard under Plugins.
3. Access "Affiliate Coupons" in the admin menu to start adding coupons.

## Setup
- Go to the "Affiliate Coupons" page in the WordPress admin menu.
- Use the form to add coupons by entering a coupon code, description, affiliate URL, and an optional expiry date.
- Save each coupon.

## Usage
- Insert shortcode `[sach_coupons]` into posts, pages, or widgets where you want coupons to appear.
- Visitors will see a list of current, active affiliate coupons with clickable links that open in new tabs.
