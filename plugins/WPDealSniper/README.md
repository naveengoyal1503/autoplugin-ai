# WPDealSniper

**Latest version:** 1.0

## Description
WPDealSniper is a WordPress plugin designed to help site owners effortlessly monetize their websites by automatically displaying the best coupon deals and discounts from various affiliate programs. It detects users' geographic location to show relevant offers, boosting affiliate conversions and generating revenue.

## Features
- Automatic aggregation of affiliate coupons and deals (mock data provided; expandable to real API integrations)
- Geo-targeted deals display based on visitor location
- Simple shortcode `[wpdealsniper_deals]` to embed deals anywhere
- Lightweight and fast with AJAX-powered dynamic loading
- Supports categories for filtering deals
- Freemium-ready for premium extensions (geo-targeting enhancements, multiple affiliate integrations, branding)

## Installation
1. Upload the `wpdealsniper.php` file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place the shortcode `[wpdealsniper_deals]` in your posts, pages, or widgets where you want to display deals.

## Setup
- Use the shortcode with optional category parameter: `[wpdealsniper_deals category="fashion"]`
- Defaults to showing all deals if no category is specified.
- Geo-location uses basic IP methods and Cloudflare headers if available.

## Usage
- Add the shortcode to your content.
- Site visitors will see fresh, geo-targeted affiliate deals.
- Track clicks and sales through your affiliate links to earn commissions.
