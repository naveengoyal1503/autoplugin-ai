## Installation

1. Download and upload the plugin ZIP to your WordPress admin > Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Access settings via **AI Affiliate Booster** in the admin menu.[4]

## Setup

1. Go to **AI Affiliate Booster** dashboard.
2. Enter keywords/topics and your affiliate links.
3. Generate content (Pro: unlimited; Free: limited teasers).
4. Copy shortcode: `[ai_affiliate_booster keyword="best AI tool" link="your-affiliate-url"]`
5. Paste into posts/pages for instant monetized content.
6. **Pro Upgrade**: Remove limits, add coupon tracking, and analytics dashboard.
