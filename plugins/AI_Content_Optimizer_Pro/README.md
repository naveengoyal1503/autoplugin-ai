# AI Content Optimizer Pro

## Description
AI-powered WordPress plugin that scans your posts for **SEO score**, readability, and engagement. Free version provides basic analysis; premium unlocks AI rewriting and advanced suggestions.

## Features
- **Free:** Instant SEO scoring based on word count, sentence length, and links.
- **Premium ($9/mo):** AI-generated keyword suggestions, content rewriting, unlimited analyses, priority support.
- Works on Posts & Pages.
- Lightweight, no bloat.

## Installation
1. Upload `ai-content-optimizer.php` to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Edit any post/page; find the **AI Content Optimizer** metabox.

## Setup
- Free: Click **Analyze Content** for instant score.
- Premium: [Purchase here](https://example.com/premium), enter license key in **Settings > AI Optimizer** (future update).

## Usage
1. Open Post/Page editor.
2. Click **Analyze Content** in the sidebar.
3. View **SEO Score** (0-100).
4. Premium users get AI tips to optimize.
