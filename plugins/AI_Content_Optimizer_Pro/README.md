# AI Content Optimizer Pro

## Features

- **AI-Powered Analysis**: Automatically scores your content for SEO, readability, and engagement.
- **Smart Suggestions**: Get actionable tips to improve headlines, keywords, and structure.
- **One-Click Optimization**: Pro version auto-rewrites content using advanced AI (OpenAI integration).
- **Freemium Model**: Basic analysis free; Pro unlocks unlimited scans, bulk processing, and API access.
- **Lightweight & Compatible**: Works with Gutenberg, Elementor, Rank Math, and Yoast.

**Pro Features** (Available via upsell):
- Real-time AI rewriting
- Keyword research integration
- Bulk post optimization
- Export reports
- Priority support

## Installation

1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Edit any post/page to see the AI sidebar meta box.

## Setup

1. Go to **Plugins > AI Optimizer Pro** to upgrade or enter Pro API key (if purchased).
2. No further setup required for free version.

## Usage

1. Edit a post or page.
2. In the sidebar, click **Analyze Content**.
3. Review your **score** (0-100) and suggestions.
4. **Pro users**: Click **Optimize with AI** for auto-improvements.
5. Save your post to apply changes.
