# AI Content Optimizer Pro

## Description
AI-powered WordPress plugin that analyzes your post content for **SEO**, **readability**, and **engagement**. Free version provides basic scoring; premium unlocks advanced AI suggestions and auto-optimization.

## Features
- **Free**: Basic content score (word count, readability check) on post edit screen.
- **Premium** ($9.99/month):
  - AI-driven SEO suggestions.
  - Auto-optimize content with one click.
  - Unlimited analyses and bulk processing.
  - Detailed reports on keyword density, sentiment analysis.
- Feature gating with upsell prompts.
- License key activation for premium.

## Installation
1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' screen in WordPress admin.
3. Go to **Settings > AI Content Optimizer** to enter premium key (optional).

## Setup
1. After activation, edit any post to see the **AI Content Optimizer** meta box.
2. For premium: Purchase at [example.com/premium](https://example.com/premium), get key, and enter in settings.

## Usage
1. Edit a post.
2. Click **Analyze Content** in the sidebar meta box.
3. View score and free suggestions.
4. Premium users: Click **Auto-Optimize** for instant improvements.
5. Save post to persist analysis.
