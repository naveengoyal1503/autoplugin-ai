# AI Content Optimizer Pro

## Features

- **Free Version**: Basic SEO analysis, keyword suggestions, readability score (5 scans/month).
- **Premium Version** ($4.99/month): Unlimited scans, AI-powered content rewrites, advanced SEO metrics, priority support.[1][2]
- Real-time post editor integration.
- Simple one-click analysis.

## Installation

1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Edit any post to see the **AI Content Optimizer** meta box.

## Setup

1. Go to **Settings > AI Optimizer** for usage stats.
2. **Premium**: Enter license key or subscribe at [example.com/premium](https://example.com/premium).

## Usage

1. Edit a post/page.
2. Click **Analyze Content** in the sidebar meta box.
3. Review suggestions and optimize your content.
4. Free users: Limited to 5 scans/month. Upgrade for unlimited access.

## Support

- Free: WordPress.org forums.
- Premium: Dedicated email support.
