# AI Content Optimizer Pro

## Description
AI Content Optimizer Pro uses simulated AI to analyze your WordPress posts for **SEO score**, readability, and optimization suggestions. Free version provides basic per-post analysis; premium unlocks bulk optimization, advanced AI integrations, and priority support.

## Features
- **Real-time SEO scoring** on post edit screen.
- Basic keyword and readability analysis (free).
- Premium: Bulk optimize entire site, OpenAI integration, export reports.
- Freemium model with easy upgrade.
- Lightweight, no performance impact.

## Installation
1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Edit any post to see the optimizer metabox.

## Setup
1. Go to **Settings > AI Optimizer**.
2. Free: Use immediately.
3. Premium: Enter license key (demo: "premium123") to activate advanced features.

## Usage
1. Edit a post.
2. Click **Analyze Now** in the sidebar metabox.
3. View **SEO Score** and suggestions.
4. Premium users: Access bulk tools via settings page.
