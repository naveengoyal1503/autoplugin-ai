# AI Content Optimizer Pro

## Description
**AI Content Optimizer Pro** is a powerful WordPress plugin that uses simulated AI to analyze and optimize your post and page content for better SEO performance. The free version offers basic readability and keyword checks, while premium features provide advanced AI-driven rewrites, unlimited scans, and integrations.

## Features
- **Basic SEO Analysis**: Word count, keyword detection, and SEO score preview in post editor.
- **Auto-Optimization Toggle**: Enable/disable per post.
- **Content Filtering**: Automatically optimizes published content (premium).
- **Settings Dashboard**: Global enable/disable and premium API key setup.
- **Freemium Upsell**: Seamless upgrade prompts for premium features.

**Premium Features** (requires subscription):
- AI-powered content rewrites.
- Unlimited optimizations.
- Advanced keyword suggestions.
- Priority support.

## Installation
1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit **Settings > AI Optimizer** to configure.

## Setup
1. Go to **Settings > AI Optimizer**.
2. Toggle 'Enable Plugin'.
3. For premium: Enter your API key (get at example.com/premium).
4. In post/page editor, use the **AI Content Optimizer** meta box to enable per post.

## Usage
- **Free**: Edit posts; see SEO score in sidebar. Publish to apply basic filters.
- **Premium**: Enter API key; get AI rewrites on enabled posts.
- View optimized content on frontend.
