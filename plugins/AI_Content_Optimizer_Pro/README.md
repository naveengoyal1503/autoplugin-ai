# AI Content Optimizer Pro

## Description
**AI Content Optimizer Pro** is a powerful WordPress plugin that analyzes and optimizes your post and page content for better SEO performance. The free version provides basic SEO scoring based on word count, headings, and title presence. Premium unlocks AI-powered content rewriting, advanced keyword suggestions, and bulk optimization.

## Features
- **Basic SEO Score**: Real-time analysis of content quality (free).
- **Meta Box Integration**: Displays score in post editor sidebar (free).
- **Shortcode Support**: Embed SEO score with `[ai_optimize_score]` (free).
- **Premium AI Rewriting**: Automatically improves content readability and SEO (premium).
- **Keyword Suggestions**: AI-generated keywords for better ranking (premium).
- **Bulk Optimization**: Optimize multiple posts at once (premium).
- **Freemium Model**: Seamless upgrade path with license key activation.

## Installation
1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The SEO score will appear in post/page editors automatically.

## Setup
1. Go to **Settings > AI Optimizer** to enter your premium license key (if purchased).
2. Premium key unlocks advanced features instantly.
3. Use the `[ai_optimize_score]` shortcode in posts/pages to display live scores.
