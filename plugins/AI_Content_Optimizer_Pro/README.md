# AI Content Optimizer Pro

## Description
AI Content Optimizer Pro uses AI to analyze and improve your WordPress posts and pages for better **SEO**, readability, and engagement. Boost your site's traffic with automated suggestions.

## Features
- **Free Version**:
  - Basic SEO score calculation.
  - Content analysis (5 free uses/day).
  - Keyword and readability suggestions.
- **Pro Version** ($9/month or $99/year):
  - Unlimited analyses.
  - AI-powered rewrite suggestions.
  - Advanced keyword research.
  - Priority support and updates.
  - Export reports.

## Installation
1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit **Settings > AI Content Optimizer** to configure.

## Setup
1. Enter your API key (Pro users: get from account dashboard).
2. For Pro: Click 'Activate Pro' and enter license key.
3. Edit any post/page; find the **Optimize with AI** meta box in the sidebar.

## Usage
1. In post/page editor, click **Analyze Content**.
2. Review SEO score and suggestions.
3. Apply changes manually (Pro: one-click AI rewrites).
4. Save post; score displays on frontend.
