# AI Content Optimizer Pro

## Description
AI Content Optimizer Pro is a powerful **freemium WordPress plugin** that uses AI to analyze your post content for **SEO optimization**. Get instant scores, keyword suggestions, and improvement tips. Upgrade to premium for unlimited scans and advanced features.

## Features
- **Free Version**:
  - AI-powered SEO score for posts
  - Basic keyword suggestions
  - Content improvement tips
  - 5 scans per day
- **Premium Version** ($9/mo or $99/yr):
  - Unlimited scans
  - Advanced AI rewrite suggestions
  - Auto-apply optimizations
  - API integrations (OpenAI, etc.)
  - Priority email support
  - Export reports

## Installation
1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit any post edit screen or use the tool via **Posts > AI Optimizer**

## Setup
1. Go to **Posts > AI Optimizer** in your admin dashboard
2. Click **Optimize This Post** on any post edit screen
3. Review AI suggestions and apply manually (premium: auto-apply)

**Premium Activation**:
1. Purchase at [example.com/premium](https://example.com/premium)
2. Enter license key in plugin settings (demo key: `demo-premium-key` for testing)

## Usage
- Edit any post
- Click **Optimize This Post with AI** button
- View SEO score, keywords, and tips
- Implement suggestions for better rankings
