# AI Content Optimizer Pro

## Description
**AI Content Optimizer Pro** is a powerful WordPress plugin that automatically analyzes and optimizes your content for **SEO**, **readability**, and **engagement** using advanced AI techniques. The free version provides basic checks, while Pro unlocks full AI-powered optimizations.

## Features
- **Automatic content scanning** on post save and display
- **SEO keyword suggestions** and density optimization
- **Readability score** improvements (Flesch-Kincaid, sentence length)
- **Pro only**: Bulk optimization, AI rewriting, analytics dashboard
- **Meta box integration** for easy per-post optimization
- **Freemium upsell** with seamless upgrade prompts

## Installation
1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access settings under **Settings > AI Optimizer**

## Setup
1. Go to **Settings > AI Optimizer**

## Usage
- **On edit screens**: Use the side meta box to input content and click **Optimize with AI** (Pro only)
- **Auto-optimize**: Published content is optimized on frontend load (Pro)
- **Dashboard**: Monitor optimization status via post meta
- **Upgrade**: Visit the Pro link in notices/settings for subscription ($9/month)
