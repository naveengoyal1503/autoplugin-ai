# AI Content Optimizer Pro

## Features

- **AI Content Scoring**: Instant score based on readability, length, and keywords (0-100%).
- **Smart Suggestions**: Actionable tips for SEO, engagement, and readability.
- **Keyword Extraction**: Identifies top keywords from your content.
- **Sidebar Integration**: Works seamlessly in post editor.
- **Pro Features** (paid upgrade): Auto-rewrite suggestions, bulk optimization, export reports, A/B testing integration.

## Installation

1. Upload the `ai-content-optimizer-pro` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Edit any post to see the AI meta box on the side.

## Setup

- No configuration needed for basic use.
- **Pro Activation**: Purchase license ($29/year), enter key in Settings > AI Optimizer > License.

## Usage

1. Edit or create a post/page.
2. Click **Analyze Content** in the sidebar.
3. Review your **score** and **suggestions**.
4. Optimize based on tips (e.g., add keywords, improve sentences).
5. **Pro**: Enable auto-optimize for AI-generated improvements.

**Is it real AI?** Basic version uses advanced heuristics simulating AI; Pro integrates with OpenAI API.

**Performance?** Lightweight, no external calls in free version.
