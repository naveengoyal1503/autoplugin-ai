# AI Content Optimizer Pro

## Description
**AI Content Optimizer Pro** uses AI to analyze and optimize your WordPress posts for SEO, readability, and engagement. Get instant scores and suggestions right in the post editor.

## Features
- **Free Version**:
  - Real-time SEO score calculation.
  - Basic AI suggestions (5 scans/day).
  - Meta box in post editor.
- **Premium Features** ($9.99/month):
  - Unlimited scans.
  - Advanced AI suggestions and auto-apply fixes.
  - Keyword research integration.
  - Priority email support.
  - Export optimization reports.

## Installation
1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate the plugin via **Plugins > Installed Plugins**.
3. Access settings in **Settings > AI Optimizer**.

## Setup
1. Go to **Settings > AI Optimizer**.

## Usage
1. Open a post in the editor.
2. Click **Scan & Optimize** in the meta box.
3. Review the **SEO Score** and suggestions.
4. Premium users: Click **Auto-Apply Fixes** to implement changes automatically.
5. Save the post to update.
