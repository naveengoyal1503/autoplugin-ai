## Installation

1. Download the plugin ZIP.
2. In WordPress Admin: **Plugins > Add New > Upload Plugin**.
3. Activate the plugin.
4. Edit any post/page to see the AI Content Optimizer metabox.

## Setup

1. Go to **Settings > AI Content Optimizer**.
2. Enter your Pro license key (if purchased) to unlock premium features.
3. That's it! No API keys needed.

## Usage

1. Edit a post or page.
2. Click **Analyze Content** in the sidebar metabox.
3. Review your **Score** (0-100%) and **Suggestions**.
4. **Pro**: Click **Optimize** for AI-improved content.
5. Publish! The score badge appears on the frontend.
