# AI Content Optimizer Pro

## Description
**AI Content Optimizer Pro** is a powerful WordPress plugin that uses AI-driven analysis to optimize your posts and pages for SEO. The free version provides basic scoring and limited scans, while Pro unlocks unlimited optimizations, AI rewrite suggestions, and advanced keyword insights.

## Features
- **AI-Powered SEO Scoring**: Get an instant optimization score (0-100%) based on content length, keywords, and readability.
- **Auto-Optimization**: Checkbox to auto-optimize on save.
- **Free Tier**: 5 scans per site.
- **Pro Features** (Premium):
  - Unlimited scans.
  - AI-generated rewrite suggestions.
  - Competitor analysis integration.
  - Priority support.
  - Export reports.

## Installation
1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The optimizer meta box appears on Post/Page editors.

## Setup
1. Go to **Settings > AI Optimizer** to enter your Pro license key (if purchased).
2. Enable auto-optimization in post editors.
3. Free users: Limited to 5 scans; upgrade for more.

## Usage
- Edit a post/page: Check the **AI Content Optimizer** sidebar.
- View your **Optimization Score**.
- Check **Auto-optimize on save** for automatic improvements.
- Pro users get advanced AI suggestions in the meta box.
