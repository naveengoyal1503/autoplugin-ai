# AI Content Optimizer Pro

## Features

- **Free Version**:
  - Real-time readability score (Flesch Reading Ease).
  - Word count and basic keyword extraction.
  - Sidebar meta box for instant analysis on posts.

- **Premium Version** ($4.99/month):
  - AI-powered content rewriting for better engagement.
  - Advanced keyword suggestions and SEO optimization.
  - Priority support and regular updates.

## Installation

1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' screen in WordPress admin.
3. Edit any post to see the AI Content Optimizer meta box on the side.

## Setup

- No setup required for free features.
- For premium: Purchase at [example.com/premium](https://example.com/premium), enter license key in plugin settings (future update).

## Usage

1. Open a post in the editor.
2. Click **Analyze Content** in the sidebar.
3. View free metrics; upgrade for AI rewrite.
4. Premium nags guide upgrades for full profitability[1][2].
