# AffiliateBoost Pro

## Description
AffiliateBoost Pro empowers WordPress site owners to launch and manage their own affiliate marketing programs easily. With real-time referral tracking, customizable commission handling, and automated affiliate registration, this plugin helps boost sales and expand brand reach.

## Features
- Simple affiliate registration and unique code generation.
- Track visitor referrals via affiliate URLs and cookies.
- Store and display commissions earned by affiliates.
- Admin dashboard to view affiliates and their earned commissions.
- Shortcode `[abp_affiliate_link code="AFFILIATE_CODE"]` to create affiliate-specific URLs.
- Basic commission tracking (expandable in premium).

## Installation
1. Upload the `affiliateboost-pro.php` file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Plugin will create necessary database tables automatically on activation.

## Setup
- Register affiliates manually via front-end AJAX (you may add forms using custom HTML).
- Affiliates receive unique codes to distribute.
- Visitors coming via affiliate links are tracked, and commissions are logged.
- View affiliates and commissions under 'AffiliateBoost Pro' menu in WordPress admin.

## Usage
- Use shortcode to embed affiliate links: `[abp_affiliate_link code="YOURCODE"]`.
- Affiliates share their unique URLs to promote your site.
- Monitor affiliate performance and commissions in the admin dashboard.

## Monetization
The plugin offers a freemium model:
- Free version includes core features for affiliate management.
- Premium subscription unlocks advanced analytics, customizable commission tiers, fraud detection, email notifications, and priority support.
