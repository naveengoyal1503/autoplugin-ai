# AffiliateBoost Pro

AffiliateBoost Pro is a powerful WordPress plugin designed to help bloggers, niche websites, and e-commerce stores manage affiliate coupon deals effectively to boost conversions and affiliate commissions.

## Features

- Easily add and manage affiliate coupon deals from your WordPress admin dashboard.
- Each coupon includes title, code, description, affiliate URL, and optional expiration date.
- Display all active coupons anywhere on your site via shortcode `[affiliateboost_coupons]`.
- Automated tracking of coupon clicks with redirect to affiliate URLs.
- Skip expired coupons automatically on the front-end display.
- Simple, clean interface for managing coupons.

## Installation

1. Upload `affiliateboost-pro.php` to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access "AffiliateBoost" from the WordPress admin menu.

## Setup

1. Go to the AffiliateBoost admin page through the WordPress dashboard.
2. Add your affiliate coupons including title, code, description, affiliate link, and optional expiry date.
3. Save your coupons.

## Usage

- Use the shortcode `[affiliateboost_coupons]` in any post or page to display the list of active affiliate coupons.
- Visitors clicking on coupon deals will be tracked and redirected via your affiliate URLs.
