# WP DealVault

WP DealVault is a unique WordPress plugin that lets you create a curated, user-friendly coupon and deal marketplace on your website. It helps bloggers, eCommerce sites, and affiliate marketers boost revenue by showcasing exclusive affiliate deals and coupons.

## Features

- Custom post type for managing deals with title, description, affiliate link, and expiry date
- Easy-to-use admin interface for adding and editing deals
- Shortcode `[wpdealvault_display count="10"]` to display current, valid deals anywhere on your site
- Automatic filtering of expired deals
- Affiliate link integration to maximize commissions
- Responsive design and clear call-to-action buttons

## Installation

1. Upload the `wpdealvault.php` file to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'Deals' custom post type in the WordPress admin to add your first deals.

## Setup

- When adding a deal, provide a title and description, an affiliate link, and optionally an expiry date.
- Optionally set featured images to make deals more attractive.
- Use the shortcode `[wpdealvault_display]` in pages or posts to display deals to your users.

## Usage

1. Create deals with valid affiliate URLs and attractive descriptions.
2. Place the shortcode `[wpdealvault_display count="5"]` in any page or widget area to display up to 5 deals.
3. Share and promote your deal pages to generate affiliate income.

This plugin is a practical, monetization-driven solution for anyone looking to capitalize on affiliate marketing by offering curated deals and coupons directly from their WordPress site.
