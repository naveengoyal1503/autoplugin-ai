## Installation

1. Upload the `ai-coupon-generator` folder to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Configure in **Settings > AI Coupon Generator** (add OpenAI API key for real AI).
4. Use shortcode in posts/pages.

## Setup

1. Go to **Settings > AI Coupon Generator**.
2. Enter your OpenAI API key (free tier works for basics).
3. Enable "Pro Version" checkbox (demo; purchase for real pro).
