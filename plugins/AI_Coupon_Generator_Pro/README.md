# AI Coupon Generator Pro

## Features

- **AI-Powered Coupon Display**: Automatically randomizes and displays personalized coupons to boost engagement and conversions.
- **Easy Shortcode Integration**: Use `[ai_coupon_box]` or `[ai_coupon_box limit="3"]` anywhere.
- **Admin Dashboard**: Manage coupons, titles, and settings via WordPress admin.
- **Copy-to-Clipboard**: One-click coupon code copying for users.
- **Freemium Model**: Free core features; Pro unlocks unlimited coupons, real AI generation, analytics, custom branding.
- **Mobile-Responsive**: Clean, professional design works on all devices.
- **Affiliate-Ready**: Perfect for monetizing blogs with personalized deals from Amazon, Shopify, etc.

## Installation

1. Download the plugin ZIP.
2. In WordPress admin, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate.
4. Configure coupons in **Settings > AI Coupons**.
5. Add `[ai_coupon_box]` shortcode to posts/pages.

## Setup

1. Go to **Settings > AI Coupons**.
2. Enter coupons in format: `Brand: CODE - Description` (one per line).
3. Customize the box title.
4. Save changes.
5. **Pro Tip**: Add to sidebars via widgets or Gutenberg block.

## Usage

- **Shortcode Examples**:
  - `[ai_coupon_box]` - Shows 5 random coupons.
  - `[ai_coupon_box limit="3"]` - Limit to 3 coupons.
