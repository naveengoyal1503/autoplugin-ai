# AI Coupon Generator Pro

## Features

- **AI-Powered Coupon Generation**: Automatically creates unique, personalized coupons for your niche (Pro feature).
- **Affiliate Link Integration**: Seamlessly embeds your affiliate tags in all coupon links.
- **Responsive Design**: Beautiful, mobile-friendly coupon cards that boost click-through rates.
- **Shortcode Support**: Easy `[ai_coupon affiliate="amazon" category="electronics" limit="3"]` usage.
- **Admin Dashboard**: Configure affiliate tags and settings.
- **Pro Upsell**: Unlimited coupons, real AI API, analytics, custom branding ($49/year).

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > AI Coupons** to configure your affiliate tag.

## Setup

1. In **Settings > AI Coupons**, enter your affiliate tag (e.g., `youramazonid-20`).
2. Use the shortcode in any post/page: `[ai_coupon]`.
3. Customize with attributes: `affiliate`, `category`, `limit`.

## Usage

- **Basic Shortcode**: `[ai_coupon]` - Shows 3 sample Amazon coupons.
- **Custom**: `[ai_coupon affiliate="amazon" category="fashion" limit="5"]`.
- **Pro Features**: Upgrade for real-time AI generation from brands like Amazon, Shopify affiliates.
- **Monetization Tip**: Place coupons in blog posts for high-converting affiliate commissions.
