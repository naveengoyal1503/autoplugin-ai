## Installation

1. Upload the `ai-coupon-generator` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > AI Coupons** and enter your OpenAI API key.

## Setup

1. Obtain a free OpenAI API key from [platform.openai.com](https://platform.openai.com).
2. Paste it in the plugin settings and save.
3. **Pro Upgrade**: For advanced features, purchase at example.com/pro.

## Usage

- Use shortcode: `[ai_coupon product="Your Product Name" link="https://affiliate-link.com"]`.
- Outputs AI-generated coupon code, description, expiry, and affiliate link.
- Example output:
  > **CODE: SAVE20**
  > Get 20% OFF on Your Product! Limited time.
  > **EXPIRES: 7 days**
  > [Shop Now & Save!](affiliate-link)

Display on posts, pages, or widgets. Perfect for blogs monetizing via affiliates and coupons.[1][2][3]
