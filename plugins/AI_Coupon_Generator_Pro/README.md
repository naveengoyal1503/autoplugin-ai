## Installation
1. Upload the plugin ZIP to your WordPress admin > Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Access settings via **AI Coupons** in the admin menu.[1]

## Setup
1. Go to **AI Coupons** in your WordPress dashboard.
2. Enter your OpenAI API key (Pro feature).
3. Configure categories and affiliates.
4. Add the shortcode to any post/page: `[ai_coupon_generator]`. Customize with `category` and `affiliate` attributes.[2][5]

## Usage
- **Frontend**: Display dynamic coupon grids that generate unique codes on-the-fly.
- **Monetization**: Earn via affiliates (e.g., Amazon) or promote your own deals. Track clicks in Pro version.[1][3]
- **Customization**: Style with CSS or upgrade for advanced options.
- **Example**: Create a dedicated coupons page for SEO and traffic.[2]
