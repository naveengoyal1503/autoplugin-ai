# Exclusive Deals Manager

## Features

- **Easy Deal Management**: Add unlimited exclusive coupons via simple admin interface (Pro: truly unlimited with analytics).
- **Shortcode Support**: Use `[exclusive_deals]` anywhere to display rotating deals.
- **Auto Widget**: Floating deals sidebar for maximum visibility.
- **Click Tracking**: Basic affiliate link tracking (Pro: Google Analytics integration + conversion reports).
- **Responsive Design**: Mobile-optimized deal displays.
- **Freemium Model**: Free core features; Pro unlocks advanced tools for higher earnings.

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Exclusive Deals** to add your affiliate deals.

## Setup

1. In the admin page, enter deals in format: `Title|Affiliate URL|Discount|Brand|active` (one per line).
2. Save and use shortcode `[exclusive_deals]` in posts/pages.
3. Enable/disable widget in settings.
4. (Pro) Connect Google Analytics for detailed commission tracking.

## Usage

- **Shortcode**: `[exclusive_deals]` - Embeds deals block.
- **Widget**: Automatically shows in footer (toggle in settings).
- **Customization**: Edit CSS in `edm.css` or override via theme.
