# SmartAffiliate Pro

An intelligent affiliate marketing management system for WordPress that automatically tracks, optimizes, and manages affiliate links with real-time performance analytics.

## Features

- **Link Management**: Create and manage unlimited affiliate links
- **Click Tracking**: Real-time tracking of clicks on each affiliate link
- **Performance Analytics**: View top-performing affiliate links
- **Simple Shortcodes**: Use `[affiliate_link name="link-name"]` to insert links
- **Dashboard**: Quick overview of total links and total clicks
- **Responsive Design**: Works on desktop and mobile
- **Easy Setup**: No coding required

## Installation

1. Download the SmartAffiliate Pro plugin
2. Go to WordPress Admin > Plugins > Add New
3. Click "Upload Plugin" and select the plugin file
4. Click "Install Now"
5. Activate the plugin

## Setup

1. Navigate to SmartAffiliate Pro in the admin menu
2. Go to "Manage Links" tab
3. Enter your affiliate link details
4. Click "Add Link"

## Usage

### Adding Affiliate Links

1. Go to SmartAffiliate Pro > Manage Links
2. Fill in the Link Name and Affiliate URL
3. Click "Add Link"

### Using Shortcodes

Insert affiliate links in posts/pages:


[affiliate_link name="Amazon Prime"]


Replace "Amazon Prime" with your link name.

### Viewing Analytics

Go to SmartAffiliate Pro > Analytics to see:
- Top performing links
- Total clicks per link
- Click performance visualization
