## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Configure in **Settings > Affiliate Popup**.

## Setup

1. Go to **Settings > Affiliate Popup**.
2. Enable the plugin.
3. Set **Show Probability** (0.1-1.0) and **Delay** (ms).
4. Add affiliate entries: Text, Link (e.g., Amazon), optional Image URL.
5. Save settings. Popups appear on frontend.

**Example Affiliate:**
- Text: "Love this blog? Get 20% off via my link!"
- Link: `https://amazon.com/dp/B08XYZ?tag=youraffiliateid`

## Usage

- Popups trigger randomly after delay, driving affiliate clicks.
- Track performance via your affiliate dashboard (e.g., Amazon Associates).
- **Pro Tip**: Use pricing psychology like $9.99 links for 28% higher conversions.[4]
