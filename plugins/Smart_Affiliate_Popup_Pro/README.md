# Smart Affiliate Popup Pro

## Features

- **Personalized Popups**: Displays affiliate products based on visitor behavior and page category (e.g., tech, fashion).[1][2][5]
- **Conversion-Optimized**: Uses exit-intent, scroll triggers, and timed popups to maximize clicks without annoying users.
- **Easy Setup**: Shortcode-based with drag-and-drop customization in Pro.
- **A/B Testing (Pro)**: Test headlines, images, and CTAs for optimal performance.
- **Analytics Integration**: Track impressions, clicks, and conversions.
- **Mobile-Responsive**: Works seamlessly on all devices.[2]
- **Freemium Model**: Basic free, Pro unlocks unlimited popups and advanced targeting ($49/year).[1][6]

## Installation

1. Download and upload the plugin ZIP to your WordPress admin > Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Add your affiliate links via shortcode: `[sap_popup affiliate_link="YOUR_LINK" image="IMAGE_URL" title="Your Title" delay="10000"]`.

## Setup

1. Go to **Settings > Smart Affiliate Popup** (Pro dashboard available).
2. Enter your default affiliate links and images.
3. Add body classes like `tech-category` to pages for behavior targeting.
4. Customize popup delay and triggers.
5. For Pro: Enable A/B testing and integrations (Google Analytics, email capture).[2][5]

## Usage

- **Embed Anywhere**: Use shortcode in posts/pages or widgets.
- **Behavior Tracking**: Plugin auto-detects categories; customize via AJAX for user actions (scroll, time on page).
