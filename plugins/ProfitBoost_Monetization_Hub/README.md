# ProfitBoost Monetization Hub

ProfitBoost Monetization Hub is an all-in-one plugin for WordPress sites to maximize revenue through ads, affiliate links, memberships, and digital product sales with smart analytics and optimization.

## Features
- Inject custom ad code into your site
- Manage and display affiliate links
- Showcase digital products
- Membership content placeholder
- Easy admin interface

## Installation
1. Upload the plugin file to your WordPress plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the ProfitBoost Monetization menu in your admin panel to configure settings.

## Setup
- Enter your ad code in the Ad Code field.
- Add affiliate links (one per line: URL|Label) in the Affiliate Links field.
- Add digital products (one per line: URL|Label) in the Digital Products field.

## Usage
- Use the shortcode `[profitboost_affiliate]` to display your affiliate links.
- Use the shortcode `[profitboost_membership]` to display membership content.
- Use the shortcode `[profitboost_product]` to display your digital products.
