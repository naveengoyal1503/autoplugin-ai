# Smart Affiliate Coupons

## Description
**Smart Affiliate Coupons** is a powerful WordPress plugin that lets you generate unique, trackable coupon codes for your affiliate links. Increase conversions by offering exclusive discounts to your audience, perfect for bloggers and marketers.[1][2]

## Features
- Generate unlimited personalized coupon codes (Pro: unlimited, Free: 10 max)
- Track coupon usage and clicks
- Shortcode `[sac_coupon]` for easy embedding
- Admin dashboard to manage coupons
- Affiliate link integration
- **Pro Only:** WooCommerce integration, analytics dashboard, auto-expiration, custom branding

## Installation
1. Upload the plugin files to `/wp-content/plugins/smart-affiliate-coupons`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **Settings > SAC Coupons** to configure your affiliate API key and limits
4. Use the shortcode `[sac_coupon]` in any post or page

## Setup
1. In the admin settings, enter your affiliate program API key (e.g., for Amazon Associates or custom programs)
2. Set coupon limits (upgrade to Pro for unlimited)
3. Customize the affiliate base URL in code if needed (advanced)

## Usage
- Embed `[sac_coupon]` on your blog posts or dedicated coupon pages
- Visitors click "Generate Coupon" to get a unique code
- Track performance in the admin dashboard
- **Pro Tip:** Create SEO-optimized coupon pages for high traffic and conversions[1][2]
