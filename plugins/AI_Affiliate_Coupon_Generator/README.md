# AI Affiliate Coupon Generator

## Features

- **AI-Powered Coupon Generation**: Automatically creates personalized coupons for niches like tech, fashion, and more[1][2].
- **Affiliate Tracking**: Embeds trackable links for Amazon, custom affiliates (Pro integrates more networks)[3].
- **Shortcode Integration**: Easy `[ai_coupon_generator niche="tech" affiliate="amazon"]` placement in posts/pages.
- **Conversion Boost**: Displays dynamic coupons to increase clicks and commissions[1].
- **Free Tier**: 5 coupons/month; **Pro ($49/year)**: Unlimited, analytics, custom branding, premium templates.

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Use shortcode `[ai_coupon_generator]` in any post/page.

## Setup

1. Go to **Settings > AI Coupons** to enter Pro key (optional).
2. Customize niches/affiliates in shortcode attributes.
3. Pro users: Connect affiliate APIs for real-time deals.

## Usage

- **Basic**: Add shortcode to blog posts for instant coupon buttons.
- **Advanced (Pro)**: Widget support, A/B testing, conversion dashboard.
- **Monetization Tip**: Pair with affiliate plugins like wecantrack for full tracking[3].
