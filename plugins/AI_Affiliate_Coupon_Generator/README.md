## Installation

1. Download and upload the plugin ZIP to your WordPress admin > Plugins > Add New.
2. Activate the plugin.
3. Go to **Settings > AI Coupons** to configure affiliate links (JSON format).
4. Add the shortcode `[ai_coupon_generator]` to any post/page.
