# AI Affiliate Coupon Generator

## Features
- **AI-Powered Coupons**: Uses OpenAI to generate unique, catchy coupon codes and descriptions instantly.
- **Affiliate Integration**: Embeds your affiliate links with dynamic coupon parameters for higher conversions[1][2].
- **Shortcode Ready**: Easy `[ai_coupon product="shoes" affiliate_url="https://youraffiliate.link"]`.
- **Freemium Model**: Free for 5 coupons/day; Pro unlocks unlimited, analytics, custom designs ($49/year).
- **Conversion Boost**: Personalized discounts improve reader engagement and sales[1].

## Installation
1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via WordPress Admin > Plugins.
3. Go to Settings > AI Coupons, enter your OpenAI API key.
4. Click "Activate Pro" (demo) or purchase Pro for full features.

## Setup
1. Get free OpenAI API key from openai.com.
2. Paste in plugin settings.
3. Add shortcode to any post/page: `[ai_coupon product="your product" affiliate_url="https://aff.link/?coupon=CODE"]`.
4. For Pro: Integrates with WooCommerce, analytics dashboard.

## Usage
- **Frontend**: Click "Generate Coupon" to create unique offers.
- **Customization**: Edit CSS in plugin styles.
- **Monetization Tips**: Target niches like software, travel[2]. Track clicks for affiliate commissions[3].
- **Pro Features**: Unlimited generations, A/B testing, email capture.
