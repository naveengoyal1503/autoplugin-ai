# SmartAffiliate Link Booster

## Description
SmartAffiliate Link Booster automatically converts your product and external links into optimized affiliate links. It supports Amazon affiliate tag insertion, basic A/B split testing for links, and geotargeting-ready affiliate parameters to maximize your affiliate revenue.

## Features
- Automatically detect and convert outbound product links into affiliate links
- Support for Amazon affiliate tag insertion with automatic affiliate ID
- Optional A/B testing for affiliate links to optimize conversion
- Add ‘nofollow’ and ‘noopener’ attributes automatically to affiliate links
- Simple dashboard settings for affiliate ID and testing toggle

## Installation
1. Upload `smartaffiliate-link-booster.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Navigate to Settings > SmartAffiliate to configure your affiliate ID and enable A/B testing

## Setup
- Enter your default affiliate ID (e.g., your Amazon associate ID) in the settings page
- Check the box to enable A/B testing for experimenting with different affiliate tags

## Usage
- Add affiliate links naturally in your posts as normal URLs
- The plugin will automatically convert recognized product links (like Amazon) into affiliate links with your ID
- When A/B testing is enabled, it randomly serves one of two affiliate tag variants to test performance
- Links open in new tabs and use rel="nofollow noopener" for SEO and security

This plugin is ideal for bloggers and affiliate marketers seeking a streamlined way to boost affiliate commissions without manual link editing.
