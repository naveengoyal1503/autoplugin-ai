# Smart Tip Jar Pro

## Features

- **Easy Tip Jars**: Add customizable donation buttons anywhere with a shortcode `[smart_tip_jar]`.
- **Preset Amounts**: Quick buttons for $3, $5, $10, etc. (customizable).
- **PayPal Integration**: Direct payments via PayPal (email setup required).
- **Free Version**: Basic jar with 1 design and limited amounts.
- **Pro Features** (Subscription $29/year):
  - Unlimited tip jars per page.
  - Premium themes and animations.
  - Analytics dashboard for donation tracking.
  - Custom thank-you pages.
  - Stripe integration.

## Installation

1. Upload the `smart-tip-jar` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Tip Jar** to configure.

## Setup

1. **Enable the plugin** and set your **PayPal email**.
2. Customize **title** and **amounts** (comma-separated, e.g., `3,5,10,20`).
3. Add shortcode `[smart_tip_jar]` to any post, page, or widget.

## Usage

- Place the shortcode in content for instant tip jars.
- Users click amounts and donate via PayPal.
- **Pro Tip**: Add to sidebars or footers for passive income.
