# Smart Deal Aggregator

## Description
Smart Deal Aggregator is a WordPress plugin that automatically curates and displays niche-specific deals, coupons, and affiliate offers on your website. It helps bloggers, affiliate marketers, and niche site owners increase revenue by promoting relevant deals with integrated affiliate links.

## Features

- Automatic deal curation based on user-defined keywords
- Affiliate ID support to track commissions
- Simple shortcode `[smart_deals]` to display deals anywhere on your site
- Settings page to configure keywords, affiliate ID, and number of deals to display
- Lightweight, single-file plugin optimized for performance

## Installation

1. Download the plugin PHP file.
2. Upload the file to your `/wp-content/plugins/` directory.
3. Activate the plugin through the WordPress admin plugins page.
4. Navigate to **Settings > Smart Deal Aggregator** to configure your affiliate ID, niche keywords, and number of deals.

## Setup

- Enter your affiliate program ID.
- Add comma-separated keywords related to your niche (e.g., laptops, smartphones).
- Select how many deals to display (default is 5).
- Save changes.

## Usage

Insert the shortcode `[smart_deals]` into any post, page, or widget where you want curated deals to appear. The plugin will show deals matching your keywords with your affiliate links embedded.

Example:


[smart_deals]


## Monetization
Use your affiliate ID in links to earn commissions on sales generated through your site. Optionally, negotiate sponsored deals to feature premium paid listings.
