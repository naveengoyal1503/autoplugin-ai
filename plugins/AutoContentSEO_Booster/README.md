# AutoContentSEO Booster

## Description
AutoContentSEO Booster automatically generates SEO-optimized content snippets and metadata suggestions using AI-powered techniques to improve your WordPress site's search rankings and user engagement.

## Features
- One-click AI-generated SEO title and description suggestions
- Live SEO snippet preview in the post editor sidebar
- Supports posts and pages
- Saves SEO metadata as custom fields
- Lightweight and easy to use
- Freemium-ready for future advanced AI capabilities

## Installation
1. Upload the plugin file to the `/wp-content/plugins/` directory or install via the WordPress admin plugin uploader.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Edit any post or page.
4. Use the 'SEO Optimizer' meta box on the right sidebar.

## Setup
- Simply click the "Generate SEO Snippet" button to auto-fill the SEO title and description based on your content.
- Customize the SEO title and description or accept the AI suggestions.
- Update or publish your post to save SEO metadata.

## Usage
- Works on post and page edit screens.
- Use suggested SEO fields to improve your metadata without leaving WordPress.
- Enhances SEO performance by providing optimized metadata that helps search engines and improves click-through rates.

Developed to empower WordPress users with automated SEO content generation, AutoContentSEO Booster helps save time while boosting site discoverability and engagement.
