A powerful WordPress plugin to cloak affiliate links, track clicks, and boost commissions with analytics.

**Free Version:**
- Shortcode `[cloaklink url="https://aff.link" text="Buy Now"]` to cloak links.
- Basic click tracking with IP and timestamp logs.
- Admin dashboard to view stats.

**Pro Version ($9/month):** (Upsell in plugin)

- Auto-optimization: Rotate high-performing links.
- Priority support and unlimited links.

## Installation

1. Upload the plugin ZIP to your WordPress admin.
2. Activate the plugin.
3. Use shortcode in posts/pages: `[cloaklink url="YOUR_AFFILIATE_URL" text="Your Link Text"]`.

## Setup

1. Go to **Settings > Affiliate Cloaker** to view click stats and activate Pro.
2. Enter Pro key `prokey123` (demo) or purchase real key from our site.
