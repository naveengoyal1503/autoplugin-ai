# Smart Affiliate Link Cloaker Pro

A powerful WordPress plugin to cloak, track, and optimize affiliate links for maximum conversions.

## Features

### Free Version
- **Link Cloaking**: Create short, branded cloaked links (e.g., `yoursite.com/sac/my-product`).
- **Click Tracking**: Basic click counters.
- **Shortcode Support**: Embed links with `[sac_link id="1"]`.
- **Admin Dashboard**: Manage links easily.

### Premium Version ($9/month)
- **Advanced Analytics**: Detailed stats, geolocation, device tracking.
- **A/B Testing**: Rotate multiple affiliate links automatically.
- **Unlimited Links**: No limits on cloaked links.
- **Link Scheduling**: Set expiration dates.
- **Priority Support**: Email support within 24 hours.
- **Export Reports**: CSV downloads for performance analysis[1][2].

## Installation

1. Upload the `smart-affiliate-cloaker` folder to `/wp-content/plugins/`.
2. Activate the plugin via **Plugins > Installed Plugins**.
3. Visit **Settings > Link Cloaker** to create your first cloaked link.
4. Flush permalinks if needed (**Settings > Permalinks > Save**).

## Setup

1. Go to **Settings > Link Cloaker**.
2. Enter a unique **slug** (e.g., `amazon-deal`), **target affiliate URL**, and **title**.
3. Click **Save Link**.
4. Use the generated URL: `https://yoursite.com/sac/slug` or shortcode `[sac_link id="1"]`.

## Usage

- **Direct Links**: Share `yoursite.com/sac/my-link` on social media, emails.
- **Posts/Pages**: Insert shortcode for dynamic links.
- **Tracking**: View clicks in the dashboard (premium unlocks full stats).
- **Monetization Tip**: Combine with affiliate programs like Amazon Associates for passive income[2][4][5].
