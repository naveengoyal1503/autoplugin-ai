# Smart Affiliate Link Cloaker Pro

A powerful WordPress plugin to cloak affiliate links, track clicks with detailed analytics, and boost conversions. **Freemium model**: Free core features, premium unlocks advanced tools like link rotation and A/B testing.

## Features

- **Link Cloaking**: Automatically masks affiliate URLs with clean, branded redirects (e.g., `yoursite.com/go/product1`).
- **Click Tracking**: Real-time analytics dashboard showing clicks per link, IP, user agent, and timestamps.
- **Shortcode Support**: Use `[sac_link id="product1" text="Buy Now"]` anywhere.
- **Premium Features** (Upgrade for $9/month):
  - Link rotation: Rotate multiple affiliate links for better payouts.
  - A/B testing: Test links automatically.
  - Custom redirects and priority support.
- **Privacy Compliant**: No data stored without consent; GDPR-ready.

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Affiliate Cloaker** to add links.

## Setup

1. In the admin dashboard: **Settings > Affiliate Cloaker**.
2. Enter **Link ID** (unique, e.g., `amazon-deal`), **Affiliate URL**.
3. Premium users: Add rotation URLs (comma-separated).
4. Optional: Enter license key for premium activation.
5. Save. Use shortcodes like `[sac_link id="amazon-deal" text="Get 50% Off!"]` in posts/pages.

## Usage

- **Embed Links**: Place shortcodes in content. All links auto-cloak and track.
- **View Analytics**: Check clicks in the settings page.
