## Description

**Smart Affiliate Link Cloaker Pro** is a powerful WordPress plugin designed to cloak your affiliate links, track clicks, and provide actionable insights to maximize your earnings. The free version offers basic cloaking and simple click logging. Upgrade to **Pro** for advanced features like conversion tracking, A/B testing, detailed analytics, and priority support.

Perfect for affiliate marketers, bloggers, and site owners looking to protect links, boost conversions, and scale revenue.

## Features

### Free Version
- **Link Cloaking**: Use shortcode `[sac_link url="https://affiliate.com/?ref=123" text="Buy Now"]` to create pretty, branded links.
- **Click Tracking**: Logs IP, user agent, and timestamps for up to 50 recent clicks.
- **Admin Dashboard**: View logs and clear data easily.
- **Pretty URLs**: Links like `yoursite.com/go/` redirect to affiliates.

### Pro Version ($49/year)
- **Advanced Analytics**: Conversion rates, geographic data, device breakdowns.
- **A/B Testing**: Rotate links automatically to find winners.
- **Conversion Tracking**: Integrate with affiliate networks to track sales.
- **Link Management**: Bulk import/export, categories, expiration dates.
- **Priority Support & Updates**: Ongoing improvements and email support.

## Installation

1. Download the plugin ZIP.
2. In your WordPress admin, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate **Smart Affiliate Link Cloaker Pro**.
4. Access settings via **Settings > Affiliate Cloaker**.

## Setup

1. **Add Links**: Use the shortcode in posts/pages:
   
   [sac_link url="https://example.com/offer?ref=you" text="Get 50% Off"]
   
2. **View Stats**: Go to **Settings > Affiliate Cloaker** dashboard.
3. **Upgrade**: Visit [Pro Upgrade](https://example.com/pro) for premium features.

## Usage

- **Shortcode Options**:
  | Attribute | Description | Example |
  |-----------|-------------|---------|
  | `url` | Affiliate URL (required) | `https://affiliate.com/?ref=123` |
  | `text` | Link text | `Shop Now` |
  | `id` | Custom ID (auto-generated if omitted) | `offer1` |

- **Dashboard**: Monitor clicks, clear logs. Pro users get charts and exports.
