# WP Smart Upsell Pro

An intelligent WooCommerce upsell and cross-sell plugin that boosts sales by recommending personalized products based on user purchase history.

## Features

- Personalized product recommendations based on customer purchase patterns
- Displays upsell options on product pages
- Easy integration with WooCommerce
- Lightweight and performance optimized
- Freemium version with core functionality
- Premium upgrade offers advanced AI-driven recommendation algorithms, analytics, and multi-funnel strategies

## Installation

1. Upload `wp-smart-upsell-pro.php` to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Ensure WooCommerce is installed and active.

## Setup

No additional setup needed for the basic free version.


## Usage

- Visit any WooCommerce product page.
- Scroll down to see personalized upsell recommendations.
- Customers can add recommended products to the cart directly.

## Premium Version

Upgrade to unlock:

- AI-powered recommendation engine
- Detailed upsell performance analytics
- Create multiple upsell funnels
- Priority support and updates
