# WP Smart Upsell Pro

Automatically recommends relevant upsell and cross-sell offers to customers based on their browsing and purchase behavior, increasing average order value and revenue.

## Features
- Automatically suggests upsell products on single product pages
- Recommends complementary items in the cart
- Easy-to-use admin settings
- Works with WooCommerce
- Responsive design

## Installation
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Ensure WooCommerce is installed and activated

## Setup
1. Go to Settings > Smart Upsell
2. Configure upsell display options
3. Assign upsell products manually or let the plugin auto-suggest

## Usage
- The plugin will automatically display upsell recommendations on product and cart pages
- Customize upsell products per product in WooCommerce product edit screen
- Monitor upsell performance in WooCommerce reports

## Requirements
- WordPress 5.0+
- WooCommerce 4.0+
