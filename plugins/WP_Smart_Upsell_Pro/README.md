# WP Smart Upsell Pro

Increase your store's average order value with smart upsell and cross-sell suggestions powered by user behavior and product relationships.

## Features
- Automatically suggests relevant products on product pages
- Simple admin settings to enable/disable upsell
- Responsive design for all devices
- Works with any WooCommerce store

## Installation
1. Upload the plugin folder to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Smart Upsell to configure

## Setup
- Enable Smart Upsell in the plugin settings
- The plugin will automatically display upsell suggestions on product pages

## Usage
- Visit any product page to see upsell suggestions
- Customize upsell logic by editing the plugin code or using premium add-ons
