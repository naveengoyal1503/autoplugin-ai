# WP Revenue Booster

Boost your WordPress site's revenue with automated coupon distribution, affiliate link tracking, and sponsored content management.

## Features
- Manage coupon codes for your audience
- Track and display affiliate links
- Easily add and update sponsored content
- Simple shortcode integration for all features

## Installation
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the 'Revenue Booster' menu to configure your settings

## Setup
- Enter your coupon code, affiliate link, and sponsored content in the plugin settings
- Use the provided shortcodes to display content on your site

## Usage
- `[revenue_booster_coupons]` - Displays the current coupon code
- `[revenue_booster_affiliate]` - Displays the affiliate link
- `[revenue_booster_sponsored]` - Displays the sponsored content
