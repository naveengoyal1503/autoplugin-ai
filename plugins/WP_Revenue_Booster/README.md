# WP Revenue Booster

Automatically optimizes ad placement, affiliate links, and coupon offers to maximize revenue for WordPress sites.

## Features
- Easy ad code injection
- Automatic affiliate link conversion in content
- Coupon offers display in footer
- Simple admin settings

## Installation
1. Upload the plugin file to your WordPress plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > Revenue Booster to configure your ad code, affiliate links, and coupons.

## Setup
- Enter your ad code (e.g., Google AdSense) in the Ad Code field.
- Add affiliate links as JSON: `{"keyword": "affiliate_url"}`
- Add coupons as JSON: `[{"title": "Offer", "url": "link", "code": "CODE"}]`

## Usage
The plugin will automatically inject ads, convert keywords to affiliate links in your content, and display coupon offers in the footer of your site.
