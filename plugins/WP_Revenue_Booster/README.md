# WP Revenue Booster

Boost your WordPress site revenue by rotating affiliate links, coupons, and sponsored banners based on user behavior and content context.

## Features
- Rotates affiliate links, coupons, and sponsored banners
- Easy-to-use admin interface
- No coding required
- Works with any WordPress theme

## Installation
1. Upload the `wp-revenue-booster` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Revenue Booster to configure your links, coupons, and banners

## Setup
- Enter your affiliate links (one per line)
- Enter your coupons (one per line)
- Enter your sponsored banners (image_url|link_url, one per line)

## Usage
Once configured, the plugin will automatically rotate and display revenue elements on your site. No additional setup required.
