# WP Revenue Booster

Automatically optimize ad placements, affiliate links, and coupon offers for maximum revenue on your WordPress site.

## Features
- Easy-to-use admin settings for ad codes, affiliate links, and coupons
- Randomized display of affiliate links and coupons for better engagement
- Responsive design for all devices

## Installation
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Revenue Booster to configure your ad codes, affiliate links, and coupons

## Setup
- Enter your ad code (e.g., Google AdSense)
- Add affiliate links (one per line)
- Add coupons in the format: `code|description` (one per line)

## Usage
Once configured, the plugin will automatically display optimized content at the bottom of every page. Customize the appearance by editing `style.css` in the plugin directory.
