# WP Revenue Booster

Automatically optimizes and manages multiple monetization streams for WordPress sites, including ads, affiliate links, and premium content.

## Features
- Inject AdSense code site-wide
- Add affiliate links to every page
- Restrict content with a premium shortcode
- Easy-to-use admin settings

## Installation
1. Upload the plugin file to your WordPress plugins directory.
2. Activate the plugin from the WordPress admin panel.
3. Go to the WP Revenue Booster settings page and enter your AdSense code, affiliate link, and premium content message.

## Setup
- Navigate to **Settings > WP Revenue Booster** to configure your monetization options.
- Use the `[premium_content]` shortcode to restrict content to logged-in users.

## Usage
- Add AdSense code and affiliate links in the settings.
- Wrap premium content with `[premium_content]Your content here[/premium_content]`.
- Customize the premium content message for non-logged-in users.
