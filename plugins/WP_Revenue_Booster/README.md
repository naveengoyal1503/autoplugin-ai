# WP Revenue Booster

Automatically optimizes and rotates monetization strategies (ads, affiliate links, coupons, memberships) based on visitor behavior and content type to maximize revenue.

## Features
- Rotates monetization strategies for optimal revenue
- Easy-to-use admin settings
- Supports ads, affiliate links, coupons, and memberships
- Lightweight and non-intrusive

## Installation
1. Upload the plugin file to your WordPress plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > Revenue Booster to configure your monetization options.

## Setup
- Enable the monetization strategies you want to use (ads, affiliate links, coupons, memberships).
- Save your settings.

## Usage
The plugin will automatically inject the selected monetization strategy into your site's header. You can monitor and adjust settings at any time in the admin panel.
