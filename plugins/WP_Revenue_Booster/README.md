# WP Revenue Booster

Automatically optimizes ad, affiliate, and coupon placements for maximum revenue on WordPress sites.

## Features
- Injects ad code, affiliate links, and coupon codes into posts
- Easy-to-use admin settings
- Responsive design

## Installation
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Revenue Booster to configure your ad, affiliate, and coupon codes

## Setup
- Enter your ad code (e.g., Google AdSense)
- Add your affiliate link
- Input your coupon code

## Usage
The plugin will automatically inject your configured content into single posts for optimized revenue generation.
