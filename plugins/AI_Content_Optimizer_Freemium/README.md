Analyze and optimize your WordPress content for **readability**, **SEO**, and **engagement** with AI-powered insights. Free basic analysis; premium unlocks advanced features.

## Features

### Free Version
- **Real-time Content Analysis**: Get **readability**, **SEO**, and **engagement** scores in post editor meta box.
- **Basic Metrics**: Word count, sentence length, link detection.
- **Settings Page**: Simple configuration.

### Premium Version ($9/month)
- **AI-Powered Suggestions**: Personalized tips to improve content.
- **One-Click Optimization**: Auto-fix readability and SEO issues.
- **Bulk Processing**: Optimize entire site content.
- **Priority Support** & regular updates.
- **Export Reports**: Download analysis CSV.

## Installation

1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Edit a post/page – see the **AI Content Optimizer** meta box on the side.
4. **Upgrade to premium** via Settings > AI Content Optimizer.

## Setup

1. Go to **Settings > AI Content Optimizer**.
2. Enter your premium license key (if purchased) to unlock pro features.
3. Re-analyze content by clicking "Re-analyze" in the meta box.

## Usage

- **In Post Editor**: Scores update on save. Premium users get suggestion lists and auto-optimize button.
- **Basic Workflow**:
  1. Write content.
  2. Check scores in meta box.
  3. Adjust based on feedback.
  4. Publish!

- **Premium Workflow**:
  1. Enable premium key.
  2. Get AI suggestions.
  3. Click **Optimize** for auto-improvements.
