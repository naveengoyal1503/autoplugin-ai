# AI Content Optimizer Freemium

**Boost your WordPress site's SEO and engagement with AI-powered content optimization.** Free basic analysis; premium unlocks advanced features.

## Features

### Free Version
- **Readability Score**: Instant score based on content length and structure.
- **Basic Suggestions**: 3 actionable tips to improve posts/pages.
- **Sidebar Meta Box**: Analyze on post edit screen.

### Premium Version ($49/year)
- **Advanced AI Keyword Suggestions**: Context-aware keywords.
- **Competitor Analysis**: Score vs. top results.
- **Bulk Optimization**: Process multiple posts.
- **Priority Support & Updates**.

## Installation

1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Edit any post/page to see the sidebar optimizer.

## Setup

1. Go to **Settings > AI Optimizer**.
2. Enter Premium Key (purchase at example.com/premium) for advanced features.
3. Click **Analyze Content** in post editor.

## Usage

- **Basic**: Edit post > Scroll to AI Content Optimizer sidebar > Click Analyze.
- **View Results**: Score and suggestions update instantly.
- **Premium Perks**: More detailed AI insights appear after activation.
