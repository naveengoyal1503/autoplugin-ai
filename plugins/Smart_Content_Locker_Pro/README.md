# Smart Content Locker Pro

## Features
- **Easy Shortcode Integration**: Use `[content_locker amount="5.00"]Premium content[/content_locker]` to lock any content.
- **PayPal Payments**: Accept one-time micro-payments or donations via PayPal buttons.
- **Customizable**: Set amount, button text, and teaser message per locker.
- **Mobile Responsive**: Works seamlessly on all devices.
- **Pro Upgrade**: Analytics, payment verification, multiple tiers, A/B testing (freemium model).

## Installation
1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via WordPress admin > Plugins.
3. Go to Settings > Content Locker to configure PayPal email and Client ID.

## Setup
1. Get a PayPal Client ID from [PayPal Developer](https://developer.paypal.com).
2. Enter your PayPal email and Client ID in plugin settings.
3. Use sandbox mode for testing.

## Usage
- Wrap content: `[content_locker amount="9.99" button_text="Unlock Premium Guide"]Your exclusive content here.[/content_locker]`
- Supports posts, pages, widgets.
- Content unlocks post-payment (pro verifies via API).
