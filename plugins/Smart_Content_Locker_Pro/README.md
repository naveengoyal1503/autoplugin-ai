# Smart Content Locker Pro

A powerful WordPress plugin for monetizing your content by locking premium material behind email subscription requirements, paywalls, and referral gates.

## Features

- **Email Gating**: Lock content behind email capture forms
- **Easy Shortcode Integration**: Simple `[content_locker]` shortcode
- **Email Marketing Integration**: Connect with Mailchimp and other platforms
- **Conversion Analytics**: Track email captures and user engagement
- **Cookie-Based Access**: Remember users after email submission
- **Customizable Lock Messaging**: Add custom button text and prompts
- **Dashboard Analytics**: View detailed conversion reports
- **Freemium Model Ready**: Premium features available for upgrades

## Installation

1. Download the Smart Content Locker Pro plugin
2. Upload the plugin folder to `/wp-content/plugins/` directory
3. Activate the plugin through the WordPress Plugins menu
4. Navigate to Content Locker in the admin sidebar
5. Configure your email marketing settings

## Setup

1. Go to Settings → Content Locker → Settings
2. Enter your Mailchimp API Key and List ID
3. Click Save Settings
4. You're ready to start locking content!

## Usage

### Basic Email Lock


[content_locker type="email"]Your premium content here[/content_locker]


### Custom Button Text


[content_locker type="email" button_text="Access Premium Content"]Your premium content here[/content_locker]


### With Redirect URL


[content_locker type="email" redirect_url="https://example.com/thank-you"]Your premium content here[/content_locker]
