# Smart Content Locker Pro

A powerful freemium WordPress plugin to lock your premium content and generate leads or revenue.

## Features

### Free Version
- **Email Unlock**: Shortcode `[content_locker type="email"]Your locked content[/content_locker]` gates content behind email submission.
- Easy shortcode integration.
- Responsive design.

### Premium Features (Upgrade for $49/year)
- Social share unlocks (Twitter, Facebook, LinkedIn).
- Payment unlocks via Stripe or PayPal.
- Analytics dashboard: Track unlocks, conversion rates.
- Unlimited lockers with A/B testing.
- Custom templates and priority support.

## Installation

1. Upload the `smart-content-locker` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode in any post/page.

## Setup

1. Go to **Settings > Content Locker** for basic config (free version notice).
2. Insert shortcode: `[content_locker title="Get Exclusive Tips!" message="Sign up free:"]Your premium content here.[/content_locker]`
3. Customize via shortcode attributes: `type`, `title`, `message`.

## Usage

- **Basic Locker**: Wrap content in shortcode. Users enter email to unlock (demo: auto-unlocks on click).
- **Admin Dashboard**: View settings and upgrade prompt.
