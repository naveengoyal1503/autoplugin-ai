# AffiliateLink Booster

AffiliateLink Booster automatically converts product mentions in your posts into affiliate links to increase your affiliate marketing revenue effortlessly.

## Features

- Auto-detects product keywords and transforms them into affiliate links within post content.
- Customizable affiliate URLs with optional nofollow and open-in-new-tab settings.
- Adds a customizable affiliate disclaimer (premium feature).
- Link cloaking in the premium version to protect affiliate commissions.
- Supports multiple affiliate programs with keyword-based linking.
- Simple admin interface to manage affiliate product keywords and URLs.

## Installation

1. Download the plugin and upload the PHP file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Settings > AffiliateLink Booster** to configure your affiliate keywords and URLs.

## Setup

- Enter product keywords and corresponding affiliate URLs in the settings page.
- Enable or disable `nofollow` and `open in new tab` options for each link.
- Save settings.

## Usage

- Write your posts naturally mentioning product names matching your configured keywords.
- The plugin will automatically convert the first occurrence of each keyword into an affiliate link.
- In the premium version, a disclosure disclaimer is appended automatically.
- Monitor affiliate link performance using premium add-ons.

## Monetization Model

The basic plugin is free and functional. A premium subscription unlocks:

- Multiple affiliate network integration
- Link cloaking and A/B testing
- Enhanced affiliate link reporting
- Automated affiliate disclosures

This plugin targets affiliate marketers and bloggers seeking an automated, user-friendly way to maximize affiliate commissions with minimal effort.
