# Smart Ad Rotator Pro

**Intelligent ad rotation and monetization plugin for WordPress.**

## Features

- **Ad Rotation**: Automatically rotates between multiple ad codes (AdSense, Amazon Affiliates, custom HTML/video banners) to optimize performance.
- **Performance Tracking**: Real-time stats for impressions, clicks, and CTR (Click-Through Rate).
- **Shortcode Support**: Use `[sar_ad zone="zone1"]` anywhere on your site.
- **Freemium**: Free version supports 1 zone with 1 ad; **Pro** unlocks unlimited zones, A/B testing, advanced analytics, and priority support.
- **Mobile-Responsive**: Ads adapt to all devices.
- **Easy Integration**: Supports Google AdSense, affiliate links, banners, and more.

**Pro Features (Upgrade for $9/mo or $79/year)**:
- Unlimited ad zones and ads per zone.
- A/B testing with auto-optimization.
- Exportable analytics reports.
- Feature gating for premium ad formats.

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Ad Rotator** to configure your ads.

## Setup

1. In the admin panel (**Settings > Ad Rotator**), add your ad codes to **Ad Zone 1** (e.g., paste AdSense script or affiliate HTML).
2. Save changes.
3. Add shortcode `[sar_ad]` to posts/pages/widgets, or use PHP: `<?php echo do_shortcode('[sar_ad]'); ?>`.
4. View stats in the dashboard.

**Pro Upgrade**: Enter license key in settings for unlimited features.

## Usage

- **Basic**: Paste ad HTML/JS into zones; plugin rotates and tracks automatically.
- **Shortcodes**: `[sar_ad zone="zone1"]` for specific zones (Pro: multiple zones).
- **Tracking**: Clicks/impressions logged to database; view in admin.
- **Optimization Tip**: Add 2-3 high-performing ads per zone for best rotation.
