# AffiliateLinkPro

AffiliateLinkPro is a smart affiliate link manager for WordPress. It tracks clicks, conversions, and revenue, and automatically rotates high-performing links to maximize your earnings.

## Features
- Create and manage affiliate links
- Track clicks and conversions
- View analytics in the admin dashboard
- Use shortcodes to insert affiliate links
- Automated link rotation (premium feature)
- Easy setup and integration

## Installation
1. Upload the `affiliatelinkpro` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the AffiliateLinkPro menu in your admin panel to add and manage links

## Setup
1. Add your affiliate links in the admin dashboard
2. Use the shortcode `[aff_link id="1"]` in your posts or pages (replace 1 with your link ID)
3. Monitor clicks and conversions in the dashboard

## Usage
- Add new links via the admin panel
- Insert shortcodes in your content
- View analytics and optimize your links

Upgrade to the premium version for automated link rotation, advanced analytics, and conversion tracking.
