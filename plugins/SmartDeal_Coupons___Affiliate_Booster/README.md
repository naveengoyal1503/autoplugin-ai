# SmartDeal Coupons & Affiliate Booster

SmartDeal Coupons & Affiliate Booster is a powerful WordPress plugin designed to **automatically display and manage coupon deals and affiliate links** on your site, helping you boost affiliate sales and conversions effortlessly.

## Features

- Easy admin interface to add, edit, and delete coupons and deals
- Store coupon title, description, affiliate link, coupon code, and expiry date
- Shortcode `[smartdeal_coupons]` to display active coupons anywhere on your site
- Automatic expiry filtering for coupons
- Click tracking via AJAX for analytics or integrations
- Responsive coupon design for great display on all devices
- Lightweight single PHP file that works out of the box

## Installation

1. Upload the plugin file `smartdeal-affiliate-booster.php` to your WordPress `/wp-content/plugins/` directory
2. Activate the plugin through the WordPress admin dashboard under Plugins -> Installed Plugins
3. A new menu item "SmartDeal Coupons" will appear in the admin sidebar

## Setup

1. Navigate to **SmartDeal Coupons** in the WordPress admin menu
2. Add coupons by filling out the fields: Title, Description, Affiliate Link, Coupon Code (optional), Expiry Date (optional)
3. Click "Add New Coupon" to add more entries
4. Save changes

## Usage

- To display the coupons on any post, page, or widget, use the shortcode:

  
  [smartdeal_coupons]
  

- Visitors will see the coupon title, description, and a coupon code if provided
- Clicks on "Claim Deal" are tracked asynchronously
