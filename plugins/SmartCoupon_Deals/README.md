# SmartCoupon Deals

SmartCoupon Deals is a WordPress plugin that enables site owners to create, manage, and display exclusive affiliate coupon offers with ease. It’s designed to boost affiliate income and user engagement by providing an attractive, user-friendly interface for coupon deals.

---

## Features

- Custom post type for Coupons with coupon codes, affiliate URLs, expiration dates, and discount details
- Easy-to-use meta box for coupon management in the WordPress admin
- Frontend shortcode `[smartcoupon_list]` to show active coupons sorted by expiry
- Copy coupon code-to-clipboard and redirect functionality for better user experience
- Automatic expiry filtering to hide outdated coupons
- Responsive and simple coupon display styled for clarity
- Redeem buttons that copy coupon codes with one click and redirect users

## Installation

1. Upload the plugin PHP file to your WordPress `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Coupons** in the WordPress admin menu to add and manage coupons.

## Setup

- Add a new Coupon post and fill in the coupon code, affiliate link, expiration date, discount description, and content details.
- Use the `[smartcoupon_list]` shortcode on any page or post to display a list of active coupon deals.

## Usage

- Place the shortcode `[smartcoupon_list]` where you want coupons to appear on your site.
- Visitors can click the "Redeem Coupon" button to automatically copy the code and open the affiliate link in a new tab.
- Expired coupons will automatically be excluded from the display.
