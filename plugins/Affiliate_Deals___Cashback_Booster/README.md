# Affiliate Deals & Cashback Booster

A WordPress plugin that aggregates affiliate coupons, deals, and cashback offers tailored to your site's niche to boost affiliate commissions and add value for site visitors.

## Features

- Aggregates deals and coupons from a configurable affiliate network API
- Supports caching offers to optimize API usage and improve performance
- Displays deals via a simple shortcode `[affiliate_deals]`
- Customizable keywords to tailor offers to your niche
- Clean and responsive deal listing

## Installation

1. Upload the plugin file to your `/wp-content/plugins/` directory or install via the WordPress plugin uploader.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Affiliate Deals Booster** to input your affiliate network API URL and define keywords for deals relevant to your site.

## Setup

- Enter the **Affiliate Network API URL** provided by your affiliate program that returns deals in JSON format.
- Set **Keywords** as comma-separated terms matching your niche to filter relevant deals.
- Save changes.

## Usage

- Place the shortcode `[affiliate_deals]` in any post, page, or widget area where you want the deals and cashback offers to appear.
- The plugin will fetch and cache offers for 1 hour to reduce API calls.

## Monetization

- Earn affiliate commissions on user clicks and purchases via integrated offers.
- Optionally offer a premium subscription version with enhanced analytics, priority API access, or additional affiliate network integrations.
