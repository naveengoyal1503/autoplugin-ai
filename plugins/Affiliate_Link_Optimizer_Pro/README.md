# Affiliate Link Optimizer Pro

**AI-Powered Affiliate Link Management for WordPress**

Automatically detects, cloaks, and optimizes your affiliate links for higher conversions. **Free version** includes link cloaking and basic click tracking. **Pro version** adds AI optimization, A/B testing, and advanced analytics.

## Features

### Free Version
- **Auto-detect affiliate links** (Amazon, ClickBank, etc.) in posts
- **Smart link cloaking** with branded text like "Check Price on Amazon"
- **Click tracking dashboard** with stats
- **[alo_link]** shortcode for manual links
- **Privacy-compliant** tracking

### Pro Version ($9/month)
- **AI optimization** suggests best link positions and text
- **A/B testing** for headlines and CTAs
- **Conversion analytics** with heatmaps
- **Unlimited links & campaigns**
- **Export reports** to CSV
- **Priority support**

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`
2. Activate via **Plugins > Installed Plugins**
3. Links auto-optimize on your posts!
4. Access dashboard at **Settings > ALO Pro**

## Setup

1. **Optional:** Enter OpenAI API key in settings for Pro AI features
2. Add links manually: `[alo_link url="https://amzn.to/xyz" text="Buy Now"]`
3. View stats in **Settings > ALO Pro > Analytics**
