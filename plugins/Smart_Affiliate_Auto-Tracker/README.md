# Smart Affiliate Auto-Tracker

**Automatically detect, track, and optimize your affiliate links for maximum conversions.**

## Features

- **Auto-Detection**: Automatically wraps detected affiliate links (Amazon, ClickBank, etc.) with tracking.
- **Click Tracking**: Logs clicks with IP, timestamp, user agent, and link ID.
- **Dashboard Analytics**: View total clicks and recent activity in WordPress admin.
- **Privacy Compliant**: Anonymizes IP tracking and respects GDPR.
- **Freemium Model**: Free core tracking; **Pro** adds A/B testing, conversion tracking, link cloaking, and exportable reports.

## Installation

1. Upload the `smart-affiliate-tracker` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' screen in WordPress admin.
3. Visit **Affiliate Tracker** in your admin menu to view dashboard.

## Setup

- No configuration needed for basic auto-tracking.
- Links matching patterns like `amazon.com` or URLs with `aff=` params are auto-wrapped.
- All data stored securely in your database.

## Usage

1. **Add Content**: Paste affiliate links in posts/pages.
2. **Automatic Tracking**: Plugin wraps eligible links on frontend.
3. **View Stats**: Check **Affiliate Tracker > Dashboard** for clicks.
4. **Pro Upgrade**: For advanced features like custom link management and A/B variants.

## Pro Features
