# AffiliateBoost Link Manager

## Description
AffiliateBoost Link Manager is a WordPress plugin designed to help bloggers and affiliate marketers efficiently manage, cloak, categorize, and track clicks on their affiliate links. This plugin enhances your affiliate marketing efforts by providing easy-to-use tools and click analytics to optimize revenue.

## Features
- Create clean, cloaked affiliate links using custom slugs
- Categorize links for easy organization
- Track click counts on each affiliate link
- Automatic 301 redirect to the target affiliate URL
- Display affiliate links using a shortcode filtered by category
- Admin interface for managing links
- Freemium model with basic features free; advanced features in Pro version (auto-linking, detailed analytics, A/B testing)

## Installation
1. Upload the plugin file to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the Affiliate Links menu in your admin dashboard
4. Add your affiliate links by specifying a unique slug, target URL, and optional category

## Setup
- Each affiliate link will be accessible on your site via `https://yourdomain.com/?aff=slug`
- Use the shortcode `[affiliateboost_links]` to display all affiliate links on any page or post
- To filter by category, use `[affiliateboost_links category="your-category"]`

## Usage
- Add new affiliate links from the admin page
- Use the shortlink URLs in your content instead of raw affiliate links to cloak your URLs and track clicks
- Review click statistics directly from the admin panel
- Upgrade to Pro for advanced optimization features

This plugin is ideal for affiliate marketers and niche bloggers aiming to increase their affiliate revenue through better link management and tracking.
