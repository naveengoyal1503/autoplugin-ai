# DealDriver WordPress Plugin

## Description
DealDriver aggregates, verifies, and displays personalized discount coupons and deals from multiple affiliate partners, helping website owners increase affiliate revenue and user engagement with minimal overhead.

## Features

- Manage multiple coupons and affiliate links easily in the WordPress admin.
- Automatically filter out expired coupons.
- Display coupons using a simple shortcode with customizable number of coupons shown.
- Sort coupons alphabetically for neat presentation.
- Affiliate-friendly links with proper nofollow and security attributes.
- Lightweight, single-file plugin with no external dependencies.

## Installation

1. Upload the `dealdriver.php` file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the DealDriver menu in the WordPress dashboard to add and manage your coupons.

## Setup

- Add coupons by filling out the coupon title, affiliate URL, and optional expiration date.
- Save settings.

## Usage

- Insert the shortcode `[dealdriver_coupons count="5"]` into any post, page, or widget. Adjust the `count` attribute to display a different number of coupons.
- Coupons will display as links directing visitors through your affiliate URLs.
