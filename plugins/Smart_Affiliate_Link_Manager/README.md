## Installation
1. Upload the plugin ZIP or copy `smart-affiliate-manager.php` to `/wp-content/plugins/smart-affiliate-manager/`.
2. Activate via **Plugins > Installed Plugins**.
3. Access **Smart Affiliate** in your admin menu.
4. Add your first link: Keyword (e.g., "best hosting"), Affiliate URL.
5. Links auto-replace in content![8]

## Setup
1. Go to **Smart Affiliate > Add Link**.
2. Enter **Keyword** (e.g., "bluehost") and your **affiliate URL**.
3. Save. Now "bluehost" in posts becomes a tracked affiliate link.
4. Flush permalinks: **Settings > Permalinks > Save**.
5. View analytics at **Smart Affiliate > Analytics**.

## Usage
- **Automatic**: Keywords replace in post content.
- **Manual**: `[sa_link keyword="yourkeyword"]` in posts/pages.
- **Tracking**: Clicks logged; view in dashboard.
- **Pro Tip**: Use high-traffic keywords for max commissions.[2][3]
