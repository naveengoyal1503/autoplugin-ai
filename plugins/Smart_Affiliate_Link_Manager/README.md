# Smart Affiliate Link Manager

## Features

- **Link Cloaking & Tracking**: Automatically cloak affiliate links (e.g., yoursite.com/go/product) and track clicks with a database.
- **Shortcode Support**: Use `[salink id="1"]` to insert tracked links anywhere.
- **Auto-Replacement**: Optionally auto-replace keywords in post content with affiliate links.
- **Admin Dashboard**: Easy interface to add, view stats, and manage unlimited links (Pro: advanced analytics).
- **SEO-Friendly**: 301 redirects preserve link juice.
- **Freemium Ready**: Basic free; upgrade for A/B testing, performance reports, auto-insertion rules.

**Pro Features (Upsell)**: Click analytics dashboard, A/B testing, bulk import, API integrations, priority support.

## Installation

1. Upload the plugin ZIP to your WordPress admin → Plugins → Add New → Upload Plugin.
2. Activate the plugin.
3. Visit **Settings → Affiliate Links** to add your first link.
4. Flush permalinks (Settings → Permalinks → Save) if rewrite rules don't work.

## Setup

1. Go to **Settings → Affiliate Links**.
2. Enter a **Name/Keyword** (e.g., "Bluehost Hosting").
3. Paste your **Affiliate URL** (e.g., raw affiliate link).
4. Submit – get instant cloaked URL like `yoursite.com/go/bluehost-hosting`.
5. Enable **Auto-replace** to automatically link keywords in posts.

## Usage

### Shortcodes
`[salink id="1"]Your Product[/salink]` – Replace 1 with link ID from dashboard.

### In Posts
Write normally; enable auto-replace to convert keywords to links.

### Tracking
All clicks logged in dashboard. Pro version shows charts, conversion estimates.

### Example
Add link: Name="Best Hosting", URL="https://affiliate-link.com". Use `yoursite.com/go/best-hosting` anywhere.
