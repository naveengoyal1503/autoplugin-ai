# Smart Affiliate Link Manager

A powerful freemium WordPress plugin for affiliate marketers to cloak, track, and optimize links effortlessly.

## Features

- **Auto-Linking**: Automatically replaces keywords in posts with cloaked affiliate links.
- **Shortcodes**: Use `[smartlink keyword="amazon"] ` for manual links.
- **Click Tracking**: Basic analytics on link clicks (Pro: Advanced reports).
- **Pretty URLs**: Links like `yoursite.com/go/amazon/` redirect to affiliates.
- **Easy Setup**: Add links via dashboard (keyword|affiliate_url).
- **Mobile-Responsive**: Clean, unobtrusive styling.

## Installation

1. Upload the plugin files to `/wp-content/plugins/smart-affiliate-manager`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Affiliate Manager** to add your links (format: `keyword|https://affiliate-url.com`).
4. Flush permalinks: **Settings > Permalinks > Save Changes**.

## Setup

1. In the settings page, enter affiliate links one per line: `amazon|https://amazon.com/aff?tag=yourtag`.
2. Links auto-appear in posts or use shortcode `[smartlink keyword="amazon"]`.
3. Clicks redirect via `/go/keyword/` and are tracked.

## Usage

- **Automatic**: Keywords in content become links.
- **Shortcode**: `[smartlink keyword="yourkeyword"]`.
- **View Stats**: Basic counts in settings (Pro for full dashboard).
