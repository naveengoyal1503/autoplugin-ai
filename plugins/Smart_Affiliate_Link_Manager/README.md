# Smart Affiliate Link Manager

## Features

- **Automatic Link Cloaking**: Convert affiliate links to shortcodes like `[sam]aff1[/sam]` for clean, branded URLs.
- **Click Tracking**: Real-time analytics on link performance with impressions and clicks.
- **Easy Management**: Add, edit, delete links via intuitive dashboard.
- **SEO-Friendly**: Maintains nofollow attributes and prevents link leakage.
- **Mobile Responsive**: Works seamlessly on all devices.
- **Freemium Ready**: Premium unlocks A/B testing, geo-targeting, and API integrations.

## Installation

1. Upload the `smart-affiliate-manager` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit **Settings > Affiliate Manager** to add your first link.

## Setup

1. Go to **Settings > Affiliate Manager**.
2. Enter the **Original URL** (e.g., Amazon affiliate link).
3. Set a **Shortcode** (e.g., `amazon-deal1`).
4. Click **Add Link**.
5. Use the shortcode in posts: `[sam]amazon-deal1[/sam]`.
