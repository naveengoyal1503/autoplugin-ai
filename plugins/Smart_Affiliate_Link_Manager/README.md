# Smart Affiliate Link Manager

## Features

- **Automatic Link Cloaking**: Replace raw affiliate URLs with branded short links in posts/pages.
- **Click Tracking**: Real-time click counts and basic analytics dashboard.
- **Shortcode Support**: Use `[sal id="123"]` anywhere for tracked links.
- **Admin Dashboard**: Manage links, view stats, add new URLs easily.
- **SEO-Friendly**: Noindex tracking pages, proper redirects (301).
- **Free Forever**: Core features free; premium add-ons available.

**Premium Add-ons (coming soon)**:
- A/B testing for link variations.
- Conversion tracking & Google Analytics integration.
- Unlimited links & custom domains.

## Installation

1. Download and upload the plugin ZIP to your WordPress admin > Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Visit **Settings > Affiliate Links** to add your first affiliate URL.
4. Insert links in posts using the shortcode or let auto-cloaking handle it.

## Setup

1. Go to **Settings > Affiliate Links**.
2. Enter an affiliate URL (e.g., `https://amazon.com/product?tag=yourid`).
3. Copy the generated shortcode `[sal id="X"]`.
4. Flush permalinks if tracking URLs don't work: **Settings > Permalinks > Save**.

## Usage

- **Manual**: Add shortcode to any post/page/content.
- **Auto-Cloak**: Paste raw affiliate links in editor; they auto-convert on save.
- **Track Stats**: View clicks in dashboard.
- **Premium Upgrade**: For A/B tests and pro features, visit [premium site](https://example.com/premium).
