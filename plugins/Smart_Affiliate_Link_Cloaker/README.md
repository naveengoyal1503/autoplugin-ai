# Smart Affiliate Link Cloaker

A powerful WordPress plugin that automatically detects, cloaks, and tracks affiliate links in your content for better conversions and analytics.

## Features
- **Auto-cloaking**: Automatically replaces affiliate links (e.g., Amazon, ClickBank) in posts and pages with branded short links.
- **Click tracking**: Real-time click counts and performance dashboard.
- **Shortcode support**: Use `[afflink url="https://example.com"]Your text[/afflink]` for manual cloaking.
- **Admin dashboard**: View all links, clicks, and stats.
- **Pro upgrade**: A/B testing, custom domains, export reports ($49/year).

## Installation
1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Flush permalinks: **Settings > Permalinks > Save Changes**.

## Setup
- No configuration needed for auto-cloaking.
- Access dashboard at **Affiliate Links** in admin menu.

## Usage
- Write posts with raw affiliate links; they auto-cloak.
- Track clicks in the dashboard.
- Use shortcode for widgets/menus: `[afflink url="YOUR_AFFILIATE_URL"]Click Here[/afflink]`.
