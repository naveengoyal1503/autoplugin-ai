# Smart Affiliate Link Cloaker

**Automatically detect, cloak, track, and monetize affiliate links on your WordPress site.**

## Features

- **Auto-Link Cloaking**: Scans posts for affiliate URLs (Amazon, ref links, etc.) and replaces with short, trackable links.
- **Click Tracking**: Real-time stats on clicks, total revenue potential dashboard.
- **SEO-Friendly Redirects**: 301 redirects preserve link juice.
- **Shortcode Dashboard**: `[salc_dashboard]` for easy stats embed.
- **Admin Panel**: Quick overview and Pro upgrade.
- **Freemium Model**: Free forever; Pro adds A/B testing, custom domains, exportable reports, revenue calculator.

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Flush permalinks: **Settings > Permalinks > Save**.

## Setup

- No configuration needed! Links auto-cloak on frontend posts/pages.
- Embed dashboard: Add `[salc_dashboard]` shortcode to any page/post.
- View stats: **Affiliate Cloaker** menu or shortcode.
