# Smart Affiliate Link Cloaker

**Automatically cloak and track affiliate links to boost commissions with analytics and smart redirects.**

## Features
- **Auto-cloaking**: Detects affiliate links (Amazon, ClickBank, etc.) in posts and cloaks them.
- **Click tracking**: Monitors clicks and displays basic stats via shortcode `[aff-stats]`.
- **Custom links**: Add manual links via admin dashboard.
- **Mobile-friendly**: Responsive redirects.
- **Freemium**: Free core; Pro adds A/B testing, geo-targeting, exports ($49/year).

## Installation
1. Upload the plugin ZIP or copy `smart-affiliate-cloaker.php` to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Visit **Settings > Affiliate Links** to manage links.
