# WooCoupon Booster

WooCoupon Booster is a WooCommerce enhancement plugin that dynamically generates personalized coupon codes based on customer purchase history, encouraging increased conversions and customer loyalty.

## Features

- Automatically generates unique percentage discount coupons based on user's past spending.
- Seamlessly applies coupon discounts in cart for logged-in users.
- Allows one-time use of personalized coupons.
- Displays attractive coupon offers via shortcode.
- Provides admin with dynamically created coupons to boost sales.

## Installation

1. Upload the `woocoupon-booster.php` file to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Ensure WooCommerce is installed and active.

## Setup

- No additional setup is required. The plugin works out of the box for logged-in users.
- Coupons are generated automatically based on user total spent.

## Usage

- Personalized coupons will appear automatically when users view their cart.
- Place `[woocoupon_booster_offer]` shortcode in posts, pages, or widgets to display the personalized coupon offer anywhere on your site.
- Coupons are limited to one use per user.
