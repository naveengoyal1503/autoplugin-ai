# Smart Affiliate Link Optimizer

## Features
- **Auto-cloaking**: Automatically detects and cloaks affiliate links in posts for better conversions.
- **Click tracking**: Logs clicks with IP and basic analytics (premium: full reports).
- **Shortcode support**: Use `[sa_link url="https://affiliate.com?aff=123" text="Buy Now"]`.
- **Rewrite rules**: Pretty `/go/ABC123` links redirect to affiliates.
- **Settings page**: Configure API keys and options.
- **Freemium model**: Free core, premium for A/B testing, geo-targeting, and exports.

## Installation
1. Upload the plugin ZIP to your WordPress admin > Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Visit **Settings > SA Optimizer** to configure.

## Setup
1. Add affiliate links to posts normally (e.g., `https://example.com?aff=123`).
2. They auto-cloak on single posts.
3. Use shortcodes for custom links.
4. For premium: Purchase upgrade, enter API key in settings.

## Usage
- **Auto-mode**: Links matching `?aff=` pattern cloak automatically.
- **Manual**: `[sa_link url="AFFILIATE_URL" text="Your Text"]`.
- **Tracking**: View server logs or upgrade for dashboard analytics.
- **Monetization tip**: Sell premium version for $29/year with advanced features.
