# Smart Affiliate Link Optimizer

## Features

- **Automatic Link Cloaking**: Replace plain URLs with tracked, cloaked shortcodes.
- **Click Tracking**: Monitor clicks and performance for each affiliate link.
- **Simple Setup**: Add links via admin panel in `original|affiliate|name` format.
- **Shortcode Integration**: Use `[sao id="name"]` anywhere.
- **Pro Features** (Upgrade for $9/mo): A/B testing, advanced analytics, auto-optimization, unlimited links, export reports.

## Installation

1. Upload the plugin files to `/wp-content/plugins/smart-affiliate-optimizer`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Affiliate Optimizer** to add your links.

## Setup

1. In the settings page, enter links one per line: `https://example.com/review|https://affiliate-link.com|Buy Now Button`.
2. Save settings.
3. Replace original URLs in posts with `[sao id="Buy Now Button"]` or let auto-replace handle it.

## Usage

- **Basic**: Write content with shortcodes like `[sao id="your-link-name"]`. Clicks are tracked automatically.
- **View Stats**: Check the settings page for click counts.
- **Pro Upgrade**: For A/B variants (e.g., two affiliate links tested automatically) and detailed analytics.
