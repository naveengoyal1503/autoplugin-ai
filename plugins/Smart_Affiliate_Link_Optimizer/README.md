# Smart Affiliate Link Optimizer

## Features

- **Auto-detect and cloak affiliate links** in posts and pages for better tracking and security.
- **Click tracking dashboard** with real-time stats on link performance.
- **Shortcode support**: Use `[saol url="https://affiliate-link.com"]Your Link Text[/saol]`.
- **Conversion booster**: Pretty "Click Here for Deal" buttons replace raw links.
- **Freemium model ready**: Premium features like A/B testing and geo-targeting via add-on.
- **Lightweight and self-contained**: No external dependencies, GDPR-compliant tracking.

## Installation

1. Download the plugin ZIP.
2. In WordPress admin, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate.
4. Visit **Settings > Affiliate Optimizer** to manage links.

## Setup

1. Activate the plugin.
2. Add affiliate links to posts; they auto-cloak if detected (Amazon, ClickBank, etc.).
3. Use shortcodes manually: `[saol url="YOUR_AFFILIATE_LINK"]Click for Deal[/saol]`.
4. View stats in the admin dashboard.

## Usage

- **Automatic**: Paste affiliate URLs in content; plugin replaces with tracked shortcodes on frontend.
- **Manual Shortcodes**: Embed anywhere with custom text.
- **Dashboard**: Manage, delete links, and check clicks.
- **Monetization Tip**: Promote your premium version on your site with WooCommerce.
