# SmartAffiliate Booster

SmartAffiliate Booster is a WordPress plugin designed to help bloggers and affiliate marketers automatically optimize their affiliate links for higher conversions and revenue. By cloaking affiliate URLs, inserting dynamic coupon codes, and presenting context-aware deal popups, it improves user engagement and increases commission potential.

## Features

- Automatically detects affiliate links from popular networks in your content.
- Cloaks affiliate URLs to provide cleaner, trackable redirection links.
- Injects clickable coupon triggers next to affiliate links to entice buyers.
- Displays contextual popups with discount codes relevant to the affiliate.
- Simple setup with no coding required.
- Admin settings page describing functionality.

## Installation

1. Upload the `smartaffiliatebooster.php` file to your WordPress `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. No additional setup needed for the free version.

## Setup

- Currently, affiliate domains and coupon texts are preset (Amazon, eBay, ShareASale).
- In future versions, customization will be possible via the settings page.

## Usage

- Add affiliate links for supported domains (e.g., amazon.com) as usual in your posts.
- The plugin will automatically cloak links and append a `[Special Deal]` coupon trigger next to them.
- Visitors who click the coupon trigger will see a popup displaying current coupon codes.
- Clicking on the cloaked links redirects visitors through a trackable URL enhancing affiliate link management.

## Monetization

The plugin follows a freemium model:

- Basic features are free.
- Premium subscriptions unlock analytics, multi-network integration, dynamic coupon fetching, and popup customization.
