## Installation

1. Download the plugin ZIP.
2. In WordPress Admin: Plugins > Add New > Upload Plugin.
3. Activate "AI Content Profit Protector".
4. Go to Settings > AI Content Protector to start scanning.

## Setup

1. Paste AI-generated content into the textarea.
2. Click **Scan for AI Detection & Profit Score**.
3. Review scores and tips (e.g., add affiliate links for revenue[1]).
4. Upgrade to Pro for humanization via button.

**Pro Activation**: Purchase at example.com/pro, enter license key in settings.
