# ContentROI Tracker

## Description

ContentROI Tracker is a powerful WordPress analytics plugin designed to help bloggers, affiliate marketers, and content creators maximize their monetization efforts. Track affiliate link clicks, measure sponsored content performance, and analyze ad revenue attribution all from one unified dashboard.

## Features

- **Affiliate Link Tracking**: Monitor clicks on all your affiliate links with detailed metrics
- **Click Analytics**: Track user behavior, referral sources, and device information
- **Performance Dashboard**: Visual overview of your top-performing links and content
- **Conversion Tracking**: Log and track conversions from affiliate sales and sponsored content
- **Multi-Link Management**: Handle unlimited tracked links across your site
- **ROI Calculations**: See which monetization strategies deliver the best returns
- **Freemium Model**: Basic tracking free, premium features at $9.99/month
- **Privacy Compliant**: GDPR-friendly IP anonymization options

## Installation

1. Download the plugin ZIP file
2. Go to WordPress Admin Dashboard → Plugins → Add New
3. Click "Upload Plugin" and select the ZIP file
4. Click "Install Now"
5. Activate the plugin from the Plugins page

## Setup

1. Navigate to **ContentROI** in the admin menu
2. Go to **Settings** to configure tracking preferences
3. Enable tracking for your site
4. Begin adding tracked links to your content

## Usage

### Adding Tracked Links

Wrap affiliate or sponsored links with the tracking class:

html
<a href="https://example.com/product" class="contentroi-track" data-link-id="product-123">Product Link</a>


### Viewing Performance

- Visit **ContentROI → Dashboard** to see overall statistics
- Check **ContentROI → Tracked Links** for individual link performance
- Premium users get AI-powered recommendations for optimization
