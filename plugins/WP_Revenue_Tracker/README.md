# WP Revenue Tracker

Track and visualize your WordPress site's revenue from ads, affiliate links, digital product sales, and sponsored content.

## Features
- Add and manage revenue entries by source (ads, affiliate, digital sales, sponsored content)
- Visualize revenue data with interactive charts
- View revenue history in a table
- Simple and intuitive dashboard

## Installation
1. Upload the plugin folder to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the Revenue Tracker dashboard from the main admin menu

## Setup
- No additional setup required. Start adding revenue entries right away.

## Usage
- Go to Revenue Tracker in your WordPress admin
- Add new revenue entries using the form
- View your revenue data in the chart and table
- Track your monetization efforts over time

## Requirements
- WordPress 5.0 or higher
- PHP 7.0 or higher
