# WP Revenue Tracker

Track and visualize your WordPress site's revenue from ads, affiliate links, digital products, and memberships in one dashboard.

## Features
- Add and manage revenue entries by type (ads, affiliate, digital product, membership)
- View total revenue and detailed history
- Simple, intuitive dashboard

## Installation
1. Download the plugin file.
2. Upload to your WordPress site via Plugins > Add New > Upload Plugin.
3. Activate the plugin.

## Setup
- Go to the Revenue Tracker menu in your WordPress admin.
- Start adding revenue entries.

## Usage
- Use the form to add new revenue entries.
- View your total revenue and history in the dashboard.
- Export data for further analysis (premium feature).
