# WP Revenue Tracker

A WordPress plugin that tracks and visualizes revenue from ads, affiliate links, and digital product sales in a unified dashboard.

## Features
- Track revenue from ads, affiliate links, and digital product sales
- Unified dashboard to view all revenue data
- Shortcode to display revenue summary on any page or post
- Easy to use and integrate

## Installation
1. Upload the `wp-revenue-tracker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the 'Revenue Tracker' menu in the admin panel to start tracking revenue

## Setup
- Add revenue entries through the admin dashboard
- Use the shortcode `[revenue_summary type="all"]` to display total revenue on any page or post
- Use `[revenue_summary type="ads"]`, `[revenue_summary type="affiliate"]`, or `[revenue_summary type="digital_product"]` to display revenue by type

## Usage
- Regularly update your revenue data in the dashboard
- Use the shortcode to showcase your earnings on your site
- Analyze your revenue trends and optimize your monetization strategies
