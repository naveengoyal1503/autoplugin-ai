# WP Revenue Tracker

Track and visualize revenue from ads, affiliate links, and digital product sales in one dashboard.

## Features
- Track revenue from ads, affiliate links, and digital products
- Visualize revenue data with charts
- Easy-to-use dashboard

## Installation
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Revenue Tracker dashboard to start tracking

## Setup
- No additional setup required

## Usage
- Use the form in the dashboard to add revenue data
- View revenue charts in the dashboard
