# WP Revenue Tracker

Track and visualize all your WordPress revenue streams in one dashboard.

## Features
- Add and track revenue from ads, affiliate links, digital sales, and memberships
- Visualize revenue with interactive charts
- View revenue history in a table
- Easy-to-use dashboard

## Installation
1. Upload the plugin folder to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Revenue Tracker' in your admin menu

## Setup
- No additional setup required. Start adding revenue entries right away.

## Usage
- Use the form to add new revenue entries
- View your revenue history and charts in the dashboard

## Requirements
- WordPress 5.0 or higher
- JavaScript enabled in your browser
