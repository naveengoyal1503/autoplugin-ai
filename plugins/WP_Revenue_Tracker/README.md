# WP Revenue Tracker

Track and visualize revenue from ads, affiliate links, digital product sales, and memberships in a unified dashboard.

## Features
- Track revenue from multiple sources (ads, affiliate, digital products, memberships)
- Visualize revenue data with interactive charts
- Simple and intuitive dashboard
- Export data (premium feature)
- Multi-site support (premium feature)

## Installation
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the dashboard under 'Revenue Tracker' in the admin menu

## Setup
- No additional setup required. Start tracking revenue by entering data in the dashboard.

## Usage
- Select revenue type, enter amount and date, then click 'Save Revenue'
