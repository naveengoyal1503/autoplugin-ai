# WP Revenue Tracker

Track and visualize revenue from ads, affiliate links, and digital product sales in a unified dashboard.

## Features
- Add revenue entries for ads, affiliate links, and digital products
- Visualize revenue data with interactive charts
- Easy-to-use admin interface

## Installation
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the dashboard from the 'Revenue Tracker' menu

## Setup
- No additional setup required

## Usage
- Add revenue entries using the form in the dashboard
- View revenue charts and analyze your earnings

## Requirements
- WordPress 5.0 or higher
- PHP 7.0 or higher
