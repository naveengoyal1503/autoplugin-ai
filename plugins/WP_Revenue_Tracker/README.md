# WP Revenue Tracker

Track and visualize revenue from multiple sources including ads, affiliate links, digital sales, and donations.

## Features
- Add revenue entries with source, amount, and date
- Visualize revenue data with interactive charts
- Easy-to-use admin interface

## Installation
1. Upload the plugin file to your WordPress plugins directory (`/wp-content/plugins/`)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the Revenue Tracker from the admin sidebar

## Setup
- No additional setup required. Start adding revenue data immediately.

## Usage
- Go to Revenue Tracker in the admin menu
- Fill in the source, amount, and date
- Click 'Save Revenue'
- View your revenue chart below
