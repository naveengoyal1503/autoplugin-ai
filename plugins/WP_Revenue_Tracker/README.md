# WP Revenue Tracker

Track and visualize your WordPress site's revenue from ads, affiliate links, and digital product sales in one dashboard.

## Features
- Add revenue entries for ads, affiliate links, or digital products
- Visualize revenue data with interactive charts
- Easy-to-use admin interface

## Installation
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the dashboard under 'Revenue Tracker' in the admin menu

## Setup
- No additional setup required. Start adding revenue data right away.

## Usage
- Go to Revenue Tracker in your admin panel
- Fill in the date, source, and amount
- Click 'Add Revenue' to record your earnings
- View your revenue chart for insights

## Requirements
- WordPress 5.0 or higher
- JavaScript enabled in your browser
