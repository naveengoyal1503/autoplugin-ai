# WP Revenue Tracker

Track and visualize revenue from ads, affiliate links, digital sales, and memberships in one dashboard.

## Features
- Add revenue entries for ads, affiliate, digital sales, and memberships
- Visualize revenue data with charts
- Simple and intuitive dashboard

## Installation
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the 'Revenue Tracker' menu to start tracking

## Setup
- No additional setup required. Start adding revenue entries right away.

## Usage
- Use the form to add new revenue entries
- View charts to see your revenue breakdown

## Requirements
- WordPress 5.0 or higher
- PHP 7.0 or higher
