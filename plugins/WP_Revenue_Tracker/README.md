# WP Revenue Tracker

Track and visualize your WordPress site's revenue from ads, affiliate links, digital products, and memberships.

## Features
- Add and track revenue from multiple sources
- Visualize revenue data with interactive charts
- Easy-to-use dashboard

## Installation
1. Upload the plugin folder to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the dashboard under 'Revenue Tracker' in the admin menu

## Setup
- No additional setup required. Start adding revenue data right away.

## Usage
- Use the form to enter revenue details (type, amount, date)
- View your revenue trends in the chart
- Export or analyze data as needed

## Requirements
- WordPress 5.0+
- JavaScript enabled in admin
