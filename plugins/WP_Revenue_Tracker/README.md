# WP Revenue Tracker

Track and visualize revenue from ads, affiliate links, and digital product sales on your WordPress site.

## Features
- Track revenue from multiple sources (ads, affiliate, products)
- Visualize revenue data with interactive charts
- Easy-to-use admin interface

## Installation
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Revenue Tracker' in the admin menu to start tracking

## Setup
- Enter your revenue data for each source in the admin panel
- View your revenue chart on the same page

## Usage
- Update your revenue data regularly to keep your charts up to date
- Use the data to optimize your monetization strategies
