# WP Revenue Tracker

Track and visualize revenue from multiple monetization sources including ads, affiliate links, digital product sales, and sponsored content.

## Features
- Add revenue entries for different sources
- Visualize revenue data with interactive charts
- Easy-to-use admin interface

## Installation
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the Revenue Tracker from the admin menu

## Setup
- No additional setup required. Start adding revenue data from the admin interface.

## Usage
- Go to the Revenue Tracker menu in your WordPress admin
- Fill in the source, amount, and date for each revenue entry
- Click 'Save Revenue' to record the entry
- View your revenue chart for visual insights

## Requirements
- WordPress 5.0 or higher
- JavaScript enabled in your browser
