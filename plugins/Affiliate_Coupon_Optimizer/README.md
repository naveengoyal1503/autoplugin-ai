# Affiliate Coupon Optimizer

## Description
Automatically aggregate and display affiliate coupons to boost conversions and revenue. Customize the coupon feed and optimize coupon offers for your audience with a simple shortcode.

## Features
- Automatic hourly coupon aggregation from affiliate sources (via API key)
- Display coupons with code, expiry, and tracked affiliate links
- Simple shortcode `[aco_coupons]` to show coupons anywhere
- Admin settings page to enter coupon API key
- Filters out expired coupons automatically
- Responsive and clean coupon display
- Freemium-ready design for easy extension with premium analytics and AI optimization

## Installation
1. Upload the plugin PHP file to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Coupon Optimizer' settings in the admin menu
4. Enter your affiliate coupon API key and save settings
5. Coupons will be fetched automatically on activation and hourly thereafter

## Usage
- Place the shortcode `[aco_coupons]` in any post, page, or widget to display the latest coupons
- Coupons show title, code, expiry date, and affiliate link
- Customize CSS in your theme or via a page builder as needed
