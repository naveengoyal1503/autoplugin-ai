# WP Smart Affiliate Coupons

## Description
WP Smart Affiliate Coupons lets you create, manage, and display affiliate coupon codes dynamically. It helps affiliate marketers and ecommerce site owners boost conversions by showing attractive coupon codes with affiliate links.

## Features
- Custom post type for easy coupon management
- Add coupon code, affiliate URL, and optional expiration date
- Shortcode `[affiliate_coupon id="coupon_id"]` to display coupon anywhere
- Automatically highlights expired coupons
- Clean, responsive design

## Installation
1. Upload the plugin file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You will see a new 'Affiliate Coupons' menu in your WordPress admin.

## Setup
1. Go to Affiliate Coupons > Add New.
2. Enter a coupon title.
3. Fill in the coupon code, affiliate URL, and optional expiry date.
4. Publish your coupon.

## Usage
To display a coupon in posts, pages, or widgets, use the shortcode:


[affiliate_coupon id="123"]


Replace `123` with the actual coupon post ID.

The coupon will show the code and a clickable button linking to your affiliate URL.
