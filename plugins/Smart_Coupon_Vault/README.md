# Smart Coupon Vault

## Description
**Smart Coupon Vault** is an innovative WordPress plugin that helps you create, manage, and display personalized coupon codes to boost affiliate conversions and monetize your site. Generate unique discounts, track usage, and embed affiliate links effortlessly.[1][2]

## Features
- **One-click coupon generation**: Create unique promo codes with descriptions and affiliate tracking.
- **Shortcode support**: Display coupons anywhere with `[scv_coupon_display]`.
- **Admin dashboard**: Manage all coupons, view usage stats.
- **Affiliate-ready**: Auto-generates trackable affiliate links.
- **Freemium model**: Premium unlocks AI-powered generation (OpenAI integration), unlimited coupons, analytics, and white-labeling.
- **Conversion booster**: Positions your site as a deal hub, increasing reader loyalty and revenue.[1][2]

## Installation
1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Coupon Vault** to configure.
4. Use shortcode `[scv_coupon_display]` in posts/pages.

## Setup
1. **Basic Setup**: Generate coupons directly from the admin panel.
2. **Display**: Add shortcode to any post, page, or widget.
3. **Premium API**: Enter your AI API key (e.g., OpenAI) for smart generation.
4. **Customization**: Style via CSS targeting `.scv-coupon` class.

## Usage
- **Generate**: Click "Generate New Coupon" in admin, input brand/discount.
- **Embed**: Use `[scv_coupon_display]` for responsive coupon boxes with affiliate links.
- **Track**: Usage counts update automatically (premium has advanced analytics).
- **Monetize**: Partner with brands for custom codes, earn via affiliates.[1][2][3]
