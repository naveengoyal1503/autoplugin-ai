# Smart Coupon Vault

## Features

- **Auto-fetch Coupons**: Aggregates discount codes from affiliate partners (demo in free; real APIs in Pro).
- **Shortcode Integration**: Easy `[smart_coupon_vault category="hosting" limit="5"]` embedding.
- **Click Tracking**: Monitors affiliate link performance (Pro).
- **Personalization**: AI-driven recommendations based on user behavior (Pro).
- **Responsive Design**: Mobile-friendly coupon displays.
- **Freemium Model**: Free core features; Pro unlocks unlimited sources, analytics, and automation.

## Installation

1. Upload the `smart-coupon-vault.php` file to `/wp-content/plugins/smart-coupon-vault/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[smart_coupon_vault]` in any post/page.

## Setup

1. Go to **Settings > Smart Coupon Vault** to configure API keys (Pro feature).
2. Add shortcodes to pages/posts for coupon displays.
3. Customize categories like 'hosting', 'themes', 'plugins'.

## Usage

- **Basic Shortcode**: `[smart_coupon_vault]` - Shows 10 general coupons.
- **Filtered**: `[smart_coupon_vault category="hosting" limit="3"]`.
- **Pro Features**: Real-time API fetching, analytics dashboard, custom branding.
