# Smart Coupon Vault

## Features

- **Easy Coupon Management**: Add, edit, and delete exclusive coupons directly from WordPress admin.
- **Shortcode Display**: Use `[scv_coupon_display]` to showcase coupons on any page or post.
- **AI Coupon Generator**: Generate unique coupon codes and descriptions (premium simulates basic AI; full AI in pro version).
- **Affiliate-Ready**: Track clicks via affiliate links with conversion-boosting designs.
- **Responsive Design**: Mobile-friendly coupon cards that increase user engagement.
- **Freemium Upsell**: Free core features; premium adds unlimited coupons, real AI integration, analytics, and white-labeling.

## Installation

1. Download the plugin ZIP file.
2. In your WordPress admin, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate **Smart Coupon Vault**.
4. Navigate to **Settings > Coupon Vault** to configure.

## Setup

1. **Enter AI API Key** (optional for basic): Get from OpenAI or similar for advanced generation (premium feature).
2. **Add Coupons**: Click "Add Coupon", enter code, description, and affiliate link.
3. **Generate Coupons**: Use the AI prompt field to create new ones automatically.
4. **Save Settings**.

## Usage

- **Display Coupons**: Add `[scv_coupon_display]` shortcode to any post, page, or widget.
- **Customize**: Style via CSS classes like `.coupon-card`, `.coupon-btn`.
- **Monetize**: Partner with brands for custom codes, embed affiliate links, and watch conversions grow.
- **Premium Upgrade**: For analytics, unlimited storage, and full AI, purchase at [premium link].
