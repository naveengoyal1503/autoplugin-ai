# SmartCouponPro

SmartCouponPro is a unique WordPress plugin that enables site owners to create and display AI-inspired smart coupons and dynamic discounts tailored to visitor behavior. It helps increase affiliate earnings and conversion rates by offering personalized deals.

## Features

- Easy shortcode to display smart coupons anywhere on your site.
- AI-powered logic to assign tailored coupon codes dynamically (mock implementation).
- Different coupon offers based on simple visitor targeting.
- Click tracking to monitor coupon usage.
- Clipboard copy button with redirect to affiliate or shop links.
- Lightweight and self-contained single PHP file.

## Installation

1. Download `smartcouponpro.php` plugin file.
2. Upload the file to your WordPress `/wp-content/plugins/` directory.
3. Go to the WordPress admin dashboard, navigate to Plugins.
4. Activate the "SmartCouponPro" plugin.

## Setup

1. Insert the shortcode `[smartcouponpro_display]` into any post or page where you want to show the coupon.
2. The plugin will randomly assign one of two sample coupons per visitor (expandable for your custom AI logic).
3. Optionally customize coupon offers in the plugin code.

## Usage

- Use `[smartcouponpro_display]` shortcode to show a coupon box.
- Visitors can copy the coupon code with a click and get redirected to your affiliate shop or product page.
- Track coupon code clicks through WordPress options (for later extension, integrate with analytics).
