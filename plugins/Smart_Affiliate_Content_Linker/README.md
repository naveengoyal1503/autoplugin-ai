# Smart Affiliate Content Linker

Automatically convert product mentions in your WordPress posts into affiliate links with comprehensive tracking, analytics, and revenue management.

## Features

- **Automatic Link Detection**: Intelligently identify product mentions in your content
- **One-Click Conversion**: Convert mentions into trackable affiliate links with a single click
- **Click Tracking**: Monitor every click on your affiliate links
- **Revenue Analytics**: Track conversions and revenue per link
- **Commission Management**: Set custom commission rates for different affiliate programs
- **Dashboard Overview**: View total links, clicks, conversions, and revenue at a glance
- **Bulk Operations**: Edit multiple links simultaneously
- **Performance Reports**: Detailed analytics on top-performing products
- **Multiple Affiliate Programs**: Manage links from different affiliate networks

## Installation

1. Download the plugin ZIP file
2. Go to WordPress Admin > Plugins > Add New
3. Click "Upload Plugin" and select the ZIP file
4. Click "Install Now" then "Activate Plugin"
5. Navigate to the "Affiliate Linker" menu in your WordPress dashboard

## Setup

1. Go to Settings and add your API key for affiliate networks
2. Configure default commission rates
3. Set up your affiliate program IDs for each network
4. Enable click tracking and analytics

## Usage

### Creating Affiliate Links

1. Open any blog post in the editor
2. Go to the "Affiliate Linker" sidebar
3. Enter the product name you mention
4. Paste your affiliate URL
5. Set the commission rate
6. Click "Add Link"

### Viewing Analytics

1. Go to Dashboard to see overall statistics
2. Visit Analytics tab for detailed performance metrics
3. Click on any product to see detailed click and conversion history
4. Export reports as CSV

### Managing Links

1. Navigate to "Manage Links"
2. View all affiliate links across your site
3. Edit, disable, or delete links as needed
4. Filter by post, product, or affiliate network
