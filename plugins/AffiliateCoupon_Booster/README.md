# AffiliateCoupon Booster

**AffiliateCoupon Booster** is a WordPress plugin that automates affiliate coupon management and increases conversion rates by delivering timely, personalized coupon notifications and displaying easy-to-use coupon listings.

## Features

- Create, edit, and manage unlimited affiliate coupons with start/end dates.
- Show coupon lists on any page via shortcode `[affcouponbooster]`.
- Popup notifications show coupon deals after user engagement delay.
- Affiliate link cloaking with clean redirect URLs.
- Schedule coupon validity dates to automate campaigns.
- Lightweight with built-in styles and scripts; no dependency overload.
- Easy-to-use admin interface.

## Installation

1. Upload the `affiliatecouponbooster.php` file to your WordPress `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. A new menu item 'AffiliateCoupon Booster' will appear in the dashboard.

## Setup

1. Navigate to the 'AffiliateCoupon Booster' admin menu.
2. Add new coupons providing title, description, affiliate URL, coupon code, validity dates, and a popup message.
3. Save coupons.

## Usage

- To display active coupons on any post/page, add the shortcode `[affcouponbooster]` where you want the coupon list to appear.
- Visitors will see popup coupon notifications automatically after a delay.
- Coupon links are automatically cloaked and redirect through the plugin for tracking and clean URLs.

## Notes

- Ensure your affiliate URLs are accurate to not lose commissions.
- Compatible with most themes.
- The plugin currently is a lightweight starter version and will be enhanced with premium add-ons for analytics and advanced targeting.
