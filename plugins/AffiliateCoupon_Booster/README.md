# AffiliateCoupon Booster

AffiliateCoupon Booster automatically curates and displays affiliate coupons with dynamic expiration countdowns and click tracking to boost affiliate conversions.

## Features

- Auto-display curated affiliate coupons via shortcode `[affiliate_coupons]`
- Shows coupon code, title, and expiration countdown
- Tracks clicks on coupon links via AJAX for conversion insights
- Responsive, clean design with easy customization
- Free core features with planned premium upgrades for analytics dashboard and multi-network affiliate integration

## Installation

1. Upload `affiliatecoupon-booster.php` to your WordPress `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place the shortcode `[affiliate_coupons]` on any page or post where you want coupons displayed.

## Setup

- Coupons are currently statically defined in plugin code for demo. Premium versions will support admin UI to add/manage coupons.
- Each coupon includes a title, code, affiliate URL, and expiry date.

## Usage

- Add `[affiliate_coupons]` shortcode to embed coupon listings.
- Visitors see coupon offers with expiration days countdown.
- When they click 'Use Coupon', their click is tracked, and the affiliate link opens in a new tab.
