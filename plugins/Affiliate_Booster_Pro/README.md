# Affiliate Booster Pro

Affiliate Booster Pro is a powerful WordPress plugin designed to help bloggers, content creators, and e-commerce sites build and manage an affiliate marketing program with ease. Its simple yet effective features enable site owners to generate additional revenue by rewarding affiliates for referral traffic.

## Features

- Add and manage affiliate profiles with name and email
- Generate cloaked affiliate links via a shortcode for easy sharing
- Automatic commission tracking on referred visits (example commission: $1 per visit)
- Admin dashboard displays affiliates and their total unpaid commissions
- Mark commissions as paid to keep track of payouts
- Lightweight, self-contained single PHP file plugin

## Installation

1. Upload the `affiliate-booster-pro.php` file to your WordPress `/wp-content/plugins/` directory.
2. Activate the plugin via the Plugins menu in WordPress.
3. Access the "Affiliate Booster" admin menu for configuration.

## Setup

1. Navigate to the Affiliate Booster admin page.
2. Add new affiliates with their name and email.
3. Share cloaked affiliate links generated by shortcode `[abp_affiliate_link id="affiliate_id"]`.
4. Monitor commissions on the admin page.

## Usage

- To create affiliate links on posts or pages, use the shortcode replacing `affiliate_id` with an affiliate's ID:

  
  [abp_affiliate_link id="aff_xxxxxxxxxxxxxxxx"]
  

- When someone visits your site with the affiliate ID in the URL query (e.g., `?ref=aff_xxxxx`), a commission of $1 is automatically credited.
- Use the admin page to view affiliates and total commissions owed.
- Mark commissions paid once you have completed payouts.
