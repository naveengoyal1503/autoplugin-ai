# Affiliate Booster Pro

**Affiliate Booster Pro** is a WordPress plugin to create, manage, and optimize an affiliate marketing program that integrates with WooCommerce and Easy Digital Downloads (EDD).

## Features

- Seamless integration with WooCommerce and EDD
- Generate affiliate referral links for logged-in users via shortcode
- Track affiliate referrals and commissions on order completion
- Set default commission rates with admin settings
- Store affiliate referral data in user meta for easy tracking
- Simple and lightweight with no external dependencies

## Installation

1. Upload the `affiliate-booster-pro.php` file to the `/wp-content/plugins/` directory.
2. Activate the plugin through the WordPress admin dashboard.
3. Navigate to **Affiliate Booster** in the admin menu to configure settings.

## Setup

- Set your default commission rate (%) for affiliates in the settings.
- Encourage users to log in and share their referral links generated by the shortcode `[affiliate_referral_link]` anywhere on your site.

## Usage

- Place the shortcode `[affiliate_referral_link]` on any page/post to display the logged-in user's unique referral URL.
- When customers complete purchases with a valid referral ID, commissions are calculated and saved.
- Monitor commissions by inspecting user meta from the database or extend the plugin to display reports.

## Monetization Model

The base plugin is free to install and use, providing essential affiliate management. Revenue can be generated by selling premium add-ons or subscriptions that introduce advanced features, such as:

- Tiered commission rates per product or campaign
- Affiliate payout management
- Detailed analytics dashboard
- Integration with email marketing

This approach targets e-commerce store owners and digital sellers who want to grow sales via affiliates.
