# ContentAI Optimizer

AI-powered content optimization and SEO enhancement plugin for WordPress.

## Features

- **Real-time Content Analysis**: Analyzes posts and pages for SEO, readability, and engagement
- **SEO Score Calculation**: Evaluates title length, meta descriptions, keyword usage, and more
- **Readability Analysis**: Uses Flesch-Kincaid scoring to measure content readability
- **Engagement Scoring**: Assesses content for images, links, structure, and multimedia
- **Smart Suggestions**: AI-powered recommendations to improve content performance
- **Dashboard Analytics**: Track optimization metrics and content improvements
- **Meta Box Integration**: Analyze content directly from the WordPress post editor
- **Freemium Model**: 5 free analyses per month with premium upgrades available

## Installation

1. Download the plugin file
2. Go to WordPress Admin > Plugins > Add New
3. Click "Upload Plugin" and select the downloaded file
4. Click "Install Now"
5. Activate the plugin
6. Access ContentAI Optimizer from the WordPress admin menu

## Setup

1. Navigate to ContentAI > Settings
2. Enter your API key (optional for basic features)
3. Configure analysis preferences
4. Enable or disable readability analysis
5. Save settings

## Usage

### Quick Analysis

1. Open any post or page for editing
2. Look for the "ContentAI Optimizer" meta box on the right sidebar
3. Click "Analyze Content"
4. View your scores and suggestions

### Dashboard

1. Go to ContentAI > Dashboard
2. View total analyses performed
3. Check average scores across your content
4. Monitor monthly analysis usage

### Getting Suggestions

1. Analyze your content
2. Review AI-powered suggestions
3. Implement recommended changes
4. Re-analyze to see improvements

## Scoring System

- **SEO Score (0-100)**: Based on title optimization, meta descriptions, content structure, and keyword usage
- **Readability Score (0-100)**: Measures sentence complexity and overall readability
- **Engagement Score (0-100)**: Evaluates multimedia usage, internal links, and content formatting

## Monetization

- **Free Tier**: 5 analyses per month, basic SEO suggestions
- **Premium**: Unlimited analyses, advanced AI suggestions, competitor tracking ($9.99/month)
