# GeoAffiliateLink Pro

GeoAffiliateLink Pro is a WordPress plugin that helps affiliate marketers **cloak**, **categorize**, and **geotarget** affiliate links to boost conversions by automatically redirecting visitors based on their country and inserting relevant links in your content.

---

## Features

- Cloak affiliate URLs with clean, readable slugs
- Categorize affiliate links for easy management
- Geotarget affiliate links to show region-specific URLs
- Auto-insert affiliate links into posts based on categories
- Simple admin interface to add, edit, and delete links
- Redirect visitors seamlessly to the appropriate affiliate URL
- Supports shortcode for manual placement: `[geo_affiliate_link slug="slug-name" default_url="https://default.com"]`

## Installation

1. Upload the `geoaffiliatelink-pro.php` file to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to the **GeoAffiliateLink** menu in your dashboard

## Setup

1. Add your affiliate links in the plugin settings page.
   - Provide the link name, actual URL, category keyword, and 2-letter country code (leave blank for global).
2. Save your links.
3. Use the shortcode `[geo_affiliate_link slug="your-link-slug" default_url="https://fallback-url.com"]` in posts or pages to manually add cloaked affiliate links.
4. The plugin will also automatically insert cloaked links in your content where category keywords appear.

## Usage

- Use shortcodes to place affiliate links anywhere with geotargeting.
- Manage and adjust your affiliate links easily via the admin panel.
- Benefit from automatic geolocation-based redirection and content insertion to increase affiliate conversions.
