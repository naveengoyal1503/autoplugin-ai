# WP Loyalty Points

Reward your users with loyalty points for actions like purchases, comments, and more. Let them redeem points for discounts, exclusive content, or rewards.

## Features
- Automatically reward points for purchases (WooCommerce) and comments
- Shortcode to display user points
- AJAX redemption system
- Simple database tracking

## Installation
1. Upload the plugin file to `/wp-content/plugins/`
2. Activate the plugin from the WordPress admin
3. Use the shortcode `[loyalty_points]` to display points

## Setup
- No additional setup required for basic functionality
- For WooCommerce integration, ensure WooCommerce is active

## Usage
- Points are awarded automatically for purchases and comments
- Users can redeem points via AJAX (requires custom frontend)
- Points can be viewed with the shortcode

## Notes
- Advanced features (tiered rewards, analytics, integrations) available in premium version
