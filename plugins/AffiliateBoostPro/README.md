# AffiliateBoostPro

AffiliateBoostPro is a WordPress plugin that enables site owners to create and manage their own affiliate marketing programs with real-time tracking, commission management, and referral link generation.

## Features
- Auto-track affiliate clicks via URL parameters
- Simple commission calculation (default 10%)
- Store affiliate data: clicks, conversions, and commissions
- Admin dashboard with top affiliates overview
- Shortcode to display user-specific referral link
- Cookie-based tracking for referrals
- Lightweight, fast, and extensible

## Installation
1. Upload the plugin PHP file to your WordPress `/wp-content/plugins/` directory.
2. Activate the plugin from the WordPress admin dashboard.
3. The plugin will create a custom database table for tracking affiliates.

## Setup
- No initial configuration required.
- The affiliate ID is currently mapped to WordPress user IDs.
- To track affiliate traffic, affiliates use referral URLs generated by the shortcode `[affboost_referral_link]`.

## Usage
- Place the shortcode `[affboost_referral_link]` on a page or post to show logged-in users their unique referral URLs.
- When visitors arrive with `?affid=USER_ID` in the URL, their click is counted.
- To simulate a purchase and record a commission for testing, navigate to `https://your-site.com/?affboost_purchase=SALE_AMOUNT` while an affiliate cookie is set.
- View affiliate stats on the "AffiliateBoostPro" admin menu.
