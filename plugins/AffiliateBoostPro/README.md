# AffiliateBoostPro

AffiliateBoostPro is a WordPress plugin designed to automate and simplify your affiliate marketing efforts with smart link cloaking, dynamic coupon integration, and detailed performance tracking.

## Features

- Create custom short affiliate links with unique slugs
- Automatic click tracking and analytics
- Easy insertion of affiliate links via shortcode
- Admin dashboard to manage and monitor all affiliate links
- Clean, SEO-friendly cloaked URLs
- Lightweight, single-file plugin for easy maintenance

## Installation

1. Upload the `affiliateboostpro.php` file to your WordPress plugins directory `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the AffiliateBoostPro admin menu to start creating affiliate links.

## Setup

- In the AffiliateBoostPro admin page, create custom slugs and associate them with your target affiliate URLs.
- Use the shortcode `[abp_affiliate_link slug="your-slug"]` to insert cloaked affiliate links into posts or pages.
