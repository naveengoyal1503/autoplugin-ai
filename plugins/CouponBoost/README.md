# CouponBoost

**CouponBoost** is a WordPress plugin that dynamically generates and displays personalized coupon codes and exclusive deals using AI-driven analytics techniques to increase affiliate conversions and user engagement.

## Features

- Dynamic display of coupons adapting to visitor context (time-based in demo)
- Easy shortcode `[couponboost]` to embed coupon lists anywhere
- Styled, responsive coupon display
- Affiliate link integration for seamless tracking
- Simple, self-contained single PHP file plugin

## Installation

1. Download the `couponboost.php` plugin file.
2. Log in to your WordPress admin dashboard.
3. Go to **Plugins > Add New**.
4. Click **Upload Plugin**, choose the `couponboost.php` file, and click **Install Now**.
5. Activate the plugin.

## Setup

- No special setup required initially.
- To add coupon listings, insert shortcode `[couponboost]` in posts, pages, or widgets.
- Future premium versions may include admin panels to add/edit coupons and integrate AI personalization.

## Usage

- Simply insert the shortcode `[couponboost]` where you want your coupons to appear.
- Visitors will see dynamically filtered coupons optimized for conversion.
