# AI Coupon Vault

## Features

- **AI-Powered Coupon Generation**: Enter a keyword (e.g., "laptop") to instantly generate unique coupon codes, discounts, and affiliate links.
- **Shortcode Display**: Use `[ai_coupon_vault id="unique-id"]` to embed professional coupon boxes on posts/pages.
- **Admin Dashboard**: Manage settings, affiliate IDs, and generate unlimited coupons (Pro).
- **Freemium Model**: Free for 5 coupons; Pro unlocks unlimited, analytics, custom branding ($49/year).
- **Revenue Boost**: Personalized deals increase conversions for bloggers and affiliates[1][2].
- **SEO-Friendly**: Coupon pages optimize for search traffic on deals and discounts.

## Installation

1. Download and upload the plugin ZIP to your WordPress admin > Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Go to **Settings > Coupon Vault** to configure affiliate programs (JSON format: `{"amazon": "your-id"}`).

## Setup

1. In the admin page, enter affiliate program IDs.
2. Click **Generate AI Coupon** with a keyword to create deals.
3. Copy the shortcode (e.g., `[ai_coupon_vault id="abc123"]`).
4. Paste shortcode in posts/pages/blocks for display.
5. **Pro Upgrade**: Enter license key for premium features.

## Usage

- **Frontend**: Shortcodes render styled coupon boxes with code, discount, and shop link.
- **Backend**: Generate/track coupons; free limits to 5, pro unlimited.
- **Monetization Tips**: Use for affiliate sites, blogs in e-commerce/travel niches. Track clicks via pro analytics[3][4].
- **Customization**: Edit CSS in plugin styles or child theme.
