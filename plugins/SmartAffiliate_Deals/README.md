# SmartAffiliate Deals

SmartAffiliate Deals dynamically curates and displays the best affiliate coupons and deals tailored to your site's content and audience, helping WordPress bloggers and affiliate marketers maximize their earnings.

## Features

- Automatically aggregates affiliate deals from multiple RSS or JSON sources.
- Displays deals via a shortcode with AJAX-powered loading for performance.
- Caches deal data to reduce API calls and speed up page loads.
- Admin settings for affiliate ID, deal source URLs, and caching duration.
- Clean, responsive deal display styled for readability.

## Installation

1. Upload `smartaffiliate-deals.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > SmartAffiliate Deals** to configure options.

## Setup

- Enter your affiliate ID provided by your affiliate program.
- Add URLs for affiliate deal feeds (in RSS or JSON format), one URL per line.
- Set the cache time (in minutes) indicating how long deals are stored before refreshing.
- Save settings.

## Usage

- Insert the shortcode `[smart_affiliate_deals]` into any post, page, or widget where you want deals to appear.
- The plugin fetches and displays a curated list of up to 20 current deals with titles, descriptions, and optional expiry.

## Monetization

- **Freemium model:** Basic deal aggregation and display is free.
- Premium plans (future upgrades) may include advanced targeting, affiliate network integrations, and detailed analytics.
