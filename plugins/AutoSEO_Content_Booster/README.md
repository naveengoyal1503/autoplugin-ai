# AutoSEO Content Booster

AutoSEO Content Booster is a WordPress plugin that gives bloggers, content creators, and SEO professionals real-time keyword suggestions and SEO improvements directly inside the WordPress post editor.

## Features

- Instant keyword suggestion generation based on entered focus keyword
- Real-time display of SEO keyword variations and long-tail keywords
- Sidebar meta box for easy access inside WordPress post editor
- Freemium-friendly architecture, scalable for future AI-powered API integrations

## Installation

1. Upload the plugin PHP file to your WordPress `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to edit any post.
4. Locate the **AutoSEO Content Booster** box on the right sidebar.

## Setup

- No setup required for the basic free features.
- Simply enter a focus keyword and click **Get Suggestions**.
- Future premium upgrades will unlock advanced features.

## Usage

1. Open any WordPress post for editing.
2. In the **AutoSEO Content Booster** meta box, type your focus keyword.
3. Click **Get Suggestions** to see a list of SEO-friendly keyword variations.
4. Use these suggestions to enhance your post title, content, and meta descriptions.

For premium users (future updates), expect AI-driven deeper content analysis, SEO audits, and competitor insights.
