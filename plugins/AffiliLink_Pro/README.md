# AffiliLink Pro

**AffiliLink Pro** is an easy-to-use WordPress plugin designed to help bloggers, affiliate marketers, and WooCommerce store owners manage, cloak, and track their affiliate links efficiently. Boost your affiliate revenue and get detailed insights on link performance.

## Features

- Cloak affiliate links under your domain for branding and better click-through
- Track clicks per affiliate link with timestamps
- Admin dashboard with sortable list of affiliate links and click stats
- Add new affiliate links easily from the WordPress admin
- Shortcodes to insert affiliate links in posts/pages
- Integration-ready: designed to work alongside WooCommerce and popular affiliate programs
- Lightweight and fast with minimal overhead

## Installation

1. Download the `affililink-pro.php` file.
2. Upload it to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Navigate to **AffiliLink Pro** in your admin menu to start adding affiliate links.

## Setup

- Use the admin page to add your affiliate IDs/slugs and their corresponding target URLs.
- Each added affiliate ID creates a cloaked link at `https://yourdomain.com/affililink/{affiliate_id}`
- The plugin tracks each click on these links automatically.

## Usage

- Add affiliate links in your posts or pages using the shortcode:

  
  [affililink id="affiliate_id" text="Your Link Text"]
  

- Replace `affiliate_id` with the slug you set in the admin and customize `Your Link Text` to display.
- Share cloaked links to improve branding and track performance.
