# Affiliate Deal & Coupon Booster

## Description
Automatically aggregates and displays affiliate deals and coupons tailored to your site's niche, enhancing visitor engagement and boosting affiliate revenue.

## Features
- Easy API integration for affiliate deal providers
- Automatic niche-based deal aggregation
- Simple shortcode `[adacb_deals niche="your-niche"]` to display deals
- Admin settings to configure API keys and default niche
- Clean, responsive markup for seamless theme integration

## Installation
1. Upload the plugin PHP file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the WordPress Plugins menu.
3. Go to **Settings > Affiliate Deal Booster**.
4. Enter your affiliate API key and default niche keyword.
5. Save changes.

## Usage
Place the shortcode `[adacb_deals]` in any post or page to display the top deals for the default niche.

To specify a niche, use:

[adacb_deals niche="electronics"]

This will show deals related to electronics.
