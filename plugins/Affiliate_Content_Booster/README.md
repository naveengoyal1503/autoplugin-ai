# Affiliate Content Booster

## Description
Affiliate Content Booster automatically inserts affiliate links into your content with keyword matching, cloaking, and click tracking, helping bloggers and content creators increase affiliate revenue effortlessly.

## Features
- Insert affiliate links automatically by keywords
- Link cloaking for cleaner URLs and better tracking
- Basic click tracking with analytics dashboard pending (pro feature)
- Support for geolocation targeting (pro version)
- Simple admin interface to manage keywords and affiliate URLs

## Installation
1. Upload `affiliate-content-booster.php` to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Affiliate Content Booster to configure keywords and URLs

## Setup
- Add keywords and affiliate URLs (one per line) in the format:
  
  keyword|https://affiliate-link.com
  
- Save settings
- The plugin will automatically link these keywords wherever they appear in your posts

## Usage
- Write your content normally
- When a configured keyword appears, it will convert into a cloaked affiliate link
- Clicks on affiliate links are tracked internally (basic tracking)
