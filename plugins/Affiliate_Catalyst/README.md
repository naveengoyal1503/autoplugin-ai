# Affiliate Catalyst

Affiliate Catalyst is a powerful WordPress plugin that helps you create and manage your own customizable affiliate programs. Increase your sales by turning customers and fans into commission-earning promoters with automated affiliate onboarding, tiered commissions, and detailed analytics.

## Features

- Easy-to-use affiliate management dashboard
- Add affiliates with custom commission rates
- Track referrals via URL parameters and cookies
- Shortcode to generate affiliate referral links
- Store affiliate data in custom database table
- Freemium-ready foundation for premium expansions (e.g., multi-tier commissions, analytics)

## Installation

1. Upload `affiliate-catalyst.php` to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Affiliate Catalyst** in the admin menu to start adding affiliates.

## Setup

- Use the dashboard to add new affiliates with their name, email, and commission percentage.
- Share affiliate referral links generated via shortcode on your site or with affiliates.
- Track referrals by appending `?ac_ref={affiliate_id}` to your site URLs; the plugin will store referral info in a cookie for 30 days.

## Usage

- Add affiliate links anywhere on your site using shortcode:

  
  [ac_affiliate_link id="123" text="Check our products"]
  

  Replace `123` with the affiliate ID and customize link text.

- Monitor and manage affiliates in the WordPress admin under **Affiliate Catalyst**.
