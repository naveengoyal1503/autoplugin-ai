# AffiliateLink Smart Manager

AffiliateLink Smart Manager helps you cloak and optimize your affiliate links on WordPress with easy-to-use management, click tracking, and smart suggestions (premium).

## Features

- Create and manage clean, memorable affiliate link slugs
- Automatically redirect clicks to your affiliate URLs
- Track clicks on each affiliate link in the admin dashboard
- Use shortcode `[affiliate_link slug="yourslug"]` to display affiliate links elegantly
- Clean, lightweight, single-file plugin with easy installation
- Premium upgrade planned: advanced analytics, auto link suggestions, integration with affiliate networks

## Installation

1. Upload `affiliate-link-smart-manager.php` to your WordPress plugins directory or install via upload in admin.
2. Activate the plugin from the Plugins menu.
3. Visit the "AffiliateLink Manager" menu to add and manage your affiliate links.

## Setup

- Add a unique slug (short identifier) and the full target affiliate URL.
- Save the link and copy the generated affiliate URL.
- Use the shortcode `[affiliate_link slug="slugname"]` anywhere in your posts or pages to display clickable links.

## Usage

- Visitors clicking on your affiliate URL slug will be redirected to the affiliate target URL.
- Monitor clicks in the admin page to measure link performance.
- For best results, combine with your existing affiliate campaigns and content.

## Frequently Asked Questions

**Q: Can I have multiple links?**
A: Yes, you can create as many affiliate link slugs as you need.

**Q: Is the plugin compatible with all WordPress themes?**
A: Yes, it works at the WordPress core level and is theme-independent.

**Q: How is the click counted?**
A: Each redirect request increments a click counter stored in the database.

**Q: Can I use this for non-affiliate URLs?**
A: While designed for affiliate marketing, it can cloak and track any URLs.
