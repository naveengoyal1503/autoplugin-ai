# Smart Affiliate AutoLink Pro

**Automatically boosts your affiliate earnings by inserting relevant links into content using smart keyword detection.**

## Features

- **Auto-Linking**: Scans posts and pages for keywords and replaces them with your affiliate links.
- **Customizable**: Define keywords and links via simple JSON in settings.
- **Link Limits**: Free version limits to 3 links per post; Pro unlimited.
- **No-Follow & Sponsored**: All links include proper rel attributes for compliance.
- **Pro Upgrades**:
  - AI-powered keyword suggestions based on content.
  - Integrations with Amazon, ClickBank, ShareASale.
  - Click analytics and A/B testing.
  - White-label branding.

## Installation

1. Upload the `smart-affiliate-autolink` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Affiliate AutoLink** to configure keywords and links.

## Setup

1. In the settings page, enter keywords and affiliate URLs in JSON format:
   
   {"keyword":"https://your-affiliate-link.com", "another":"https://another-link.com"}
   
2. Set max links per post (default: 3).
3. Save changes. Links auto-apply to new and existing content.

## Usage

- Write content with natural keywords (e.g., "best hosting").
- Plugin auto-inserts links on frontend.
- Test on a staging site first.
