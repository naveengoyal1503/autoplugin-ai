# Affiliate Deal Booster

Affiliate Deal Booster is a WordPress plugin that automatically aggregates and displays exclusive affiliate coupons and deals relevant to your site's content, helping to boost your affiliate revenue and user engagement.

## Features

- Automatically fetches affiliate deals and coupons (simulated with sample data).
- Displays deals in a responsive, styled layout.
- Simple shortcode to insert deals anywhere: `[affiliate_deals]`.
- Internal caching reduces external requests for performance.
- Ready for future integration with multiple affiliate networks.

## Installation

1. Upload the `affiliate-deal-booster.php` file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place the shortcode `[affiliate_deals]` in any post, page, or widget where you want to display deals.

## Setup

No special configuration required out of the box. Future premium versions will include settings to add your affiliate keys and customize deal sources.

## Usage

- Use the shortcode `[affiliate_deals]` to display a list of curated affiliate deals and coupons.
- Style the deals with your theme or customize the plugin CSS.

This plugin is ideal for affiliate marketers and bloggers who want to provide their visitors with up-to-date money-saving coupon deals automatically, increasing conversions and commissions.
