# Affiliate Deal Booster

## Features

- Automatically displays affiliate coupon deals from a configurable feed URL
- Limits displayed deals with settable maximum number
- Basic click tracking via AJAX to help measure user engagement
- Smart shortcode `[adb_deals]` for easy insertion into posts and pages
- Admin settings panel for affiliate ID, coupon feed URL, and display limit

## Installation

1. Upload `affiliate-deal-booster.php` to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Affiliate Deal Booster settings in the WordPress admin sidebar.
4. Enter your Affiliate ID and your external coupon feed URL (JSON format recommended).
5. Set the max number of deals to show and save settings.

## Setup

- Ensure you have an affiliate coupon feed URL in JSON format available.
- Configure the feed URL in the plugin settings to automatically fetch deals.
- Use `[adb_deals]` shortcode inside any post or page to display deals.

## Usage

- Place the shortcode `[adb_deals]` where you want deal blocks to appear.
- Monitor clicks and conversions by reviewing your affiliate dashboard or logs.
- Consider upgrading to premium version for advanced targeting, analytics, and multi-network support (future).
