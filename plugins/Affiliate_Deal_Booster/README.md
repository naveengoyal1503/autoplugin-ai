# Affiliate Deal Booster

Affiliate Deal Booster automatically displays relevant affiliate coupons and deals beneath your posts, helping you increase affiliate conversions and provide extra value to your visitors.

## Features
- Automatically extracts keywords from post titles to find relevant deals.
- Displays exclusive affiliate coupons related to your content.
- Simple settings page to configure your affiliate ID.
- Lightweight and compatible with most themes.
- Freemium-ready: basic deals display included, premium upgrade can add network integration and analytics.

## Installation
1. Upload the `affiliate-deal-booster.php` file to your WordPress plugins directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Navigate to Settings > Affiliate Deal Booster to configure your affiliate ID.

## Setup
- Enter your affiliate ID to append it to affiliate URLs for tracking.
- Optionally customize the deals if you upgrade to premium.

## Usage
- Publish posts as usual. Relevant affiliate deals will automatically appear below the post content.
- Use this plugin especially on affiliate marketing or coupon blogs to boost revenue.
