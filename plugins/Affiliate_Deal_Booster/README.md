# Affiliate Deal Booster

Affiliate Deal Booster is a WordPress plugin designed to help bloggers and affiliate marketers curate, display, and track affiliate coupon deals tailored to their niche. By showcasing high-converting deals with easy coupon code copying and click tracking, it helps optimize affiliate revenue.

---

## Features

- Display a curated list of affiliate deals and coupons
- Automatic expiration of outdated deals
- Click tracking on affiliate links to measure performance
- Easy copy coupon code buttons
- Responsive front-end display via shortcode `[affiliate_deals]`
- Lightweight, single-file plugin for easy installation

## Installation

1. Download the plugin ZIP file.
2. In your WordPress admin dashboard, go to Plugins > Add New > Upload Plugin.
3. Upload the ZIP and activate the plugin.
4. By default, sample deals are added. Customize via code or extend for an admin UI.

## Setup

- Use shortcode `[affiliate_deals]` in any post or page to display the affiliate deals list.
- Modify deals by changing the plugin code or extend to add custom admin settings.

## Usage

- Visitors see a list of affiliate deals.
- Clicking on a deal link tracks the click for analytics.
- Clicking "Copy Code" copies the coupon code to the clipboard for easy use.
- Monitor the performance by adding analytics to the plugin code or extending it with an admin dashboard.
