# Affiliate Deal Booster

## Description
Automatically aggregates and displays niche-specific affiliate coupons and deals with real-time tracking links to boost user engagement and maximize affiliate income.

## Features
- Automatically generate affiliate deals based on your niche keyword.
- Display attractive coupon links with expiration dates.
- Easy shortcode insertion: use `[affiliate_deals]` to show deals.
- Settings page for affiliate ID and niche keyword customization.
- Clean, responsive deal listing.
- Built-in affiliate link tracking and cloaking in premium versions (planned).

## Installation
1. Upload `affiliate-deal-booster.php` to your `/wp-content/plugins/` directory.
2. Activate the plugin through the WordPress 'Plugins' menu.
3. Navigate to 'Affiliate Deal Booster' in the WordPress admin menu to enter your Affiliate ID and Niche Keyword.

## Setup
- Enter your affiliate ID provided by your affiliate program.
- Enter a niche keyword relevant to your blog or audience (example: "fitness", "tech gadgets").

## Usage
- Add the shortcode `[affiliate_deals]` anywhere in your posts, pages, or widgets.
- The plugin will display curated deals relevant to your niche, linking through your affiliate ID for commissions.

## Notes
- This basic version shows mock deals for demonstration.
- Premium versions will integrate real deal APIs, advanced analytics, and automated affiliate link management for increased profitability.
