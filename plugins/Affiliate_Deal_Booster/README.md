# Affiliate Deal Booster

## Description
Affiliate Deal Booster lets WordPress site owners create, manage, and display affiliate deals with custom coupon codes and affiliate URLs to maximize affiliate marketing revenue.

## Features
- Add unlimited affiliate deals with titles, URLs, coupons, and descriptions.
- Display deals anywhere using the shortcode `[affiliate_deals]`.
- Simple admin interface to add, view, and delete deals.
- Deals presented with clear calls to action and coupon codes.

## Installation
1. Upload the plugin file to your `/wp-content/plugins/` directory or install via WordPress admin.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the 'Affiliate Deals' menu in the WordPress dashboard to add and manage your deals.

## Setup
- Go to **Affiliate Deals** in your WordPress admin dashboard.
- Fill out the form to add new deals including the title, affiliate URL, optional coupon code, and description.
- Save your deals.

## Usage
- Insert the shortcode `[affiliate_deals]` within any page, post, or widget to display the list of affiliate deals.
- Visitors will see deals with clickable links and coupon codes where applicable.

Optimize your affiliate marketing by showcasing deals clearly and attractively to increase conversions and monetization.
