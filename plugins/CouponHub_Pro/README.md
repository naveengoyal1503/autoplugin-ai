# CouponHub Pro

**Version:** 1.0

**Description:**
CouponHub Pro is a WordPress plugin designed for affiliate marketers, bloggers, and eCommerce sites to easily create and manage a centralized, customizable coupon and deal aggregator. It allows you to display attractive coupons and affiliate offers, boosting user engagement and increasing affiliate revenue.

---

## Features

- Admin interface to add, edit, and remove coupons with title, code, affiliate URL, and description.
- Frontend shortcode `[couponhubpro]` to display all active coupons and deals.
- Widget support to place coupon lists in sidebars or footer areas.
- Easy affiliate link insertion with coupon codes that copy to clipboard on click.
- Responsive and clean design for any theme.
- Freemium-ready: offers baseline coupon management with potential for pro extensions.

---

## Installation

1. Upload the `couponhubpro.php` file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the WordPress 'Plugins' menu.
3. You will see a new admin menu called `CouponHub Pro`.

---

## Setup

1. Navigate to the `CouponHub Pro` admin page.
2. Use the form to add coupon details including title, coupon code, affiliate URL, and a short description.
3. Click **Save Coupons** to store your entries.

---

## Usage

- Insert `[couponhubpro]` shortcode anywhere in posts or pages to display your coupons.
- Use the **CouponHub Pro Widget** in Appearance > Widgets to display coupons in sidebars or footers.
- Clicking the coupon link copies the coupon code to user clipboard and opens the affiliate link in a new tab.
