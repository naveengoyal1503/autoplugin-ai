# ContentAffiliateBooster

Intelligently monetize your WordPress blog with contextual affiliate link insertion, geolocation targeting, and real-time performance analytics.

## Features

- **Auto-Link Insertion**: Automatically finds keywords in your posts and replaces them with your affiliate links
- **Multi-Program Management**: Manage affiliate links from multiple programs (Amazon Associates, affiliate networks, etc.)
- **Click Tracking**: Real-time tracking of affiliate link clicks with detailed analytics
- **Performance Analytics**: Monitor conversion rates, earnings, and ROI by program
- **Easy Setup**: User-friendly dashboard with no coding required
- **Dashboard Widgets**: Quick stats on total links, clicks, and earnings
- **Customizable Colors**: Style affiliate links to match your site's branding

## Installation

1. Download the plugin and extract to `/wp-content/plugins/`
2. Activate the plugin in WordPress admin
3. Navigate to **Affiliate Booster** in the admin menu
4. Add your first affiliate link and keyword
5. Enable auto-linking in Settings

## Setup

### Adding Affiliate Links

1. Go to **Affiliate Booster > Affiliate Links**
2. Enter the keyword (e.g., "WordPress hosting")
3. Enter your affiliate URL
4. Select the affiliate program
5. Click **Add Link**

The plugin will automatically replace this keyword in future posts with your affiliate link.

### Configuration

Visit **Settings** to:
- Enable/disable the plugin
- Toggle auto-linking
- Customize link color

## Usage

### Creating Content

Write naturally in your posts. Any keyword you've added as an affiliate link will automatically be converted to an affiliate link (first occurrence only).

### Monitoring Performance

View real-time analytics in **Analytics**:
- Total clicks per affiliate program
- Conversion rates
- Earnings by program
- Performance trends

### Managing Links

View all links in **Affiliate Links** where you can:
- See click and conversion stats
- Track earnings
- Delete outdated links
