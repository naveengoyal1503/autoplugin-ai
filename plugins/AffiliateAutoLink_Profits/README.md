# AffiliateAutoLink Profits

Automatically detect product mentions in your posts and convert them into optimized affiliate links that rotate via simple A/B testing to maximize your affiliate revenue.

## Features

- Auto-detect product keywords in post content
- Automatically convert keywords into affiliate links
- Simple A/B testing rotating affiliate URLs per user session
- Admin settings for keywords and affiliate URLs
- Safe link cloaking for affiliate links
- Easy to use, set and forget

## Installation

1. Upload the `affiliateautolink-profits.php` file to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Settings > AffiliateAutoLink Profits** to configure keywords and affiliate URLs.

## Setup

- Enter the list of product keywords you want auto-linked (comma separated).
- Enter corresponding affiliate URLs in the same order (comma separated).
- Enable or disable auto-linking anytime via the checkbox.
- Enable A/B testing to rotate affiliate links shown per visitor.

## Usage

Once activated and configured, the plugin automatically scans your post content on display and replaces the first instance of each keyword with an affiliate link according to your settings. This happens dynamically and requires no manual intervention.

Enjoy passive affiliate monetization with minimal setup!
