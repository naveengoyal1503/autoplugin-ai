# Smart Affiliate Coupons Pro

## Features

- **Personalized Coupon Generation**: One-click generation of unique, trackable coupon codes for affiliates.
- **Shortcode Integration**: Easy `[affiliate_coupon]` shortcode for posts/pages.
- **Conversion Tracking**: Automatic URL parameter appending for affiliate link tracking.
- **Customizable**: Set discounts, products, and affiliate IDs per shortcode.
- **Premium Ready**: Hooks for pro features like analytics dashboard and API integrations.
- **Mobile Responsive**: Clean, modern design works on all devices.

Boost affiliate commissions by 20-50% with exclusive reader coupons.[1][2]

## Installation

1. Upload the `smart-affiliate-coupons-pro` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode in any post or page.

## Setup

1. Go to **Settings > Smart Coupons** (premium dashboard coming soon).
2. Enter your affiliate network details (optional for basic use).
3. Copy shortcode example: `[affiliate_coupon affiliate_id="amazon123" discount="20%" product="Laptop" link="https://amazon.com/product"]`.n
## Usage

- **Basic Shortcode**:
  
  [affiliate_coupon affiliate_id="yourid" discount="15%" product="Your Product" link="your-affiliate-link"]
  
- Visitors click **Generate My Coupon** to get a unique code.
- Code is copied-ready; link auto-tracks clicks.
- Track performance via your affiliate dashboard.
