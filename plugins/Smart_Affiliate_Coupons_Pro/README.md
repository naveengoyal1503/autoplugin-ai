## Installation

1. Upload the `smart-affiliate-coupons` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Smart Affiliate Coupons** to add your coupons.
4. Use the `[sac_coupon]` shortcode in posts/pages.

## Setup

1. In the admin panel, enter coupons one per line: `Brand Name|COUPONCODE|https://your-affiliate-link.com`.
2. Save settings.
3. Embed shortcode in content for instant display.

**Example Output**:

> **Brand1**: `DISCOUNT10` [Shop Now](https://affiliate.link)

## Usage

- **Posts/Blogs**: Add shortcode to recommend deals and earn commissions.[3]
- **Sidebar/Widgets**: Embed via shortcode widget.
- **Customization**: Style via CSS classes `.sac-coupons` and `.sac-item`.
