# Smart Affiliate Coupons Pro

## Description
**Smart Affiliate Coupons Pro** is a powerful WordPress plugin that helps bloggers and site owners monetize their content by automatically generating and displaying trackable affiliate coupons. Boost conversions with exclusive deals, personalized promo codes, and click analytics.

## Features
- **Shortcode-based coupons**: Easy `[sac_coupon]` integration for custom codes and links.
- **Auto-insertion**: Automatically adds coupons to blog posts (33% chance, customizable in Pro).
- **Click tracking**: Logs affiliate link clicks for performance insights.
- **Responsive design**: Mobile-friendly coupon boxes.
- **Freemium model**: Free core features; Pro unlocks unlimited coupons, dashboard analytics, A/B testing, and API integrations.
- **Privacy compliant**: GDPR-ready with nonce security.

## Premium Features (Pro Version)
- Unlimited coupons and brands.

- Custom coupon designs and animations.
- Integration with WooCommerce and affiliate networks.
- Scheduled coupon rotations and geo-targeting.

## Installation
1. Upload the `smart-affiliate-coupons-pro` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. No configuration needed for basic use—coupons auto-appear!

## Setup
1. **Basic Shortcode**: Use `[sac_coupon code="SAVE20" afflink="https://youraffiliate.link" desc="20% off!" brand="Your Brand"]` in posts/pages.
2. **Auto-insert**: Enabled by default on single posts.
3. **Customize**: Edit plugin file or upgrade to Pro for dashboard.
4. **Track Clicks**: View logs in `wp_options > sac_click_logs` or Pro dashboard.

## Usage
- **Embed manually**: Add shortcode anywhere.
- **Automatic**: Coupons appear randomly in content for testing.
- **Monetize**: Replace `afflink` with your affiliate URLs (Amazon, etc.).
- **Test**: Click 'Redeem Now'—tracks via AJAX, then redirects.
