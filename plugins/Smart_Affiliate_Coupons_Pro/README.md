## Installation

1. Download the plugin ZIP.
2. In WordPress admin, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate.
4. Use the shortcode in posts/pages.

## Setup

1. Place shortcode: `[affiliate_coupon affiliate_id="amazon1" discount="20%" product="Laptop" link="https://amazon.com/affiliate-link"]`.
2. Customize via shortcode attributes.
3. View stats in **Tools > Affiliate Coupons** (Pro feature).

## Usage
