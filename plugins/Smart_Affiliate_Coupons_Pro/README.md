# Smart Affiliate Coupons Pro

## Features

- **Easy Coupon Management**: Add unlimited affiliate coupons via simple admin dashboard (Pro: unlimited, Free: 5 max).
- **Conversion Tracking**: Tracks clicks on each coupon with real-time stats.
- **Shortcode Integration**: Use `[affiliate_coupon id="0"]` to display coupons anywhere.
- **Popup Display**: Auto-popup on exit intent to boost conversions (Pro feature).
- **SEO-Friendly**: Generates schema markup for coupons.
- **Pro Upgrade**: A/B testing, email capture, WooCommerce integration ($49/year).

## Installation

1. Upload the `smart-affiliate-coupons` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Aff Coupons** to add your coupons in JSON format: `[{ "code": "SAVE10", "afflink": "https://yourafflink.com", "discount": "10%", "expiry": "2026-12-31" }]`. 
4. Use shortcode `[affiliate_coupon id="0"]` in posts/pages.

## Setup

1. **Add Coupons**: In admin, enter array of coupons (code, affiliate link, discount, expiry).
2. **Embed**: Place shortcode in content or use widget (Pro).
3. **Track**: View clicks in dashboard. Upgrade for detailed analytics.
4. **Customize**: Edit CSS in plugin files or child theme.

## Usage

- **Basic Shortcode**: `[affiliate_coupon id="1"]` displays coupon #1.
