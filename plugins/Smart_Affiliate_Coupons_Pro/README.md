## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via WordPress Admin > Plugins.
3. Go to **Settings > SAC Pro** to add your coupons.
4. Insert `[sac_coupon]` in posts/pages.

## Setup

1. In **Settings > SAC Pro**, enter coupons in format:
   
   Coupon Code: SAVE20
   Affiliate Link: https://your-affiliate-link.com
   Description: 20% off!
   
   Coupon Code: DISCOUNT10
   Affiliate Link: https://another-link.com
   Description: Limited time deal!
   
2. Save settings.
3. **Pro Tip**: Upgrade for auto-expiring codes and A/B testing.

## Usage

- **Shortcode**: `[sac_coupon]` displays first coupon; `[sac_coupon id="2"]` for specific (Pro).
- **Customization**: Style via CSS: `.sac-coupon { border: 1px solid #ccc; padding: 20px; }`
- **Tracking**: Clicks redirect with UTM for analytics (Pro: dashboard shows clicks/conversions).
- **Monetization**: Perfect for blogs monetizing via affiliates and coupons[1][3]. Earn commissions on tracked deals!
