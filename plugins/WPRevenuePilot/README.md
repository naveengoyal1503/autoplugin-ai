# WPRevenuePilot

Automatically optimize your WordPress site's revenue by testing and deploying the most profitable monetization strategies.

## Features
- Easy-to-use admin dashboard
- Toggle monetization strategies (ads, affiliate, membership, courses)
- Premium subscription unlocks advanced analytics and A/B testing
- Injects monetization code automatically

## Installation
1. Download the plugin file.
2. Upload to your WordPress site via Plugins > Add New > Upload Plugin.
3. Activate the plugin.

## Setup
1. Go to the WPRevenuePilot menu in your admin panel.
2. Select the monetization strategies you want to enable.
3. Save your settings.

## Usage
- The plugin will automatically inject the necessary code for your chosen strategies.
- Upgrade to premium for advanced analytics and A/B testing features.
