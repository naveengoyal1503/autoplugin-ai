# Smart Content Monetizer Pro

An intelligent WordPress plugin that automatically optimizes content monetization by combining affiliate links, display ads, memberships, and sponsored content in a unified dashboard.

## Features

- **Unified Dashboard** - Monitor all monetization channels from one place
- **Membership Management** - Create and manage premium membership tiers with customizable pricing
- **Affiliate Link Management** - Automatically insert and track affiliate links with commission tracking
- **Ad Integration** - Seamlessly integrate Google AdSense and other ad networks with smart placement
- **Sponsored Content** - Manage sponsored posts and brand collaborations
- **Real-time Analytics** - Track clicks, impressions, revenue, and conversions (Premium)
- **AI-Powered Recommendations** - Get suggestions on optimal monetization mix (Premium)
- **Easy Shortcodes** - Quick integration with `[scm_membership]` and `[scm_affiliate]`
- **User-Friendly Interface** - Intuitive admin panel with setup wizard

## Installation

1. Download the plugin files
2. Upload the plugin directory to `/wp-content/plugins/`
3. Activate the plugin through the WordPress admin panel
4. Navigate to Dashboard > Monetizer to begin setup

## Setup

1. Go to Monetizer > Setup in your WordPress dashboard
2. Configure your membership pricing
3. Set affiliate commission rates
4. Choose preferred ad placement (top, middle, or bottom of posts)
5. Save your settings

## Usage

### Adding a Membership Box

Use the shortcode in any post or page:


[scm_membership]


### Adding Affiliate Links

Insert affiliate links with custom text:


[scm_affiliate url="https://example.com/product" text="Check This Out"]


### Dashboard Overview

The main dashboard displays:
- Total revenue across all channels
- Active membership count
- Affiliate click tracking
- Ad impression metrics
