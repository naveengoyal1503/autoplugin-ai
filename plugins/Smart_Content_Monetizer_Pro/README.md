# Smart Content Monetizer Pro

Intelligent WordPress plugin that automates and optimizes multiple monetization streams including display advertising, affiliate marketing, memberships, and donations.

## Features

- **Multi-Channel Monetization**: Manage ads, affiliates, memberships, and donations from one dashboard
- **Smart Ad Placement**: Automatically inject AdSense ads at optimal positions (above, middle, below content)
- **Affiliate Link Management**: Automatic affiliate disclosure integration
- **Donation System**: Built-in PayPal donation buttons
- **Performance Analytics**: Track revenue by source and content performance
- **Revenue Tracking**: Database-driven analytics for all monetization methods
- **AJAX Tracking**: Real-time view and click tracking without page reloads
- **Freemium Model**: Free tier with essential features, premium tier for advanced analytics

## Installation

1. Download the plugin file
2. Upload to `/wp-content/plugins/smart-content-monetizer/`
3. Activate through WordPress admin panel
4. Navigate to SCM Pro > Settings to configure monetization options

## Setup

### Step 1: Configure Ad Settings

1. Go to SCM Pro > Settings
2. Enable "Display Ads" checkbox
3. Enter your Google AdSense Publisher ID
4. Select ad positions (above, middle, or below content)
5. Save settings

### Step 2: Affiliate Configuration

1. Enable "Enable Affiliate Links"
2. Customize the affiliate disclosure text
3. The plugin automatically adds disclosure to all posts

### Step 3: Enable Donations

1. Check "Enable Donations" box
2. Customize donation prompt text
3. Add your PayPal hosted button ID for payment processing

### Step 4: Membership Features (Premium)

Premium subscribers get access to:
- Membership paywalls
- Advanced member management
- Exclusive member content
- Recurring payment handling

## Shortcodes

`[scm_donation]` - Display a standalone donation box
`[scm_affiliate_disclosure]` - Show affiliate disclosure
`[scm_ad_unit]` - Insert an ad unit anywhere
