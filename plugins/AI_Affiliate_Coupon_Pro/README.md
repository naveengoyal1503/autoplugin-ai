## Installation

1. Upload the `ai-affiliate-coupon-pro` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit **Settings > AI Coupons** to add your first coupon.
4. Use shortcode `[ai_coupon id="1"]` in posts/pages.
