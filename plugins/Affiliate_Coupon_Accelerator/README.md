# Affiliate Coupon Accelerator

## Description
The Affiliate Coupon Accelerator plugin automates the management and display of affiliate coupon deals on your WordPress site. It enables you to easily add, edit, and show active coupons to your visitors using a simple shortcode. This helps increase click-throughs and affiliate commissions by presenting timely, attractive deals prominently.

## Features

- Admin interface to add, edit, and remove affiliate coupons
- Set expiration dates for coupons; expired coupons automatically hide
- Shortcode `[affiliate_coupons]` to display coupon list anywhere on site
- Automatically limits displayed coupons to 5 for optimal impact
- Clean and responsive coupon display styles
- Lightweight, self-contained plugin

## Installation

1. Upload `affiliate-coupon-accelerator.php` to your WordPress plugin directory (`/wp-content/plugins/`).
2. Activate the plugin through the WordPress Admin > Plugins menu.
3. Go to the "Affiliate Coupons" menu item in the admin sidebar.
4. Add your affiliate coupons with title, code, URL, and optional expiry date.

## Setup

- Add as many coupons as you want in the admin interface.
- Use the expiry date field to disable outdated coupons automatically.

## Usage

- Insert the shortcode `[affiliate_coupons]` into posts, pages, or widget areas to display your active coupons list.
- Customize styles with CSS if needed.
