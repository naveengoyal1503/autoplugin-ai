# WP Deal Magnet

## Description

WP Deal Magnet is a WordPress plugin that allows you to create and display dynamic coupon and deal sections on your website. It helps bloggers, affiliate marketers, and eCommerce site owners increase visitor engagement and boost sales by showcasing exclusive discounts, affiliate offers, and timed deals.

## Features

- Custom post type for managing deals with coupon codes, affiliate URLs, and expiry dates
- Shortcode `[wp_deal_magnet]` to display active deals anywhere on your site
- Automatically hides expired deals
- Admin page to configure default display settings
- Clean, responsive design for deal listings
- Easy-to-use coupon code display and buttons to claim deals

## Installation

1. Upload the plugin folder to the `/wp-content/plugins/` directory or use WordPress plugin uploader
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'WP Deal Magnet' in the admin menu to add your deals

## Setup

- Add new deals in the 'WP Deal Magnet' custom post type admin area
- Enter deal title, description, coupon code, affiliate URL, and expiry date
- Use the shortcode `[wp_deal_magnet count="5" order="DESC"]` in your posts, pages or widgets to show deals
- Configure default settings via Settings > WP Deal Magnet

## Usage

- Place shortcode in your content to display deal lists:
  [wp_deal_magnet count="3"]
- Customize count and order with shortcode attributes
- Promote affiliate deals with trackable URLs and coupon codes
- Regularly update deals to keep content fresh and relevant

## Monetization Model

The plugin follows a freemium model: basic features are free, while advanced options such as automation, premium templates, and integrations with affiliate networks are planned as premium add-ons or subscription upgrades, encouraging users to pay for additional value.
