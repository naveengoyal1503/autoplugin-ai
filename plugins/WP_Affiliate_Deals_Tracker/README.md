# WP Affiliate Deals Tracker

## Description
WP Affiliate Deals Tracker helps WordPress site owners create, manage, and display exclusive affiliate coupon deals, while tracking clicks on affiliate links to improve monetization performance.

## Features
- Easy admin interface to add, edit, and manage affiliate deals
- Display affiliate deals using a simple shortcode `[affiliate_deals]`
- Track clicks on affiliate deal links to gather performance data
- Show coupon codes prominently to encourage conversions
- Clean, minimal design ready for customization

## Installation
1. Upload the plugin file `wp-affiliate-deals-tracker.php` to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the 'Affiliate Deals' menu in the admin dashboard to add new deals

## Setup
- Add new deals by entering a title, description, coupon code, and your affiliate URL.
- Use the shortcode `[affiliate_deals]` in posts, pages, or widgets to display all active deals.

## Usage
- Visitors will see your special coupon deals and can click to use affiliate links.
- Clicks on affiliate links are tracked automatically for performance insights.
- Monitor clicks and deals performance directly in the WordPress admin under 'Affiliate Deals'.
