# Smart Donation Pro

## Features
- **Easy Donation Buttons**: Add PayPal donation forms anywhere with a simple shortcode `[smart_donation]`.
- **Progress Bar**: Visual fundraising goal tracker that updates automatically.
- **Customizable**: Set button text, default amounts, currency, and goals via settings.
- **Mobile-Responsive**: Works perfectly on all devices.
- **Freemium Ready**: Free core; premium for recurring donations, analytics, and custom themes.
- **Lightweight**: Single-file plugin, no bloat.

## Installation
1. Download the plugin ZIP.
2. Upload via **Plugins > Add New > Upload Plugin**.
3. Activate the plugin.
4. Go to **Settings > Donation Pro** to configure.

## Setup
1. Enter your PayPal email in settings.
2. Set a fundraising goal (e.g., $1000).
3. Customize button text (e.g., "Buy Me a Coffee").
4. Save settings.

## Usage
- Use shortcode: `[smart_donation amount="20"]` for $20 default.
- Embed in posts, pages, sidebars, or widgets.
- Progress bar shows real-time donations (simulated for demo).
- Donations go directly to your PayPal.
