# Smart Donation Pro

## Features

- **Easy Donation Buttons**: Add customizable donation buttons anywhere with a simple shortcode `[smart_donation]`.
- **Progress Bars**: Visual goal tracking to motivate more donations.
- **PayPal Integration**: One-click PayPal payments for one-time donations.
- **Admin Dashboard**: Configure settings like amounts, goals, and messages from WordPress admin.
- **Mobile Responsive**: Works perfectly on all devices.
- **Freemium Upsell**: Teaser for pro features like recurring Stripe payments and analytics.

**Pro Version ($29/year)**: Recurring donations, Stripe/PayPal full integration, donation analytics, custom themes, email notifications, and more.

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Donation Settings** to configure your PayPal email, default amount, goal, etc.
4. Add `[smart_donation]` shortcode to any post, page, or widget.

## Setup

1. Enter your PayPal business email in settings.
2. Set a fundraising goal and default donation amount.
3. Optionally update current raised amount manually (Pro: auto-updates via IPN).
4. Customize button text and thank-you message.

## Usage

- **Shortcode**: `[smart_donation]` - Renders full widget with button, progress bar, and form.
- **Customization**: All styles are inline CSS for easy override via theme CSS.
- **Testing**: Use PayPal sandbox email for testing donations.
- **Monetization Tip**: Place on sidebars, footers, or high-traffic pages to maximize collections.
