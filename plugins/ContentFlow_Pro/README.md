# ContentFlow Pro

## Description

ContentFlow Pro is a powerful WordPress plugin that automates content repurposing and distribution. Transform your blog posts into engaging variations optimized for multiple platforms including Twitter, LinkedIn, Instagram, email, and YouTube. Built-in analytics tracking helps you monitor performance and revenue across all channels.

## Features

### Core Features
- **Automatic Content Variation Generation**: Convert blog posts into platform-specific content variations instantly
- **Multi-Platform Support**: Generate optimized content for Twitter, LinkedIn, Instagram, email, and YouTube
- **Built-in Analytics Dashboard**: Track clicks, impressions, conversions, and revenue by platform
- **Affiliate Link Tracking**: Monitor affiliate clicks and commissions integrated with your repurposed content
- **Bulk Content Processing**: Generate variations for multiple posts simultaneously
- **SEO-Optimized Variations**: Automatically include keywords and hashtags relevant to your content
- **Revenue Tracking**: Monitor earnings from each content variation across platforms

## Installation

1. Download the plugin files
2. Upload to `/wp-content/plugins/contentflow-pro/`
3. Activate the plugin through WordPress Admin Dashboard
4. Navigate to ContentFlow Pro menu to configure settings

## Setup

1. **Enable API Integration**:
   - Go to ContentFlow Pro > Settings
   - Enter your API key for advanced features (premium)
   - Configure your affiliate network credentials

2. **Configure Analytics Tracking**:
   - Enable analytics tracking checkbox
   - Set up conversion tracking for your affiliate links
   - Configure revenue tracking sources

3. **Set Repurposing Preferences**:
   - Choose which platforms to generate content for
   - Define keyword focus areas
   - Set character limits and tone preferences

## Usage

### Basic Repurposing

1. **Create or Edit a Post**:
   - In the post editor, locate the "ContentFlow Pro - Repurposing Options" meta box on the right sidebar
   - Check "Enable Repurposing"
   - Enter focus keywords (optional)
   - Click "Generate Variations"

2. **View Generated Content**:
   - Variations appear in the ContentFlow Pro dashboard
   - Edit variations as needed before publishing to social platforms
   - Copy variations directly for use in scheduling tools

### Bulk Processing

1. Navigate to ContentFlow Pro > Repurposer
2. Select multiple posts from the list
3. Click "Generate All"
4. All variations are automatically generated and stored

### Monitor Performance

1. Go to ContentFlow Pro > Analytics
2. View real-time statistics by platform
3. Track clicks, impressions, conversions, and revenue
4. Download reports for analysis

### Dashboard Overview

- Quick Stats: View total performance metrics
- Recent Repurposing: See latest generated variations
- Platform Performance: Compare metrics across social channels
- Revenue Tracking: Monitor earnings from each variation

## Database Tables

The plugin creates two custom database tables:

- `wp_contentflow_repurposing`: Stores generated content variations
- `wp_contentflow_analytics`: Tracks performance metrics and revenue

## Troubleshooting

**Variations not generating?**
- Ensure post has sufficient content
- Check that API key is valid (for premium features)
- Verify JavaScript is enabled in your browser

**Analytics not tracking?**
- Enable analytics tracking in settings
- Verify conversion tracking codes are installed
- Check browser console for JavaScript errors
