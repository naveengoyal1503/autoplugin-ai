# Affiliate Coupon Vault

## Features

- **Automatic Coupon Display**: Shortcode `[acv_coupons]` fetches and shows exclusive affiliate coupons.
- **Click Tracking**: Tracks all affiliate link clicks for performance insights (basic stats in free, full dashboard in Pro).
- **Customizable Design**: Responsive coupons with expiration checks.
- **Easy Integration**: Works with any theme; no coding required.
- **Pro Features**: Unlimited feeds, real-time API pulls (CJ, ShareASale), analytics dashboard, custom branding ($49/year).

## Installation

1. Download and upload the plugin ZIP to your WordPress admin > Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Use the shortcode `[acv_coupons]` in any post/page.

## Setup

1. Go to **Settings > Coupon Vault** to view basic stats.
2. Customize via shortcode: `[acv_coupons limit="3"]`.
3. For Pro: Enter affiliate API keys in settings (Pro only).

## Usage

- Add `[acv_coupons]` to sidebar widgets, posts, or pages to display live coupons.
- Clicks auto-redirect to affiliate links and get tracked.
- Monitor clicks in admin dashboard.
- **Boost Revenue**: Coupons increase conversions by 20-50% per industry benchmarks[1][2].
