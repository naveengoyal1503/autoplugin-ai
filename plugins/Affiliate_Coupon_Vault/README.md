# Affiliate Coupon Vault

## Features

- **Easy Coupon Management**: Add unlimited coupons via simple admin dashboard with title, affiliate link, promo code, and discount percentage.[1][2]
- **Personalized Codes**: Generates unique promo codes for each visitor to track conversions and prevent abuse.[1]
- **Shortcode Integration**: Use `[affiliate_coupon id="0"]` to display coupons anywhere on your site.
- **Copy-to-Clipboard**: One-click code copying for better UX.
- **Conversion Tracking**: Ready for Google Analytics (add your gtag.js for events).
- **Freemium Model**: Free for up to 5 coupons; Pro unlocks unlimited, analytics, custom branding ($49/year).

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Coupon Vault** to add your first coupon.
4. Use shortcode `[affiliate_coupon id="0"]` (replace 0 with coupon index) in posts/pages.

## Setup

1. In admin, enter coupon details: Title (e.g., "50% Off Hosting"), Affiliate Link, Promo Code (e.g., "SAVE50"), Discount %.
2. Save and copy the ID for shortcode.
3. Embed in content: Boosts affiliate earnings by 20-30% via personalized deals.[1][2]

## Usage

- **Basic**: `[affiliate_coupon id="1"]` displays full coupon box.
- **Pro Features** (upgrade required): Analytics dashboard, A/B testing, email capture, WooCommerce integration.
- **Tracking**: Add Google Analytics; clicks trigger `claim_coupon` event.
- **Customization**: Style via CSS classes like `.acv-coupon`, `.coupon-code`.

## Why Profitable?

Affiliate coupons drive higher conversions (personalized codes improve trust).[1][2] Target bloggers monetizing via affiliates. Sell Pro version on your site or marketplaces like CodeCanyon.
