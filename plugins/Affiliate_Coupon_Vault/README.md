# Affiliate Coupon Vault

## Features

- **Interactive Coupon Reveals**: Eye-catching buttons that reveal exclusive coupon codes on click, increasing user engagement.
- **Affiliate Tracking**: Unique tracking IDs for every coupon to monitor conversions and commissions.
- **Shortcode Simplicity**: Easy `[affiliate_coupon affiliate_url="https://example.com" discount="20%"]` integration.
- **Customizable Design**: Responsive, mobile-friendly coupons with customizable text and expiry dates.
- **Pro Upgrade**: Unlimited coupons, analytics dashboard, A/B testing, and premium templates ($49/year).

## Installation

1. Download the plugin ZIP file.
2. In your WordPress dashboard, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate the plugin.
4. Use the shortcode in any post, page, or widget.
