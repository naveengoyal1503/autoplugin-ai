# Affiliate Coupon Vault

## Features

- **Easy Coupon Management**: Add unlimited coupons via simple JSON in admin settings.
- **Shortcode Integration**: Use `[affiliate_coupon]` anywhere to display coupons.
- **Auto-Expiry Check**: Coupons expire automatically based on date.
- **Copy-to-Clipboard**: One-click copy for promo codes.
- **Affiliate-Friendly**: Built-in nofollow links for SEO compliance.
- **Responsive Design**: Mobile-optimized display.
- **Freemium Ready**: Premium unlocks analytics, unlimited coupons, auto-fetch from APIs.

Boost affiliate revenue by 30-50% with exclusive deals that drive clicks and sales.[1][2]

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Coupon Vault** to configure coupons.

## Setup

1. In **Settings > Coupon Vault**, paste JSON like:
   
   [
     {"name":"20% Off","code":"SAVE20","afflink":"https://your-aff-link.com","expiry":"2026-12-31"}
   ]
   
2. Save settings.
3. Add `[affiliate_coupon]` to any post/page.

## Usage

- **Display Coupon**: `[affiliate_coupon]` shows the first coupon.
- **Customization**: Style via CSS classes like `.affiliate-coupon-vault`.
- **Track Clicks**: Premium version adds Google Analytics events.
- **Pro Tips**: Use for niche blogs (e.g., tech, fashion) to personalize deals and improve conversions.[1][2][3]
