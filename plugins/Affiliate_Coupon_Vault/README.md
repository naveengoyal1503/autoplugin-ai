# Affiliate Coupon Vault

## Features

- **Generate Unique Promo Codes**: Automatically creates trackable coupon codes for affiliates.
- **Shortcode Integration**: Easy `[affiliate_coupon]` shortcode for posts/pages.
- **Conversion Boost**: Personalized discounts increase clicks and sales.[1][2]
- **Freemium Model**: Free core features; Pro unlocks analytics, unlimited coupons, integrations ($49/year).
- **Mobile-Responsive Design**: Eye-catching coupon boxes optimized for all devices.
- **SEO-Friendly**: Custom discount pages improve rankings for deal searches.[2]

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Configure defaults in **Settings > Coupon Vault**.

## Setup

1. Go to **Settings > Coupon Vault** to set default affiliate links/discounts.
2. Add shortcode to any post/page: `[affiliate_coupon discount="20%" link="https://your-affiliate-link.com" expires="2026-01-31"]`.
3. **Pro Upgrade**: Visit [example.com/pro] for advanced tracking and branding removal.

## Usage

- **Basic Shortcode**: `[affiliate_coupon]` uses site defaults.
- **Custom**: `[affiliate_coupon affiliate="amazon" discount="15% OFF" code="SAVE15"]`.
- Track unique codes via user ID in shortcode output.
- **Monetization Tip**: Partner with brands for exclusive codes to boost commissions.[1][2][3]
