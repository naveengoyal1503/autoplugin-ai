# Affiliate Coupon Vault

## Description
**Affiliate Coupon Vault** is a powerful WordPress plugin that automatically generates and displays exclusive affiliate coupons from popular networks like Amazon Associates, hosting providers, and SaaS tools. Boost your affiliate commissions by 30-50% with reader-first coupon displays.[1][2]

## Features
- **Shortcode-based coupons**: Use `[affiliate_coupons]` to display coupons anywhere.
- **Copy-to-clipboard functionality** for instant code copying.
- **Responsive design** works on all devices.
- **Sample coupons** for testing (Pro: Live API integration with 10+ networks).
- **Admin dashboard** for easy configuration.
- **Pro upsell** integration for seamless monetization.

## Installation
1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[affiliate_coupons limit="5"]` in any post or page.

## Setup
1. Go to **Settings > Coupon Vault** to configure (Pro features coming soon).
2. Add your affiliate links in the sample data or upgrade for auto-generation.
3. Customize networks: `hosting`, `tools`, `wordpress`.

## Usage
**Basic shortcode:**
`[affiliate_coupons network="hosting" limit="3"]`

**Examples:**
- Display 5 tool coupons: `[affiliate_coupons category="tools"]`
- Hosting deals: `[affiliate_coupons network="hosting"]`
