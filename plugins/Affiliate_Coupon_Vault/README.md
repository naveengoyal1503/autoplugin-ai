# Affiliate Coupon Vault

**Affiliate Coupon Vault** is a WordPress plugin that lets site owners create, manage, and display curated affiliate coupon codes easily. Boost your affiliate revenue by offering your visitors exclusive deals and making coupon management seamless.

## Features

- Add unlimited coupons with title, code, affiliate link, and optional descriptions.
- Display coupons anywhere using a simple shortcode: `[acv_coupons]`
- Widget support to showcase coupons in sidebars or widget areas.
- Responsive and clean coupon display styles.
- Easy admin interface for adding, editing, and removing coupons.
- Fully self-contained in a single PHP file.

## Installation

1. Download the plugin file and upload it to your WordPress `/wp-content/plugins/` directory.
2. Go to your WordPress admin dashboard.
3. Navigate to **Plugins** and activate **Affiliate Coupon Vault**.
4. A new menu item **Affiliate Coupons** will appear in your admin menu.

## Setup

1. Click **Affiliate Coupons** in the admin menu.
2. Add your coupons by filling in the title, coupon code, affiliate URL, and optionally a description.
3. Save your coupons.

## Usage

- Insert the shortcode `[acv_coupons]` into any post, page, or widget to display your coupons.
- Add the **Affiliate Coupon Vault** widget to any widget area to show your coupons in sidebars or footers.
