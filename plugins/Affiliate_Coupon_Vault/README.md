# Affiliate Coupon Vault

## Features

- **Automated Coupon Display**: Shortcode `[affiliate_coupon_vault]` generates eye-catching coupon boxes with your affiliate links.
- **Customizable Coupons**: Add/edit coupons via admin settings (JSON format).
- **Responsive Design**: Mobile-friendly gradient cards with hover effects.
- **Pro Upgrade Teaser**: Built-in upsell for premium features like API integrations, analytics, and more networks.
- **Easy Integration**: Works with any theme, no dependencies.

**Pro Features (coming soon)**: Amazon, CJ Affiliate integration, click tracking, custom promo code generator, A/B testing.[1][2]

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via WordPress Admin > Plugins.
3. Go to **Settings > Coupon Vault** to configure coupons.
4. Add `[affiliate_coupon_vault]` or `[affiliate_coupon_vault limit="2"]` to any post/page.

## Setup

1. In **Settings > Coupon Vault**, edit the JSON coupons array:
   
   [
     {"code":"SAVE20","desc":"20% off","afflink":"https://yourafflink.com"}
   ]
   
2. Replace `afflink` with your affiliate URLs for commissions.[3][5]
3. Save and test the shortcode.

## Usage

- **Shortcode Options**:
  - `limit`: Number of coupons to show (default: 3).
- Embed in sidebars, posts, or pages for high-conversion placements.
- **Monetization Tip**: Use personalized coupons from brands to boost uniqueness and earnings.[1][2]
