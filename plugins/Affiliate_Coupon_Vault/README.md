# Affiliate Coupon Vault

## Description
**Affiliate Coupon Vault** is a powerful WordPress plugin that helps bloggers and affiliate marketers generate **unique, trackable coupons** automatically. Boost your commissions by offering exclusive promo codes to your audience, increasing conversions and building trust.[1][2]

Perfect for niches like software, eCommerce, travel, and wellness where personalized discounts drive sales.[2]

## Installation
1. Download and upload the plugin ZIP to your WordPress site
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access settings via **Settings > Coupon Vault**
4. Use the shortcode in any post/page: `[affiliate_coupon]`

## Setup
1. Go to **Settings > Coupon Vault** to configure sample coupons
2. Add your affiliate links in the shortcode: `[affiliate_coupon link="https://your-affiliate-link.com?ref=%CODE%"]`
3. Customize discount, expiration, and affiliate name
4. Embed shortcodes in posts, sidebars, or popups

## Usage
### Basic Shortcode

[affiliate_coupon affiliate="Amazon" discount="15%" link="https://amazon.com/deal" expires="2026-02-01"]


### Example Output
- Generates: **Save 15% OFF** with code `ACV-XYZ789`
- Button: "Get Deal Now (Affiliate Link)"
- Tracks via `?ref=ACV-XYZ789&coupon=ACV-XYZ789`

### Widget/Block Support
Add to Gutenberg blocks, widgets, or via PHP: `echo do_shortcode('[affiliate_coupon]');`
