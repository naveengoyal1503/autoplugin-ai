# Affiliate Coupon Vault

## Description
**Affiliate Coupon Vault** is a powerful WordPress plugin that lets you create unique, trackable affiliate coupons instantly. Boost your affiliate earnings by offering exclusive discounts to your audience with personalized promo codes and click tracking.

Perfect for bloggers and marketers monetizing via affiliates[1][2][3].

## Features
- **One-click coupon generation** with unique codes
- **Shortcode integration** for easy embedding in posts/pages
- **Built-in click tracking** for performance analytics (Pro)
- **Customizable designs** with discount badges and CTAs
- **Affiliate link tracking** with UTM parameters
- **Freemium model**: Unlimited basic coupons; Pro adds analytics, API integrations, and white-labeling

## Pro Features (Upgrade for $49/year)
- Unlimited coupons and campaigns
- Detailed analytics dashboard
- Custom coupon expiration
- Email capture integration
- Priority support

## Installation
1. Download and upload the plugin ZIP to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode in any post or page

## Setup
1. Go to **Settings > Coupon Vault** to configure
2. Enter your affiliate links and customize settings
3. **Free limit**: 10 coupons/month. Upgrade for unlimited[7]

## Usage
Embed with shortcode:

[affiliate_coupon 
 affiliate_url="https://your-affiliate-link.com" 
 discount="30%" 
 description="Exclusive deal for readers!"]


**Example output**: Eye-catching coupon box with unique code, redeem button, and tracking.
