# Affiliate Coupon Vault

## Description

**Affiliate Coupon Vault** is a powerful WordPress plugin that automatically generates unique, trackable coupon codes for your affiliate links. Display eye-catching coupon boxes on your blog to skyrocket conversions and commissions. Perfect for bloggers and affiliate marketers.[1][2]

## Features

- **Unique Promo Codes**: Generates personalized coupon codes for each visitor (e.g., ACV-uuid-emailhash)
- **Customizable Shortcodes**: Easy integration with `[affiliate_coupon]` shortcode
- **Conversion-Optimized Design**: Mobile-responsive, animated coupon boxes
- **Affiliate Link Tracking**: Append coupon codes to any affiliate URL
- **Freemium Model**: Free version for testing, Pro unlocks unlimited usage & analytics
- **One-Click Copy**: JavaScript-powered code copying
- **Pro Upsell**: Built-in upgrade prompts for maximum revenue[7]

## Installation

1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode `[affiliate_coupon]` anywhere on your site
4. Configure settings in **Settings > Coupon Vault**

## Setup

### Basic Usage

[affiliate_coupon product="iPhone 15" discount="50%" link="https://your-affiliate-link.com"]


### Admin Settings
- Go to **Settings > Coupon Vault**
- Set your default affiliate links
- Enable Pro features (upgrade required)

## Usage Examples

**Amazon Deal Box:**

[affiliate_coupon affiliate="amazon" product="Echo Dot" discount="30% OFF" link="https://amzn.to/yourlink"]


**Custom Coupon:**

[affiliate_coupon product="WordPress Course" discount="75%" link="https://yourcourse.com/?ref=blogger" code="WPBLOG75"]
