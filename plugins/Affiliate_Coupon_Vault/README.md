# Affiliate Coupon Vault

## Features

- **Easy Shortcode Integration**: Add `[affiliate_coupon]` anywhere to display eye-catching coupon boxes.
- **Boost Affiliate Conversions**: Exclusive discount codes increase clicks and sales.
- **Customizable Design**: Branded coupons with your affiliate links.
- **Free Version**: Unlimited basic coupons.
- **Pro Features** ($49/year): Auto-coupon generation, click analytics, unlimited affiliates, custom templates, expiration tracking.

## Installation

1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Coupon Vault** to configure.

## Setup

1. In **Settings > Coupon Vault**, enter your Pro key (optional).
2. Add shortcode to any post/page: `[affiliate_coupon affiliate="Shopify" code="AFF10" discount="10% OFF" link="https://your-aff-link.com" expires="2026-12-31"]`.
3. Customize attributes as needed.

## Usage

- **Basic Example**: `[affiliate_coupon affiliate="Amazon" code="WPDEAL" discount="20%" link="https://amzn.to/yourlink"]`.
- **Pro Benefits**: Unlock analytics dashboard to track clicks, best-performing coupons, and revenue estimates.
- **Monetization Tip**: Perfect for niches like software, eCommerce, travel. Partner with brands for custom codes.
