# Affiliate Coupon Vault

## Features

- **Easy Coupon Management**: Add unlimited affiliate coupons via simple JSON in admin settings.
- **Click Tracking**: Tracks clicks on deals for performance analytics (Pro: Detailed reports).
- **Shortcode Integration**: Use `[affiliate_coupon id="0"]` to display coupons anywhere.
- **Conversion Booster**: Personalized deals increase affiliate commissions.[1][2]
- **Freemium Model**: Free for basics, Pro unlocks auto-generation, analytics, and more ($49/year).

## Installation

1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Coupon Vault** to add your coupons.

## Setup

1. In the admin page, enter coupons in JSON format:
   
   [
     {"name":"10% Off Hosting","code":"WP10","url":"https://affiliate-link.com/?ref=yourid","desc":"Exclusive deal"}
   ]
   
2. Save settings.
3. Use shortcode `[affiliate_coupon id="0"]` in posts/pages.

## Usage

- Display coupons in sidebars, posts, or widgets.
- Clicks are tracked and redirect to affiliate links.
- **Pro Features**: Unlimited coupons, exportable stats, AI-suggested deals, integrations with WooCommerce/Affiliate plugins.[3][4]
