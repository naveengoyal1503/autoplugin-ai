- **Easy Coupon Creation**: Add titles, affiliate links, promo codes, discounts, expiration dates, and images via intuitive admin dashboard.[1][2]

- **SEO Optimized**: Coupon pages are schema-ready for rich snippets and AI search visibility.[2]

## Installation

1. Download and upload the plugin ZIP to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Coupon Vault** in your admin sidebar to create your first coupon.[4]

## Setup

1. Go to **Coupon Vault > Add New**.
2. Fill in: Title, Affiliate Link (e.g., Amazon), Promo Code, Discount %, Expiration, Image URL.
3. Save and copy the coupon ID.
4. Add shortcode to any post/page: `[affiliate_coupon id="your-coupon-id"]`.
5. **Pro Tip**: Create a dedicated /coupons/ page for SEO traffic.[1][2]
