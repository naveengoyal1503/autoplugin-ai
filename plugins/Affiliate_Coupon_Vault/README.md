# Affiliate Coupon Vault

## Features

- **Easy Coupon Creation**: Add affiliate URLs, custom codes, expiry dates, and usage limits via intuitive admin dashboard.[1][2]
- **Personalized Tracking**: Auto-generates unique referral links for each click with usage counters.
- **Shortcode Integration**: Embed coupons anywhere with `[affiliate_coupon id="1"]`.
- **Auto-Expiry & Limits**: Coupons deactivate after expiry or max uses for exclusivity.[2]
- **Conversion Boost**: Personalized discounts improve affiliate rates and reader loyalty.[1][3]
- **Freemium Model**: Premium unlocks analytics, unlimited coupons, and exports ($49/year).

## Installation

1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`.
2. Activate the plugin via **Plugins > Installed Plugins**.
3. Access **Coupons** menu in admin to create your first coupon.

## Setup

1. Go to **Coupons** in WordPress admin.
2. Fill in title, affiliate URL (e.g., Amazon link), promo code, optional expiry, and max uses.
3. Save and copy the shortcode (e.g., `[affiliate_coupon id="1"]`).
4. Paste shortcode in posts/pages for display.

## Usage

- **Frontend**: Visitors see styled coupon box with code and "Get Deal" button. Clicks track automatically.
- **Backend**: Monitor uses and status in admin table.
- **Pro Upgrade**: For advanced features like click analytics and bulk import.
