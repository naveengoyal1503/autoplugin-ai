# Affiliate Coupon Vault

## Features

- **Easy Coupon Management**: Add unlimited coupons via simple JSON in settings (Pro: Unlimited).
- **Shortcode Integration**: Use `[affiliate_coupon id="0"]` to display coupons anywhere.
- **Click Tracking**: Unique tracking IDs for affiliate links with Google Analytics events.
- **Copy-to-Clipboard**: One-click coupon code copying with visual feedback.
- **Responsive Design**: Mobile-friendly coupon displays.
- **Pro Features**: Advanced analytics, custom designs, auto-expiring coupons, API integrations ($49/year).

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Coupon Vault** to add your coupons.

## Setup

1. In **Settings > Coupon Vault**, enter coupons as JSON:
   
   [
     {"name":"SAVE20","code":"SAVE20","afflink":"https://yourafflink.com","description":"20% off"}
   ]
   
2. Save settings.
3. Use shortcode `[affiliate_coupon id="0"]` in posts/pages.

## Usage

- **Display Coupon**: Paste shortcode in Gutenberg, Classic Editor, or widgets.
- **Track Performance**: Monitor clicks via Google Analytics (gtag events).
- **Customize**: Edit inline CSS in plugin file for branding.
- **Pro Upgrade**: Unlock analytics dashboard at [example.com/pro](https://example.com/pro).
