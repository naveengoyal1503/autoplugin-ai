# Affiliate Coupon Vault

## Features

- **Personalized Coupons**: Generates unique, trackable promo codes for each visitor/user to boost affiliate conversions.[1][2]
- **Easy Shortcode**: Use `[affiliate_coupon]` anywhere to display random exclusive deals.[1]
- **Admin Dashboard**: Manage coupon lists (network|discount|code format) with one-click save.
- **Copy-to-Clipboard**: One-click code copying for seamless user experience.
- **Freemium Pro**: Upgrade for unlimited coupons, analytics, auto-rotation, A/B testing, and premium affiliate integrations ($49/year).[7]
- **SEO-Friendly**: Coupon blocks are optimized for search engines and AI crawlers.[2]

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Configure coupons in **Settings > Coupon Vault** (free tier limits to 5 coupons).
4. Add `[affiliate_coupon]` shortcode to posts/pages.

## Setup

1. Go to **Settings > Coupon Vault**.
2. Enter coupons in format: `Network|Discount|Code` (one per line), e.g.:
   
   Amazon|20% Off|AMZ20SAVE
   Shopify|Free Trial|SHOPFREE
   
3. Save. Free version randomizes from your list; Pro auto-generates personalized codes.

## Usage

- **Shortcode**: `[affiliate_coupon]` - Displays a random coupon vault.
- **Customization**: Style via CSS classes like `.acv-coupon`.
- **Tracking**: Pro version logs clicks/impressions for affiliate optimization.[3]
- **Monetization Tip**: Pair with affiliate links for commissions on software, eCommerce, and deals.[1][2][3]
