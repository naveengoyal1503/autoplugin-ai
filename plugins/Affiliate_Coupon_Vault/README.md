# Affiliate Coupon Vault

## Description
**Affiliate Coupon Vault** is a powerful WordPress plugin that lets you create, manage, and display exclusive affiliate coupons on your site. Boost conversions with eye-catching coupon displays, built-in affiliate link tracking, and shortcode integration. Perfect for bloggers and marketers monetizing via affiliates.[1][2]

## Features
- **Easy Coupon Management**: Add unlimited coupons (pro) with name, code, description, redeem URL, and affiliate link via intuitive admin panel.
- **Shortcode Support**: Use `[affiliate_coupon id="0"]` to embed coupons anywhere.
- **Responsive Design**: Mobile-friendly coupon cards with copy-to-clipboard and click tracking.
- **Pro Upgrade**: Unlimited coupons, analytics dashboard, custom branding, A/B testing ($49/year).
- **SEO Optimized**: Generates schema-friendly markup for better search visibility.[3]

## Installation
1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Coupon Vault** to add your first coupon.
4. Use shortcode `[affiliate_coupon id="0"]` in posts/pages.

## Setup
1. In admin, add coupons with affiliate links (e.g., Amazon Associates, custom promo codes).[1][2]
2. Customize via CSS in **Appearance > Customize > Additional CSS**.
3. Upgrade to Pro for tracking: Monitors clicks and integrates with Google Analytics.

## Usage
- **Basic**: Embed shortcode in Gutenberg blocks or classic editor.
- **Advanced**: Style coupons with `.affiliate-coupon-vault` class.
- **Tracking**: Pro version logs clicks to dashboard; free uses UTM params.
- **Monetization Tip**: Pair with high-traffic niches like software, travel, eCommerce for max commissions.[1][2][3]
