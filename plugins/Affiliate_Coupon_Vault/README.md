# Affiliate Coupon Vault

## Features

- **Easy Coupon Management**: Add unlimited coupons (3 in free version) via intuitive admin dashboard with name, affiliate URL, promo code, and discount percentage.[1][2]
- **Click Tracking**: Real-time click tracking and stats display on frontend shortcodes for performance monitoring.[3]
- **Shortcode Integration**: Use `[acv_coupon id="0"]` to embed trackable coupons anywhere on your site.
- **Personalized Promo Codes**: Generate unique, reader-first coupons to boost conversions and affiliate earnings.[1][2]
- **Freemium Model**: Free core features; Pro unlocks unlimited coupons, analytics dashboard, custom branding, and API integrations.
- **SEO-Friendly**: Optimized for search engines with schema-ready structures.[2]

## Installation

1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Coupon Vault** to add your first coupon.
4. Use shortcode `[acv_coupon id="0"]` (replace 0 with coupon ID) in posts/pages.

## Setup

1. In admin, add coupons: Enter name (e.g., "50% Off Hosting"), affiliate URL, promo code (e.g., "SAVE50"), discount (e.g., "50").
2. Save changes. Free version limits to 3 coupons.
3. Embed shortcodes and watch clicks track automatically.
4. Upgrade to Pro for advanced features: [Get Pro](https://example.com/pro).

## Usage

- **Frontend**: Coupons display with claim button that tracks clicks and redirects to affiliate link.
