# Affiliate Coupon Vault

## Description

**Affiliate Coupon Vault** is a powerful WordPress plugin that helps affiliate marketers and bloggers display dynamic coupon codes and deals to boost conversions and commissions. Use shortcodes to embed exclusive offers from hosting, tools, and e-commerce brands.

Perfect for monetizing blogs with personalized discounts that drive clicks and sales.[1][2]

## Features

- **Easy Shortcodes**: `[affiliate_coupons count="5"]` - Display customizable coupon grids.
- **Demo Deals**: Pre-loaded high-converting offers from popular affiliates (Hostinger, Elementor, etc.).
- **Responsive Design**: Mobile-optimized layouts.

## Installation

1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[affiliate_coupons]` in any post, page, or widget.
4. Go to **Settings > Coupon Vault** to configure.

## Setup

1. **Add Shortcode**: Place `[affiliate_coupons count="3" category="hosting"]` where you want coupons to appear.
2. **Settings Page**: Navigate to **Settings > Coupon Vault** to enter Pro API keys (if upgraded).
3. **Customization**: Edit CSS in `/wp-content/plugins/affiliate-coupon-vault/style.css` for styling.
