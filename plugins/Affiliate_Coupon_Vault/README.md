# Affiliate Coupon Vault

## Features

- **Easy Coupon Management**: Add unlimited affiliate coupons via simple JSON in admin settings[1][2].
- **Shortcode Display**: Use `[acv_coupons]` or `[acv_coupons limit="3"]` to showcase randomized deals on any page/post.
- **Revenue Tracking Ready**: Pro version includes click/conversion analytics (upgrade for $49/year).
- **Mobile-Responsive Design**: Clean, professional coupon grids that boost conversions[1].
- **Affiliate-Friendly**: Auto nofollow links, personalized codes for brands[2].
- **Freemium Model**: Free core features; premium unlocks API integrations, dashboards, unlimited storage.

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Coupon Vault** to add your coupons.

## Setup

1. In admin: **Settings > Coupon Vault**.
2. Enter coupons as JSON: `[{ "name":"Brand", "code":"SAVE20", "afflink":"https://aff.link", "desc":"20% off" }]`. 
3. Save. Demo coupon auto-installs on activation.
4. Add `[acv_coupons]` to any post/page.

## Usage

- **Frontend**: Coupons display in a styled grid. Clicks track to affiliates.
- **Customization**: Edit CSS in plugin or theme.
- **Pro Upgrade**: Visit admin page for WooCommerce purchase link (unlimited features, analytics).
- **Best Practices**: Partner with brands for exclusive codes to increase engagement and commissions[1][2].
