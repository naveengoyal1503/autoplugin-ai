# Affiliate Coupon Vault

## Description
**Affiliate Coupon Vault** is a powerful WordPress plugin that automatically generates and displays exclusive affiliate coupons from popular networks like Amazon, CJ Affiliate, and ShareASale. Boost your site's conversions and commissions with eye-catching coupon displays.[1][2]

## Features
- **Shortcode Integration**: Use `[affiliate_coupons network="amazon" category="electronics" limit="5"]` anywhere.
- **Demo Coupons**: Free version shows high-converting demo coupons with your affiliate links.
- **Admin Settings**: Customize your affiliate ID and track clicks.
- **Responsive Design**: Mobile-optimized coupon displays.
- **Pro Features** (Upgrade for $49/year): Real API integrations, analytics dashboard, custom designs, A/B testing.[3][7]

## Installation
1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Coupon Vault** to enter your affiliate ID.
4. Add shortcode to any post/page: `[affiliate_coupons]`. 

## Setup
1. In admin settings, input your affiliate tracking ID (e.g., Amazon Associates ID).
2. Customize coupon networks and categories via shortcode attributes.
3. Premium: Connect API keys for live coupons from 10+ networks.[2]

## Usage
- **Basic**: `[affiliate_coupons limit="3"]` for default Amazon electronics.
- **Advanced**: `[affiliate_coupons network="cj" category="fashion"]`.
- Embed in sidebars via widgets or Gutenberg blocks.
- Track clicks with Google Analytics (Pro: Built-in dashboard).
