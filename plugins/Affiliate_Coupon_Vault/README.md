# Affiliate Coupon Vault

A freemium WordPress plugin for managing and displaying affiliate coupons with built-in click tracking to maximize monetization.[1][2]

## Features

- **Easy Coupon Management**: Add, edit, and delete coupons via intuitive admin interface.
- **Shortcode Display**: Use `[acv_coupons limit="10"]` to showcase coupons on any page or post.
- **Click Tracking**: Automatically tracks clicks and redirects to affiliate URLs for commissions.[3]
- **Expiry Management**: Set expiration dates for time-sensitive deals.
- **Responsive Design**: Mobile-friendly coupon cards.
- **Freemium Upsell**: Premium includes unlimited coupons, analytics, custom branding ($49/year).

## Installation

1. Download the plugin ZIP.
2. In WordPress admin, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate **Affiliate Coupon Vault**.
4. Access settings via **Settings > Coupon Vault**.

## Setup

1. Go to **Settings > Coupon Vault**.
2. Click **Add New Coupon**.
3. Enter **Title**, **Coupon Code**, **Affiliate URL** (your tracked affiliate link), **Description**, and optional **Expiry Date**.
4. Save and copy the shortcode `[acv_coupons]`.
5. Paste shortcode in posts, pages, or widgets.

## Usage

- **Frontend**: Coupons display as styled cards with a tracked "Get Deal" button. Clicks redirect to your affiliate link and increment counters.[1]
- **Admin Dashboard**: View all coupons with clicks, edit/delete inline.
- **Monetization Tips**: Partner with brands for exclusive codes, optimize for SEO with dedicated coupon pages.[2][3]
- **Shortcode Options**: `[acv_coupons limit="5"]` to limit display.
