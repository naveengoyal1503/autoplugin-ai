# Affiliate Coupon Vault

## Description
**Affiliate Coupon Vault** is a powerful WordPress plugin that helps bloggers and affiliate marketers create and display exclusive, personalized coupons directly in posts and pages. Boost conversions and commissions with eye-catching coupon shortcodes. Freemium model: Free for basics, Pro for advanced features.

## Features
- **Easy Shortcode Integration**: Use `[affiliate_coupon]` to embed stunning coupons anywhere.
- **Customizable Design**: Auto-styles with golden, attention-grabbing layouts.
- **Expiration Tracking**: Set coupon expiry dates for urgency.
- **Pro Features** (paid upgrade):
  - Unlimited coupons (free limits to 5).
  - Built-in analytics dashboard.
  - Auto-generate unique codes.
  - Premium affiliate network integrations (Amazon, etc.).
  - Custom branding & priority support.

## Installation
1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access settings via **Settings > Coupon Vault** to enter Pro key (optional).

## Setup
1. Go to **Settings > Coupon Vault** to configure.
2. Upgrade to Pro for $49/year via the settings page link.
3. Use the shortcode in any post/page:
   
   [affiliate_coupon code="SAVE20" afflink="https://your-affiliate-link.com" desc="Get 20% off exclusive deal!" expires="2026-02-01"]
