# Affiliate Coupon Vault

## Features

- **Easy Coupon Management**: Add unlimited coupons (free: basic limit) via simple admin interface. Format: Name|Affiliate URL|Discount.
- **Personalized Promo Codes**: Generates unique codes like "WPDeal-uuid" for each user to boost exclusivity and tracking.
- **Click Tracking**: Logs clicks, codes, and IP addresses (free: 100 entries; pro: unlimited with analytics dashboard).
- **Shortcode Integration**: Use `[acv_coupon]` for random or `[acv_coupon id="Name"]` for specific coupons anywhere.
- **Conversion-Optimized Design**: Eye-catching UI with reveal buttons to encourage clicks and affiliate commissions.
- **Freemium Model**: Free core features; pro unlocks advanced analytics, custom branding, auto-expiration, and affiliate network APIs.
- **SEO-Friendly**: Coupons are schema-ready for rich snippets.

## Installation

1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Coupon Vault** to add your coupons.
4. Use shortcode `[acv_coupon]` in posts/pages.

## Setup

1. In admin: Enter coupons like `BlackFriday|https://affiliate.link?ref=you|70% off` (one per line).
2. Free version auto-generates sample coupons on activation.
3. Customize CSS in the plugin's footer output if needed.

## Usage

- **Display Coupon**: `[acv_coupon]` - Shows random featured coupon.
- **Specific Coupon**: `[acv_coupon id="WPDeal"]`.
- **Widget/Page Builder**: Embed shortcode in Gutenberg, Elementor, etc.
- **Track Performance**: View basic logs in database (pro: dashboard).
- **Monetize**: Earn commissions from tracked affiliate links + upsell pro to users.
