# Affiliate Coupon Vault

## Description
**Affiliate Coupon Vault** is a lightweight WordPress plugin that enables bloggers and site owners to create dedicated coupon sections with exclusive affiliate promo codes. Track clicks, display usage stats, and boost commissions effortlessly.[1][2]

## Features
- **Easy Coupon Management**: Add unlimited coupons via intuitive admin dashboard with title, affiliate URL, promo code, and description.
- **Click Tracking**: Real-time click analytics to monitor performance and optimize affiliate earnings.[3]
- **Shortcode Integration**: Embed coupon grids anywhere with `[acv_coupons limit="10"]`.
- **SEO-Friendly**: Custom promo codes improve conversions and position your site as a deal hub.[1]
- **Mobile-Responsive**: Clean, modern design works on all devices.
- **Freemium Ready**: Premium unlocks advanced analytics, unlimited coupons, and exports.

## Installation
1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Visit **Settings > Coupon Vault** to add your first coupon (sample included).
4. Use shortcode `[acv_coupons]` in posts/pages.

## Setup
1. Go to **Settings > Coupon Vault**.
2. Add coupons with your **affiliate links** (e.g., Amazon Associates) and custom codes.
3. Copy shortcode to sidebar, posts, or dedicated pages.
4. Clicks auto-track and display usage counts.

## Usage
- **Display Coupons**: `[acv_coupons limit="5"]` shows top 5 active coupons.
- **Admin Stats**: View clicks per coupon to identify top performers.
- **Customization**: Style with CSS targeting `.acv-coupons`, `.acv-coupon`, `.acv-btn`.
- **Monetization Tip**: Partner with brands for exclusive codes to increase value.[2]
