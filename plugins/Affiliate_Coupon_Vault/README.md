## Installation

1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Coupon Vault** to configure your coupons (JSON format).
4. Use shortcode `[acv_coupon]` in posts/pages.

## Setup

1. In admin: Add coupons as JSON, e.g., `{"code":"SAVE20","afflink":"https://yourafflink.com","desc":"20% Off"}`.
2. Free version shows first coupon; Pro unlocks all.
3. Replace sample affiliate link with your own (Amazon, etc.).[3]

## Usage

- **Shortcode**: `[acv_coupon id="0"]` (id matches JSON array index).
- **Display**: Eye-catching box with code and "Get Deal" button.
- **Tracking**: Clicks redirect to affiliate link with logging.
- **Monetize**: Earn commissions from tracked sales. Ideal for blogs promoting deals.[1][2]
