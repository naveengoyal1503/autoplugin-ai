## Installation

1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access **Coupon Vault** in your admin sidebar to add coupons.
4. Use shortcode `[acv_coupons]` on any page or post.

## Setup

1. Go to **Coupon Vault > Add New**.
2. Fill in **Title**, **Description**, **Affiliate URL**, **Coupon Code** (optional), **Expiry Date** (optional), and **Image URL**.
3. Set **Active** and save.
4. Add `[acv_coupons]` to a page for a deals section.

**Database**: Plugin auto-creates table on activation.

## Usage

- **Frontend**: Embed shortcodes in posts/pages. Customize with `limit` or `category` attributes.
- **Tracking**: Clicks auto-tracked; view stats in admin list.
- **Affiliate Links**: Appends tracking params to URLs automatically.
- **Customization**: Edit CSS in plugin for branding.

**Example Shortcode**:

[acv_coupons limit="3"]
