# Affiliate Coupon Vault

## Features
- **Easy Coupon Management**: Add affiliate coupons via simple admin panel (Brand|Code|Discount|URL format).
- **Click Tracking**: Tracks clicks on deals (basic in free, advanced analytics in Pro).
- **Shortcode Integration**: Use `[acv_coupon]` to display coupons anywhere.
- **Pro Upgrade**: Unlimited coupons, custom branding, conversion analytics, email capture ($49/year).
- **SEO-Friendly**: Generates reader-first discount pages for better engagement and revenue [1][2].

## Installation
1. Upload the plugin ZIP to WordPress admin > Plugins > Add New > Upload.
2. Activate the plugin.
3. Go to Settings > Coupon Vault to add your coupons.

## Setup
1. In Settings > Coupon Vault, enter coupons one per line: `Brand1|DISCOUNT10|50|https://your-aff-link.com`.
2. Free version limits to 5 coupons.
3. Use shortcode `[acv_coupon]` in posts/pages.

## Usage
- Embed coupons in blog posts for affiliate commissions.
- Track clicks in dashboard (Pro: detailed reports).
- Customize styles via CSS.
- **Monetization Tip**: Partner with brands for exclusive codes to boost conversions [2][3].

## Pro Features
- Unlimited coupons.
- Analytics dashboard.
- Custom promo code generator.
- Integration with affiliate networks.
