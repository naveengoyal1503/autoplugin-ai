# Affiliate Coupon Vault

## Features

- **Easy Coupon Management**: Add unlimited coupons (Pro) with affiliate links, codes, and expiry dates via simple admin panel[1][2].
- **Shortcode Integration**: Use `[affiliate_coupon id="BrandName"]` anywhere on your site for instant coupon displays.
- **Conversion Boost**: Exclusive deals improve affiliate clicks and reader loyalty[1][2].
- **Pro Features**: Advanced analytics, auto-expiry, API integrations, unlimited coupons ($49/year).
- **SEO-Friendly**: Custom discount pages optimized for search engines[2].

## Installation

1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Coupon Vault** to configure your coupons.

## Setup

1. In the admin panel, enter coupons as JSON array:
   
   [
     {"name":"Brand1","code":"SAVE10","afflink":"https://your-aff-link.com","expires":"2026-12-31"}
   ]
   
2. Save settings.
3. Use shortcode: `[affiliate_coupon id="Brand1"]` in posts/pages.

## Usage

- **Display Coupons**: Embed shortcodes in blog posts for niche deals (e.g., software, travel)[2].
