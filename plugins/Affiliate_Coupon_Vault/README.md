# Affiliate Coupon Vault

## Features

- **Generate Exclusive Coupons**: Create custom, trackable coupon codes with affiliate links.
- **Auto-Expiry & Usage Tracking**: Coupons expire on date or after 100 uses (Pro: Unlimited).
- **Shortcode Integration**: Use `[affiliate_coupon id="0"]` to display coupons anywhere.
- **Admin Dashboard**: Easy management of codes, links, expiry, and usage stats.
- **Conversion Booster**: Personalized discounts increase clicks and commissions.[1][2]
- **Pro Upgrade**: Analytics, email capture, unlimited coupons, custom branding ($49/year).

## Installation

1. Upload the `affiliate-coupon-vault` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Coupon Vault** in the admin menu to add your coupons.

## Setup

1. Go to **Coupon Vault** dashboard.
2. Edit coupon fields: **Code** (e.g., SAVE20), **Affiliate Link** (your ref link), **Expiry** (date), **Uses** (auto-tracks).
3. Save changes.
4. Use shortcode `[affiliate_coupon id="0"]` for first coupon, `id="1"` for second, etc.

## Usage

- **On Posts/Pages**: Add shortcode to display trackable coupon boxes.
- **Boost Revenue**: Partner with brands for custom codes, track conversions via usage stats.[1][2]
- **Embed Example**:
