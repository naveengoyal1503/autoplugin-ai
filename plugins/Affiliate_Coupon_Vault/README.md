## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Coupon Vault** to add your coupons (format: `Name|Code|Brand|Affiliate Link|Active`).
4. Add `[affiliate_coupons]` shortcode to any page/post.

## Setup

1. In admin, enter coupons one per line:
   
   50% Off Hosting|HOST50|Bluehost|https://your-affiliate-link.com|Active
