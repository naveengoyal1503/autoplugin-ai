# Affiliate Coupon Vault

## Features

- **Auto-Generate Exclusive Coupons**: Creates unique, trackable discount codes for each visitor to boost affiliate conversions.[1][2]
- **Shortcode Integration**: Easy `[affiliate_coupon]` shortcode for posts/pages.
- **Admin Dashboard**: Manage affiliate links via JSON (name, URL, discount).
- **Pro Upgrade**: Unlimited coupons, analytics, custom branding, A/B testing ($49/year).
- **Conversion Optimized**: Personalized codes improve click-through and commissions.[1][2]
- **Lightweight & Self-Contained**: Single PHP file, no dependencies.

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Coupon Vault** to add affiliate links (JSON format).
4. Use `[affiliate_coupon]` shortcode in posts/pages.

## Setup

1. **Configure Links**: In settings, enter JSON like:
   
   [
     {"name":"10% Off Hosting","url":"https://aff.link","discount":"10% OFF"},
     {"name":"Free Trial Software","url":"https://aff.link","discount":"TRY FREE"}
   ]
   
2. **Pro Activation**: Enter license key from [example.com/pro](https://example.com/pro).
3. **Embed**: Add shortcode to sidebar, posts, or via Gutenberg block.

## Usage

- **Frontend**: Displays rotating coupons with unique codes (e.g., `A1B2C3D4`).
- **Tracking**: Codes append to affiliate URLs for analytics.
- **Customization**: Style via CSS targeting `#acv-coupon`.
- **Limits**: Free: 5 coupons/day. Pro: Unlimited + stats.
