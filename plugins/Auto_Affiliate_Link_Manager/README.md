# Auto Affiliate Link Manager

## Description
Automatically converts specified keywords in your posts into cloaked affiliate links, helping affiliate marketers boost revenue without manual linking.

## Features
- Define your own keyword-to-affiliate URL mappings
- Automatically converts first occurrence of keywords in posts to affiliate links
- Cloaks affiliate links with rel="nofollow noopener noreferrer" and opens in a new tab
- Simple admin interface under Settings > Affiliate Link Manager
- Prevents keyword linking inside existing links to avoid conflicts

## Installation
1. Upload `auto-affiliate-link-manager.php` to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **Settings > Affiliate Link Manager** to configure keywords and URLs

## Setup
- Add your keywords and corresponding affiliate URLs in the settings page
- Click 'Add New Keyword' to add multiple entries
- Save changes

## Usage
- Write posts normally without worrying about affiliate link placement
- The plugin will automatically convert the first instance of specified keywords into affiliate links on the front end
- Links will open in new tabs and are marked nofollow for SEO
