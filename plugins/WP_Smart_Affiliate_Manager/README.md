# WP Smart Affiliate Manager

Automate affiliate program management, tracking, payouts, and reporting for digital product sellers.

## Features
- Easy affiliate sign-up form
- Affiliate dashboard for tracking earnings and payouts
- Admin panel to manage affiliates
- Automated affiliate code generation
- Integration-ready for WooCommerce and Easy Digital Downloads

## Installation
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcodes `[affiliate_signup]` and `[affiliate_dashboard]` on any page

## Setup
- Add the `[affiliate_signup]` shortcode to a page for affiliate registration
- Add the `[affiliate_dashboard]` shortcode to a members-only page for affiliate dashboard
- Manage affiliates from the WordPress admin menu under 'Affiliate Manager'

## Usage
- Affiliates sign up and get a unique code
- Track earnings and payouts in the admin panel
- Customize the plugin with premium add-ons for advanced features
