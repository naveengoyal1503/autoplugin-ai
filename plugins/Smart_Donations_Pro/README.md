# Smart Donations Pro

## Features
- **Customizable donation buttons** with shortcodes: `[smart_donation goal="1000" title="Support Us"]`
- **Progress bars** showing real-time donation goals and totals
- **Database tracking** of donations with donor emails
- **PayPal integration** (set email in settings)
- **Admin dashboard** for configuration and viewing totals
- **Mobile-responsive** design
- **Freemium-ready**: Pro adds Stripe, unlimited goals, analytics, and email notifications

## Installation
1. Upload the `smart-donations-pro` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **Settings > Smart Donations** to set your PayPal email
4. Add shortcode to any page/post: `[smart_donation]`

## Setup
1. Configure PayPal email in admin settings
2. Customize shortcode attributes: `id`, `goal`, `title`, `button_text`, `paypal_email`
3. Optional: Add `smart-donations.js` and `smart-donations.css` (included in pro bundle)
