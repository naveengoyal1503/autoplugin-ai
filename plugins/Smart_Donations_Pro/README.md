# Smart Donations Pro

## Features

- **Easy Donation Buttons**: Add customizable donation shortcodes anywhere on your site with one line of code.
- **Progress Bars**: Visual goal trackers showing raised amounts to encourage more donations[3].
- **Suggested Amounts**: Quick-select buttons for $5, $10, $25 to boost conversions[1][6].
- **PayPal Integration**: Secure one-time payments via PayPal (recurring in Pro version)[3].
- **Total Raised Tracker**: Automatically tracks and displays site-wide donation totals.
- **Mobile-Responsive**: Works perfectly on all devices.
- **Freemium Upsell**: Premium features include Stripe support, recurring subscriptions, analytics dashboard, and custom designs[1][2].

## Installation

1. Download and upload the plugin ZIP to your WordPress admin → Plugins → Add New → Upload Plugin.
2. Activate the plugin.
3. Go to **Settings → Donations Pro** and enter your PayPal email address.
4. Use the shortcode `[smart_donation]` in any post, page, or widget.

## Setup

1. Configure your PayPal business email in the settings page.
2. Customize shortcode attributes:
   - `amount`: Default donation amount (e.g., `10`).
   - `goal`: Fundraising goal for progress bar (e.g., `1000`).
   - `button_text`: Button label (e.g., `Support Our Work`).
   - `currency`: Symbol (e.g., `$`, `€`).
3. Example: `[smart_donation amount="25" goal="5000" button_text="Buy Me a Coffee" currency="$"]`.

## Usage

- Place the shortcode in sidebars, footers, posts, or pages to start collecting donations instantly.
- Track total raised via the progress bar (updates on donation completion).
- **Pro Tip**: Use on membership sites, blogs, or non-profits for passive income[1][6].
- Premium: Add recurring options (monthly/annual) for steady revenue[1].
