# Smart Donations Pro

## Description
Smart Donations Pro is a lightweight, self-contained WordPress plugin that helps you monetize your site by adding attractive donation buttons with progress bars. Perfect for bloggers, creators, and non-profits. Supports one-time donations via PayPal with easy shortcode integration.

## Features
- **Customizable donation buttons** with progress bars showing goal progress.
- **Shortcode support**: `[sdp_donate goal="500" received="120" button_text="Support Us" currency="$"]`.
- **PayPal integration** for secure payments.
- **Admin dashboard** for settings and total received tracking.
- **Mobile-responsive** design.
- **No database bloat**: Lightweight and fast.
- **Freemium ready**: Premium unlocks recurring donations and analytics.

## Installation
1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Donations** to enter your PayPal email.

## Setup
1. Configure your PayPal email in **Settings > Donations**.
2. Add the shortcode to any post/page: `[sdp_donate goal="1000" received="250"]`.
3. Edit `goal` and `received` values to show progress (update `received` manually or via premium).

## Usage
- Place shortcodes in posts, pages, sidebars, or widgets.
- Customize with attributes: `goal`, `received`, `button_text`, `currency`.
- Donations go directly to your PayPal account.
- Track total donations in settings (premium for detailed analytics).
