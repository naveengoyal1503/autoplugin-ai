# Smart Affiliate AutoInserter

**Automatically boosts your affiliate revenue by inserting relevant links into content using smart keyword matching.**

## Features
- **Automatic Link Insertion**: Scans posts and pages, inserts affiliate links based on keywords without disrupting readability.
- **Customizable Rules**: Define keywords, link text, and URLs via intuitive dashboard.
- **Limits & Control**: Set max links per post to avoid over-optimization.
- **SEO-Friendly**: Adds `nofollow`, `noopener`, and opens in new tab.
- **Freemium Ready**: Core free; pro add-ons for AI context analysis, A/B testing, analytics.

## Installation
1. Upload the `smart-affiliate-autoinserter` folder to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Configure in **Settings > Affiliate AutoInserter**.

## Setup
1. Go to **Settings > Affiliate AutoInserter**.
2. Enable auto-insertion.
3. Add affiliate links: keyword (e.g., "best laptop"), link text (e.g., "Check Price"), URL (your affiliate link).
4. Set max links per post (default: 3).
5. Save changes.
