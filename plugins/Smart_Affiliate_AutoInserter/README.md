## Installation

1. Download the plugin ZIP.
2. In WordPress admin: **Plugins > Add New > Upload Plugin**.
3. Activate the plugin.
4. Upgrade to Pro for full features.

## Setup

1. Go to **Settings > Affiliate AutoInserter**.
2. Enter your **Amazon Affiliate Tag** (e.g., yourtag-20).
3. (Pro) Enter Amazon API key for dynamic products.
4. Save settings. Links auto-appear in new/existing posts.

**Note**: Free version uses demo links. Replace `YOURTAG` in code with your tag. Pro unlocks real API.

## Usage
