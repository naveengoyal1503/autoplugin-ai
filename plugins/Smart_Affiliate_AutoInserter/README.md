# Smart Affiliate AutoInserter

**Automatically monetize your WordPress site by inserting relevant Amazon affiliate links into posts and pages based on contextual keywords.**

## Features

- **Smart Auto-Insertion**: Detects keywords like "buy" or "purchase" and inserts affiliate links to relevant Amazon products (e.g., laptops, phones).
- **Customizable**: Set your Amazon affiliate tag, trigger keywords, link text, and max links per post.
- **Non-Intrusive**: Limits links to avoid spamming content; only on single posts/pages.
- **Easy Setup**: Configure in minutes via WordPress admin.
- **Freemium Ready**: Free version hooks users; **Pro** adds AI matching, multi-network support (ClickBank, ShareASale), analytics dashboard, A/B testing ($49/year).

## Installation

1. Upload the plugin files to `/wp-content/plugins/smart-affiliate-autoinserter` or install via WordPress admin.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to **Settings > Affiliate Inserter** to configure.

## Setup

1. **Get Amazon Affiliate Tag**: Sign up at [Amazon Associates](https://affiliate-program.amazon.com/) and copy your tag (e.g., `yourid-20`).
2. In plugin settings:
   - Enable auto-insertion.
   - Enter your affiliate tag.
   - Customize keywords (e.g., `buy,purchase,order`).
   - Set link text (e.g., `Check price on Amazon`).
   - Limit max links (default: 2).
3. Save and test on a post containing trigger words.

## Usage

- Write posts with natural buying intent keywords.
- Plugin scans content, matches product terms (laptop, book, etc.), and inserts `nofollow` links.
- Earn commissions on clicks/sales via Amazon.
