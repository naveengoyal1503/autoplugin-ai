# Smart Affiliate AutoInserter

**Automatically monetize your WordPress site by inserting relevant Amazon affiliate links into posts and pages using smart keyword matching.**

## Features
- **Auto-insertion**: Scans content for keywords and inserts affiliate links seamlessly.
- **Amazon Associates integration**: Uses your affiliate tag for commissions.
- **Free version**: Up to 3 links per post with default keywords.
- **Pro upgrade** ($29/year): Unlimited links, custom keywords, click analytics, A/B testing, link cloaking.
- **Mobile-responsive** and SEO-friendly (nofollow/sponsored attributes).
- **Easy dashboard**: Configure in minutes.

## Installation
1. Download the plugin ZIP.
2. In WordPress admin: **Plugins > Add New > Upload Plugin**.
3. Activate the plugin.
4. Go to **Settings > Affiliate Inserter** to configure.

## Setup
1. Sign up for [Amazon Associates](https://affiliate-program.amazon.com/) and get your **Affiliate Tag**.
2. In plugin settings:
   - Enter your **Affiliate Tag**.
   - Add keywords as JSON: `{"laptop":"B08N5WRWNW","phone":"B0C7W46YXK"}` (find ASINs on Amazon).
3. Save settings. Links auto-appear in new/existing posts.

## Usage
- Write content with keywords like "laptop" or "phone".
- Plugin auto-links them to Amazon products.
- Track earnings in Amazon Associates dashboard.
- **Pro tip**: Use Pro for performance stats and optimization.
