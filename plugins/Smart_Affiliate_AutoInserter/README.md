# Smart Affiliate AutoInserter

**Automatically inserts relevant affiliate links into your WordPress posts and pages to maximize commissions passively.**

## Features

- **Keyword-Based Auto-Insertion**: Scans content for keywords and inserts your affiliate links naturally.
- **Configurable Limits**: Set max links per post to avoid over-optimization.
- **Easy Dashboard**: Add/edit keywords, affiliate URLs, and custom link text.
- **Randomized Placement**: Uses probability to mimic natural linking.
- **nofollow & noopener**: Links are SEO-safe with proper attributes.
- **Freemium Ready**: Free core; pro for AI matching, analytics, and more.

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Affiliate Inserter** to configure.

## Setup

1. Enable auto-insertion.
2. Set max links per post (default: 3).
3. Add affiliate links:
   - **Keyword**: e.g., "best laptop"
   - **Affiliate URL**: Your Amazon/ClickBank link
   - **Link Text**: Optional custom text (defaults to keyword)
4. Save and test on a post.

## Usage

- Links auto-insert on frontend content (posts/pages).
- Edit settings anytime; changes apply instantly.
- **Pro Tip**: Use 5-10 keyword/link pairs for best results.
- Monitor via Google Analytics or pro analytics add-on.
