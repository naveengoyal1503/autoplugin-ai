# Smart Affiliate AutoInserter

**Automatically boosts your affiliate earnings by inserting relevant Amazon links into posts based on smart keyword matching.**

## Features

- **Auto-Insert Links**: Scans content for keywords and inserts Amazon affiliate links seamlessly.
- **Customizable**: Define your keywords and product URLs easily.
- **Performance Limits**: Control max links per post to maintain readability.
- **Mobile-Responsive**: Links work perfectly on all devices.
- **Pro Upgrade**: Unlimited keywords, click analytics, A/B testing, custom link text, and multi-site support ($49/year).

## Installation

1. Download the plugin ZIP.
2. In WordPress admin, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate.
4. Go to **Settings > Affiliate Inserter** to configure.

## Setup

1. Sign up for [Amazon Associates](https://affiliate-program.amazon.com/) and get your affiliate tag.
2. In plugin settings:
   - Enter your **Amazon Affiliate Tag**.
   - Add keywords and products: e.g., `laptop:https://amazon.com/dp/B08N5WRWNW\nphone:https://amazon.com/dp/B09G9FPGT6` (one per line).
   - Set **Max Links per Post** (default: 3).
3. Save settings. Links auto-insert on publish/update.

## Usage

- Write posts with natural keywords (e.g., "best laptop").
- Plugin auto-adds affiliate links like: [laptop](https://amazon.com/dp/B08N5WRWNW?tag=yourtag-20).
- **Test**: View a post and inspect HTML for links.
- **Pro Tips**:
  - Use 3-5 high-converting keywords.
  - Track earnings in Amazon dashboard.
  - Upgrade to Pro for stats and optimization.
