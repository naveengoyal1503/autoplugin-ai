# Smart Affiliate AutoInserter

**Automatically boosts your affiliate earnings by inserting relevant Amazon links into your content using smart keyword matching.**

## Features

- **Auto-Insert Affiliate Links**: Scans posts and pages for keywords and replaces them with cloaked Amazon affiliate links.
- **Easy Setup**: Enter your Amazon Associate ID and keyword-product pairs.
- **Freemium Model**: Free for basic manual keywords; **Pro ($29/year)** unlocks AI-powered auto-keyword detection, click analytics, link cloaking, and multi-site support.
- **SEO-Friendly**: Links are nofollow and open in new tabs to protect user experience.
- **Mobile-Responsive**: Works seamlessly on all devices.

## Installation

1. Download the plugin ZIP file.
2. In your WordPress admin, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate **Smart Affiliate AutoInserter**.
4. Go to **Settings > AffiliateInserter** to configure.

## Setup

1. **Get Amazon Affiliate ID**: Sign up at [Amazon Associates](https://affiliate-program.amazon.com/) and copy your tag (e.g., `yourid-20`).
2. **Add Keywords**: In settings, enter JSON like `{"laptop":"B08N5WRWNW","phone":"B0C7H9X7QW"}` (keyword => ASIN).
3. **Save & Test**: Publish a post with keywords to see links auto-inserted.

**Pro Upgrade**: Click the upgrade button in settings for advanced features.

## Usage

- Links auto-appear in `the_content` (posts/pages).
- Earn commissions on clicks/sales via Amazon.
- **Pro Analytics**: Track clicks and optimize keywords.
- **Best Practices**: Use 3-5 links per post; test on staging site.
