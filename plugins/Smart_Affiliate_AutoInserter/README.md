# Smart Affiliate AutoInserter

Automatically detects keywords in your posts and inserts relevant **Amazon affiliate links** to maximize commissions with minimal effort.

## Features

- **Automatic Insertion**: Scans content for keywords and adds contextual Amazon affiliate links.
- **Easy Setup**: Enter your Amazon affiliate tag and keywords in settings.
- **Non-Intrusive**: Links appear naturally near matching keywords.
- **Mobile-Responsive**: Links work perfectly on all devices.
- **Free Core**: Powerful basic functionality at no cost.

**Pro Features** (available via upgrade):
- Advanced keyword targeting and exclusions
- Click analytics and performance tracking
- Custom link positions and styling
- Unlimited keywords and sites
- Priority support

## Installation

1. Download the plugin ZIP from [WordPress.org](https://wordpress.org/plugins/smart-affiliate-autoinserter/).
2. In your WordPress dashboard, go to **Plugins > Add New > Upload Plugin**.
3. Upload the ZIP file and activate the plugin.
4. Navigate to **Settings > Affiliate Inserter** to configure.

## Setup

1. **Get Amazon Affiliate Tag**: Sign up at [Amazon Associates](https://affiliate-program.amazon.com/) and get your unique tag (e.g., `yourid-20`).
2. Go to **Settings > Affiliate Inserter**.
3. Enter your **Amazon Affiliate Tag**.
4. Add **keywords** (comma-separated, e.g., `laptop,phone,book,headphones`).
5. Save changes. Links auto-appear in new and existing posts!

## Usage

- Write posts with natural keywords like "best laptop" ‚Äì the plugin inserts a link like "Check laptop on Amazon".
- View live on frontend; no shortcodes needed.
- Test by searching your site for inserted links.

**Pro Tip**: Use high-converting keywords from your analytics for best results.
