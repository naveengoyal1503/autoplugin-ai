## Installation

1. Download the plugin ZIP file.
2. In your WordPress admin, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate **Smart Affiliate AutoInserter**.
4. Go to **Settings > Affiliate AutoInserter** to configure.

## Setup

1. **Get Amazon Affiliate Account**: Sign up at [affiliate-program.amazon.com](https://affiliate-program.amazon.com) and get your **Associate Tag** (e.g., `yourtag-20`).
2. **Optional: Amazon API Key**: For Pro version â€“ get from Amazon Product Advertising API.
3. **Configure Keywords**: Add comma-separated keywords like `laptop, phone, book, camera`.
4. **Set Max Links**: Choose 1-5 links per post.
5. **Enable Auto-Insert** and save.
