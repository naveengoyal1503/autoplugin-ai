# Smart Affiliate AutoInserter

**Automatically inserts relevant Amazon affiliate links into your WordPress posts and pages to boost monetization effortlessly.**

## Features
- **Auto-Insert Links**: Scans content for keywords and inserts Amazon affiliate links with a smart probability to avoid spamming.
- **Easy Setup**: Enter your Amazon Associates tag and API key in settings.
- **Customizable**: Toggle auto-insertion on/off.
- **Mobile-Responsive**: Links work perfectly on all devices.
- **Freemium Pro**: Upgrade for AI keyword detection, multiple affiliate networks (e.g., Amazon, ClickBank), click analytics, and A/B testing.

## Installation
1. Download the plugin ZIP.
2. In WordPress admin, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate **Smart Affiliate AutoInserter**.
4. Go to **Settings > Affiliate AutoInserter** to configure.

## Setup
1. Sign up for [Amazon Associates](https://affiliate-program.amazon.com/) and get your **Associate Tag**.
2. Optionally, get an Amazon Product Advertising API key (for pro features).
3. In plugin settings:
   - Enter your **Affiliate Tag**.
   - Enable **Auto-Insert**.
   - Save changes.

## Usage
- Write posts with keywords like "book", "phone", or "laptop".
