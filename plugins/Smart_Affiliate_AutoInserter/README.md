# Smart Affiliate AutoInserter

**Automatically inserts relevant affiliate links into your WordPress posts and pages to maximize monetization.**

## Features

- **Keyword-based Auto-Insertion**: Matches keywords in content and inserts affiliate links seamlessly.
- **Customizable Limits**: Set max links per post to avoid over-optimization.
- **Easy Admin Panel**: Add, edit, and remove affiliate links with keyword, text, and URL.
- **nofollow & noopener**: All links include proper SEO attributes.
- **Freemium Ready**: Pro version adds AI matching, A/B testing, click analytics.
- **Lightweight & Self-Contained**: Single PHP file, no dependencies.

## Installation

1. Download the plugin ZIP.
2. Upload `smart-affiliate-autoinserter.php` to `/wp-content/plugins/` via FTP or WordPress uploader.
3. Activate the plugin from **Plugins > Installed Plugins**.
4. Go to **Settings > Affiliate Inserter** to configure.

## Setup

1. In the admin panel, toggle **Enable Auto-Insertion**.
2. Set **Max Links per Post** (default: 3).
3. Add affiliate links:
   - **Keyword**: Word/phrase to match (e.g., "best laptop").
   - **Link Text**: Display text (e.g., "Check Amazon").
   - **Affiliate URL**: Your affiliate link.
4. Click **Add Affiliate** for more entries.
5. Save changes. Links auto-insert on single posts.

## Usage
