## Installation

1. Download the plugin ZIP from WordPress.org or GitHub.
2. In your WordPress dashboard: **Plugins > Add New > Upload Plugin**.
3. Activate the plugin.
4. Go to **Settings > Affiliate Inserter** to configure.

## Setup

1. Navigate to **Settings > Affiliate Inserter**.
2. Add keywords (e.g., "laptop") and your affiliate URLs.
3. Set max links per post (default: 3).
4. Save changes. Links auto-insert on frontend!

**Example:**
- Keyword: `coffee maker`
- Link: `https://amazon.com/affiliate-link`

## Usage

- **Test It**: Create a post with your keywords; view frontend to see links inserted.
- **Best Practices**: Use 3-5 links/post; target long-tail keywords for higher conversions.
- **Troubleshooting**: Ensure no caching conflicts; clear cache after setup.
