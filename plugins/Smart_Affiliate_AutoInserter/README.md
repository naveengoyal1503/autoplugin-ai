# Smart Affiliate AutoInserter

**Automatically boosts your affiliate earnings by inserting relevant links into posts using smart keyword matching.**

## Features

- **AI-Powered Keyword Matching**: Scans posts for keywords and auto-inserts your affiliate links.
- **Freemium Model**: Free version limits to 2 links/post; Pro unlocks unlimited links, A/B testing, analytics.
- **Easy Setup**: Add keywords and URLs in settings; no coding required.
- **SEO-Friendly**: Adds `nofollow` and `sponsored` attributes.
- **Pro Features**: Advanced targeting, performance tracking, priority support ($49/year).

## Installation

1. Download the plugin ZIP.
2. In WordPress admin: **Plugins > Add New > Upload Plugin**.
3. Activate the plugin.
4. Go to **Settings > Affiliate Inserter** to configure.

## Setup

1. In settings, enter affiliate links as `keyword|https://affiliate-url.com` (one per line).
2. Set max links per post (free: max 2).
3. For Pro: Enter license key after purchase.
4. Save settings. Links auto-insert on published posts/pages.

## Usage

- Write content with natural keywords (e.g., "best laptop").
- Plugin matches and links automatically.
- View earnings via your affiliate dashboards.
- **Pro Tip**: Use high-converting keywords for best results.
