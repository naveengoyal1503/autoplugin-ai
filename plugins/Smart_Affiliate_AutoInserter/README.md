# Smart Affiliate AutoInserter

## Description
**Smart Affiliate AutoInserter** is a powerful WordPress plugin that automatically detects keywords in your posts and pages and replaces them with your Amazon affiliate links. Earn passive income from your existing content without manual link insertion. Free version supports up to 3 links per page; upgrade to Pro for unlimited features.

## Features
- **Automatic Keyword Replacement**: Smart matching inserts affiliate links seamlessly.
- **Click Tracking**: Monitor performance (basic stats in free version).
- **Mobile-Responsive**: Links work perfectly on all devices.
- **Easy Setup**: Configure via WordPress admin in minutes.
- **Pro Features**: Unlimited links, A/B testing, detailed analytics, custom link positions, and priority support ($49/year).

## Installation
1. Download and upload the plugin ZIP to your WordPress admin → **Plugins** → **Add New** → **Upload Plugin**.
2. Activate the plugin.
3. Go to **Settings** → **Affiliate AutoInserter** to configure.

## Setup
1. **Get Amazon Affiliate ID**: Sign up at [Amazon Associates](https://affiliate-program.amazon.com/) and note your tag (e.g., `yourid-20`).
2. **Prepare Keywords**: Create a JSON object like:
   
   {
     "laptop": "https://amazon.com/dp/B08N5WRWNW",
     "phone": "https://amazon.com/dp/B0BTM1NWFY",
     "headphones": "https://amazon.com/dp/B07NM3RSK7"
   }
   
3. Enter your Affiliate ID and JSON in the settings page.
4. Save changes. Links auto-appear in content!

## Usage
- Write content with keywords (e.g., "best laptop").
- Plugin replaces matches with affiliate links.
- View basic click stats in settings.
- **Pro Tip**: Use 1-3 links per 1000 words for optimal conversions.
