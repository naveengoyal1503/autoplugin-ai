# Smart Affiliate Link Cloaker & Tracker

## Description
Automatically cloak, manage, and track your affiliate links to increase revenue with detailed click analytics. Easy to use shortcode and redirect URLs allow for seamless integration.

## Features
- Define affiliate link slugs and target URLs in admin
- Cloak affiliate links using shortcode `[affiliate_link slug="your-slug"]Link Text[/affiliate_link]`
- Supports direct redirect URLs at `/go/your-slug`
- Tracks clicks with timestamp, IP, and user agent
- Admin dashboard summary of clicks per affiliate link
- Basic click tracking via AJAX for improved accuracy
- SEO-friendly cloaked links

## Installation
1. Upload the plugin PHP file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the WordPress Admin Plugins page.
3. Go to the **Affiliate Cloaker** menu in WordPress admin.
4. Add your affiliate links with unique slugs and target URLs.
5. Save settings.

## Setup
- Use the shortcode anywhere in your posts/pages: `[affiliate_link slug="my-affiliate"]Click Here[/affiliate_link]`
- Or link directly using `https://yoursite.com/go/my-affiliate`
- The plugin will cloak the link and track clicks.

## Usage
- Monitor click stats on the Affiliate Cloaker admin page.
- Use the data to optimize your affiliate marketing strategy.
- Optionally extend the plugin for premium features like link rotation or export.
