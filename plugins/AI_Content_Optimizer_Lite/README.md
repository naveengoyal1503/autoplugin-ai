# AI Content Optimizer Lite

**AI-powered plugin to analyze and optimize your WordPress content for readability, SEO, and engagement.**

## Features

### Free Version
- **Readability Analysis**: Get a score based on sentence length and content depth.
- **Quick Tips**: Actionable suggestions to improve your posts.
- **Limited Scans**: 5 free analyses per site.
- **Sidebar Integration**: Easy access in post editor.

### Premium Features (Upgrade for $4.99/month)
- Unlimited analyses.
- AI-powered content rewriting suggestions (OpenAI integration).
- Advanced SEO insights (keyword density, meta optimization).
- Export reports and priority support.

## Installation

1. Upload the `ai-content-optimizer-lite` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Edit any post to see the **AI Content Optimizer** meta box.

## Setup

1. Go to **Settings > AI Optimizer** to enter your premium API key (if upgraded).
2. No further setup required for free version.

## Usage

1. **Analyze Content**: Open a post editor, click **Analyze Content** in the sidebar.
2. **View Score**: See your readability score (0-100) and tips.
3. **Optimize**: Follow tips to improve, re-analyze as needed.
4. **Upgrade**: For unlimited use, visit [premium link](https://example.com/premium).
