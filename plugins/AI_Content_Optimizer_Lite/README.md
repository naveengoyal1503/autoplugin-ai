# AI Content Optimizer Lite

## Description
**AI Content Optimizer Lite** is a powerful WordPress plugin that uses AI to analyze your post content for SEO, readability, and engagement. Get instant suggestions to improve your content. Free version limited to 5 analyses per month; upgrade to premium for unlimited access, advanced rewrites, and more[1][2][4].

## Features
- **Basic AI Content Analysis**: SEO score, readability score, word count, and optimization suggestions.
- **Post Integration**: Side meta box on edit screens for one-click analysis.
- **Usage Limits**: 5 free analyses per month (resets monthly).
- **Premium Teaser**: Clear upgrade prompts to unlock unlimited features.
- **Lightweight & Self-Contained**: Single PHP file, no dependencies.

## Premium Features (Upgrade for $9/mo)
- Unlimited analyses.
- AI-powered content rewrites.
- Keyword research integration.
- Export reports.
- Priority support[1][2][6].

## Installation
1. Download the plugin ZIP.
2. In WordPress admin, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate.
4. Free uses are auto-set to 5 on activation.

## Setup
1. Go to **Settings > AI Content Optimizer** to view usage and enter premium API key (if upgraded).
2. Edit any post; find the **AI Content Optimizer** meta box in the sidebar.

## Usage
1. Edit a post.
2. Click **Analyze Content** in the sidebar box.
3. View results: SEO score, suggestions, etc.
4. Apply suggestions manually (premium auto-applies rewrites).
5. When free limit reached, upgrade via the link.
