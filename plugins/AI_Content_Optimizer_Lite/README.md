# AI Content Optimizer Lite

## Features

- **Readability Analysis**: Calculates a simple Flesch-like score based on sentence length and word count.
- **SEO Score**: Evaluates headings, lists, images, links, and keyword density.
- **AI Suggestions**: Provides actionable tips to improve content.
- **Content Optimization**: Applies basic fixes like adding lists and formatting.
- **Free Tier**: 5 optimizations per month.
- **Premium Upsell**: Unlimited optimizations, bulk processing, advanced AI ($49/year).

## Installation

1. Upload the `ai-content-optimizer-lite` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Edit any post and look for the AI Optimizer tool.

## Setup

- No configuration needed for free version.
- Premium users: Enter license key in plugin settings (future update).

## Usage

1. Go to any Post/Page editor.
2. Click the 'AI Optimizer' button (appears in admin bar or tools).
3. Paste or select content to analyze.
4. View scores and suggestions.
5. Apply optimized content.
