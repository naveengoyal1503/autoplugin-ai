# AI Content Optimizer Lite

[![WordPress plugin](https://img.shields.io/wordpress/plugin/downloads/ai-content-optimizer.svg)](https://wordpress.org/plugins/ai-content-optimizer/)

A powerful **freemium** WordPress plugin that uses AI to analyze your content for **SEO**, **readability**, and engagement. Get instant scores and suggestions to improve your posts.

## Features

### Free Version
- Analyze up to **5 posts per month**.
- **Word count**, readability score, and basic SEO score.
- AI-generated suggestions for improvements.
- Simple dashboard interface.

### Premium Version (Upgrade for Unlimited)
- **Unlimited scans**.
- Advanced AI insights (keyword density, LSI terms).
- **One-click auto-optimization**.
- Export reports and priority support.
- Integration with major SEO tools.

## Installation

1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Navigate to **AI Optimizer** in your admin dashboard.

## Setup

1. Go to **AI Optimizer** page.
2. Paste content or enter a Post ID.
3. Click **Analyze Content**.
4. Review scores and apply suggestions.

## Usage

- **Basic Analysis**: Paste text into the textarea and analyze.
- **Track Limits**: Free users get 5 scans/month; upgrade notice appears when limit reached.
- **Premium Upsell**: Click upgrade links for full features.
