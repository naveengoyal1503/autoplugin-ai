# AI Content Optimizer Lite

A powerful **freemium** WordPress plugin that uses AI to analyze and optimize your post content for **better SEO** and readability. Free version offers basic analysis (3 scans/day); premium unlocks unlimited features.

## Features

### Free Version
- **Basic SEO Score**: Analyzes content length, keywords, and structure.
- **AI Suggestions**: Provides actionable tips to improve posts.
- **Post Meta Box**: Easy access from the WordPress editor.

### Premium Version ($4.99/month)
- Unlimited daily scans.
- **AI-Powered Rewrites**: Automatically rewrite content for optimal SEO.
- Advanced keyword research and integration.
- Priority support and regular updates.
- Export optimized content reports.

## Installation

1. Download the plugin ZIP file.
2. In your WordPress admin, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate the plugin.
4. The optimizer meta box appears automatically on post edit screens.

## Setup

1. After activation, edit any post to see the **AI Content Optimizer** sidebar.
2. No configuration needed for free version.
3. For premium, purchase at [example.com/premium](https://example.com/premium) and enter license key in **Settings > AI Optimizer**.

## Usage

1. Edit a post or page.
