# AI Content Optimizer Lite

## Description
**AI Content Optimizer Lite** is a powerful WordPress plugin that uses simulated AI to analyze your post content for **SEO**, **readability**, and **keyword density**. Get instant scores and suggestions to improve your content. Free version limited to 3 scans per site; upgrade to premium for unlimited access and advanced AI rewrites.

## Features
- **Content Analysis**: Word count, readability score (Flesch-like), keyword extraction, and overall SEO score.
- **Actionable Suggestions**: Tailored tips to optimize readability and keywords.
- **Post Integration**: Works directly in WordPress posts/pages.
- **Free Limit**: 3 analyses per site (resets on plugin deactivation).
- **Premium Upsell**: Unlimited scans, AI-powered rewrites, bulk optimization (via Freemius).

## Installation
1. Upload the `ai-content-optimizer` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Edit any post/page and look for the **Content Optimizer** tool under Posts > Content Optimizer or via meta box.

## Setup
- No configuration needed for free version.
- Premium: Purchase license at [example.com/premium](https://example.com/premium) and enter key in plugin settings (future update).

## Usage
1. Go to **Posts > Content Optimizer** or edit a post.
2. Paste or load content and click **Analyze**.
3. Review **score**, **metrics**, and **suggestions**.
4. Apply changes manually (premium auto-optimizes).
5. Free users: Limited to 3 scans. Upgrade for more!
