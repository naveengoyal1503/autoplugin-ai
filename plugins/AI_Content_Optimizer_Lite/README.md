# AI Content Optimizer Lite

A powerful **freemium** WordPress plugin that uses AI-inspired analysis to optimize your content for **SEO** and **readability**.

## Features
- **Free Version**:
  - Content analysis: Word count, Flesch Reading Ease score, top keywords.
  - Optimization suggestions.
  - 3 free scans per day.
- **Premium Features** (Upgrade for $4.99/month):
  - Unlimited scans.
  - Advanced AI suggestions and auto-rewrites.
  - SEO meta tag generator.
  - Bulk post optimization.
  - Priority support.

## Installation
1. Upload the `ai-content-optimizer-lite` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > AI Content Optimizer** to start analyzing.

## Setup
- No configuration needed for free version.
- Premium: Enter your license key after upgrade at [Premium Upgrade](https://example.com/premium-upgrade).

## Usage
1. Navigate to **Settings > AI Content Optimizer**.
2. Paste your post content into the textarea.
3. Click **Analyze Content** (limited to 3/day in free version).
4. Review results: readability score, keywords, and suggestions.
5. Apply suggestions manually or upgrade for automation.
