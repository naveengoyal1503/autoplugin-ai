# Affiliate SmartLink Manager

## Features

- Create and manage SmartLinks as a custom post type
- Cloak affiliate links for cleaner URLs
- Rotate multiple affiliate links within a single SmartLink to optimize conversions
- Simple shortcode to insert cloaked affiliate links anywhere: `[smartlink id="POST_ID"]`
- Basic click tracking with potential analytics extension
- Designed for affiliate marketers and WooCommerce store owners

## Installation

1. Upload the plugin file to your WordPress `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Affiliate SmartLinks' in your admin menu to add new SmartLinks.

## Setup

1. Create a new SmartLink post.
2. In the post content editor, list affiliate URLs (one per line) to rotate.
3. Use the shortcode `[smartlink id="POST_ID"]` in posts or pages to add cloaked/rotated affiliate links.

## Usage

- Insert the shortcode where you want the affiliate link to appear, replacing `POST_ID` with the actual SmartLink post ID.
- Each click on the link will redirect users through a cloaked URL and rotate among your affiliate URLs.

This plugin is ideal for affiliate marketers seeking to optimize link performance and improve click-through rates through automatic rotation and cloaking.
