# Affiliate Link Tracker Pro

## Features

- **Automatic Affiliate Link Detection**: Identifies links with affiliate parameters (ref=, aff=, tag=, etc.) in posts, pages, and menus.
- **Click Tracking**: Logs clicks with IP, user agent, referer, and timestamp.
- **Dashboard Analytics**: View top-performing links, click counts, and basic stats.
- **Freemium Model**: Free for basics; **Pro** unlocks unlimited tracking, advanced reports, conversion matching, and export.
- **Privacy Compliant**: Anonymizes IPs, GDPR-ready.
- **Easy Setup**: Works out-of-the-box, no configuration needed.

## Installation

1. Download the plugin ZIP.
2. In WordPress admin: **Plugins > Add New > Upload Plugin**.
3. Activate **Affiliate Link Tracker Pro**.
4. Visit **Affiliate Tracker** in the admin menu to view stats.

## Setup

- No setup required for basic tracking.
- **Pro Upgrade**: Click "Activate Pro" in dashboard (processes via Stripe: $9/month or $99/year).
- Tracked data stored in `wp_alt_clicks` table.

## Usage

1. Add affiliate links to your content normally (e.g., Amazon ?tag=yourid).
2. Plugin auto-detects and wraps them for tracking on frontend.
3. Clicks redirect seamlessly while logging data.
4. Monitor performance in **Affiliate Tracker > Dashboard**.
5. **Pro Users**: Access detailed reports, filter by date/link, CSV export.
