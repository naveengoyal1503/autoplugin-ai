# AffiliDeals Curator

AffiliDeals Curator is a WordPress plugin that **automatically curates and displays affiliate coupons and deals** tailored to your site's niche using AI-driven filters. Increase your affiliate revenue by showing your visitors the most relevant and high-converting deals.

## Features

- Automatically generates tailored affiliate coupon and deal listings based on your website's niche keywords.
- Easy-to-use shortcode `[affilideals_deals]` to display curated deals anywhere.
- Admin settings to configure affiliate API key, niche keywords, and number of deals to show.
- Caches deal data to improve performance and reduce API calls.
- Simple, clean, and responsive deal display.

## Installation

1. Upload the `affilideals-curator.php` file to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Settings > AffiliDeals Curator** to configure your API key and keywords.

- Use the shortcode `[affilideals_deals]` in any post, page, or widget area to display curated affiliate deals.
