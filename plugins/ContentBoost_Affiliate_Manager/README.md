# ContentBoost Affiliate Manager

## Overview

ContentBoost Affiliate Manager is a powerful WordPress plugin designed to help content creators, bloggers, and digital marketers monetize their websites through affiliate marketing. The plugin provides advanced link management, automatic keyword linking, and comprehensive click tracking analytics.

## Features

- **Affiliate Link Management**: Create, organize, and manage multiple affiliate links from a single dashboard
- **Click Tracking**: Real-time tracking of affiliate link clicks with detailed analytics
- **Auto-Linking Technology**: Automatically convert keywords in your content to affiliate links
- **Multiple Network Support**: Manage links from Amazon Associates, Commission Junction, and custom networks
- **Analytics Dashboard**: View comprehensive click statistics, click-through rates, and performance metrics
- **Shortcode Support**: Insert affiliate links using simple shortcodes
- **Easy Configuration**: Intuitive admin interface with customizable settings
- **No Coding Required**: User-friendly interface for non-technical users

## Installation

1. Download the plugin file (`contentboost-affiliate-manager.php`)
2. Log in to your WordPress admin dashboard
3. Navigate to **Plugins > Add New > Upload Plugin**
4. Select the plugin file and click **Install Now**
5. Click **Activate** to enable the plugin
6. Access the plugin from the WordPress admin menu under **ContentBoost**

## Setup

### Initial Configuration

1. Go to **ContentBoost > Settings**
2. Enable **Auto-Linking** if you want keywords automatically converted to affiliate links
3. Set the **Links per Post** limit (1-10, default: 3)
4. Save your settings

### Adding Affiliate Links

1. Navigate to **ContentBoost > Affiliate Links**
2. Fill in the following fields:
   - **Link ID**: A unique identifier (e.g., "amazon-headphones")
   - **Affiliate URL**: Your affiliate tracking URL
   - **Display Text**: Text shown to users (optional)
   - **Affiliate Network**: Select your network
   - **Auto-Link Keywords**: Enter keywords separated by commas
3. Click **Add Affiliate Link**

## Usage

### Using Shortcodes

Insert affiliate links in your posts or pages using the shortcode:


[contentboost_link id="amazon-headphones" text="Check price on Amazon"]


**Parameters:**
- `id` (required): The Link ID you created
- `text` (optional): Custom display text for the link

### Auto-Linking Keywords

If enabled in settings, keywords you've defined will automatically be converted to affiliate links throughout your content. The plugin respects your "Links per Post" limit to maintain natural content flow.

### Viewing Analytics

1. Go to **ContentBoost > Analytics**
2. View performance metrics:
   - Total clicks per link
   - Click-through rate (CTR)
   - Last clicked timestamp
3. Use this data to optimize your affiliate strategy
