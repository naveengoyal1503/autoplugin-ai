# Auto Affiliate Coupon Aggregator

## Description
Auto Affiliate Coupon Aggregator automatically fetches and displays affiliate coupons and deals from multiple sources on your WordPress site. It helps affiliate marketers and bloggers boost conversions by showcasing fresh offers with coupon codes and direct affiliate links.

## Features
- Automatically gathers coupons from multiple affiliate sources
- Caches coupons to optimize performance
- Easy shortcode [auto_affiliate_coupons] to display coupons anywhere
- Responsive, clean coupon display layout
- Filters out expired coupons automatically

## Installation
1. Upload the plugin file `auto-affiliate-coupon-aggregator.php` to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[auto_affiliate_coupons]` in posts, pages, or widgets.

## Setup
No additional setup required for the basic plugin version. Coupons are fetched from predefined demo sources. For premium versions, API keys and more sources will be available.

## Usage
- Place the shortcode `[auto_affiliate_coupons]` anywhere in your content to display the current coupon deals.
- Customize appearance by overriding CSS in your theme or child theme.
- Coming soon: premium add-ons for more sources and advanced customization.
