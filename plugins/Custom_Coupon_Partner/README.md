# Custom Coupon Partner

## Features

- **Easy Coupon Management**: Add custom promo codes, descriptions, and affiliate links via simple JSON in admin dashboard.[1][2]
- **Shortcode Integration**: Embed coupons anywhere with `[custom_coupon]` shortcode.
- **Conversion Boost**: Personalized discounts improve affiliate clicks and sales.[1][2]
- **Free Version**: Unlimited sites, first coupon display.
- **Premium ($49/year)**: Unlimited coupons, analytics, expiry dates, custom designs, API for brand partnerships.[7]

## Installation

1. Upload the plugin ZIP to your WordPress admin > Plugins > Add New > Upload.
2. Activate the plugin.
3. Go to **Custom Coupons** in the admin menu to configure.[1]

## Setup

1. In **Custom Coupons**, enter JSON array like:
   
   [
     {"code":"SAVE20","desc":"20% off first purchase","afflink":"https://affiliate-link.com"},
     {"code":"WELCOME10","desc":"$10 off wellness products","afflink":"https://partner-link.com"}
   ]
   
2. Save. Free version shows first coupon.[2]
3. Add `[custom_coupon]` to any post/page.

## Usage

- **Blogs/Affiliates**: Create exclusive deals to drive traffic and commissions.[1][3]
- **Customization**: Style with CSS targeting `.ccp-coupon`.
- **Monetization Tip**: Partner with brands for custom codes, list on dedicated pages for SEO.[2]
- **Shortcode Options** (Premium): `[custom_coupon id="1"]` for specific coupons.
