# Affiliate Accelerator

## Description
Affiliate Accelerator is a WordPress plugin that allows bloggers and affiliate marketers to easily manage affiliate links, automatically recommend affiliate products in content based on keywords, and track clicks for performance analysis. The plugin leverages basic AI-powered keyword matching to boost affiliate conversion rates.

## Features
- Manage and store affiliate links with associated target keywords
- Automatically recommend affiliate links inside posts via shortcode
- Tracks and logs affiliate link clicks for conversion tracking
- Simple admin interface for adding and viewing affiliate links
- Enqueue JavaScript to track clicks asynchronously

## Installation
1. Upload the `affiliate-accelerator.php` file to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Upon activation, the plugin creates a custom database table to store affiliate click data.

## Setup
1. Navigate to the **Affiliate Accelerator** menu in your WordPress admin dashboard.
2. Add your affiliate links and enter keywords that describe the product or service.

## Usage
- Insert the shortcode `[affiliate_accelerator_recommendations]` into your posts or pages where you want to display relevant affiliate link recommendations.
- When visitors click on these links, their clicks will be tracked automatically.
