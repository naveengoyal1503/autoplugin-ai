# WP Coupon Vault

## Features
- Create and manage exclusive coupons and deals.
- Display coupons via shortcode on any page or post.
- Easy-to-use admin interface for adding and editing coupons.
- Premium upgrade for unlimited coupons, analytics, and brand management.

## Installation
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Coupons > Add New to start adding coupons.

## Setup
- Add a new coupon by entering the title, description, coupon code, and affiliate URL.
- Use the shortcode `[coupon_vault limit="5"]` to display coupons on any page.

## Usage
- Place the shortcode on your deals page or sidebar to showcase coupons.
- Upgrade to premium for advanced features and unlimited coupons.
