# Smart Donation Booster

## Features
- **Animated Progress Bar**: Visually track donation goals with smooth animations.
- **One-Click PayPal Integration**: Easy donation buttons linked to your PayPal account [1][3].
- **Customizable Goals**: Set target amounts, current progress, and motivational text via admin settings.
- **Shortcode Ready**: Embed `[donation_goal]` anywhere on your site.
- **Mobile Responsive**: Works perfectly on all devices.
- **Freemium Model**: Free core; premium unlocks recurring donations, analytics, and custom themes.

## Installation
1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Donation Booster** to configure.

## Setup
1. Enter your **Goal Amount** (e.g., 1000).
2. Set **Current Amount** (starts at 0).
3. Add your **PayPal Email** for donations.
4. Customize **Goal Text** (e.g., "Fund our next project!") [1].
5. Save settings.

## Usage
- Insert `[donation_goal]` in any post, page, or widget.
- Progress auto-updates; manually adjust **Current Amount** or integrate PayPal IPN in premium.
- Example: Shows progress bar with "$250 / $1000 (25%)" and Donate button.
