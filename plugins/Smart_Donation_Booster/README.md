# Smart Donation Booster

## Features
- **Personalized donation prompts** with customizable text and AI-suggested amounts.
- **Real-time progress bar** tracking donations toward your goal.
- **One-click PayPal integration** for seamless payments (demo in free version).
- **Shortcode support**: `[smart_donation goal="1000" prompt="Support us!"]`.
- **Admin dashboard** to set goals and view totals (premium analytics coming soon).
- **Mobile-responsive** design for all devices.
- **Freemium model**: Upgrade for advanced features like email capture and A/B testing.

## Installation
1. Upload the `smart-donation-booster` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Smart Donation Booster** to set your goal.

## Setup
1. In the plugin settings, enter your donation goal (e.g., 1000).
2. Optionally, add your PayPal email for live payments (Pro feature).
3. Insert the shortcode `[smart_donation]` in any post, page, or widget.

## Usage
- Embed the widget anywhere with the shortcode.
- Visitors enter an amount and click 'Donate with PayPal'.
- Progress updates automatically on successful donations.
- Track totals in **Settings > Smart Donation Booster**.
