# Smart Donation Booster

## Description
A powerful, lightweight WordPress plugin to supercharge donations on your site. Features animated progress bars, customizable goals, and seamless PayPal integration. Perfect for bloggers, nonprofits, and creators turning content into recurring support.[1][3]

## Features
- **Donation Progress Bars**: Visual goal trackers with real-time progress.
- **One-Click PayPal Buttons**: Accept payments instantly without accounts.[3]
- **Customizable Shortcodes**: `[sdb_donation]` for easy embedding.
- **Admin Dashboard**: Set goals, track totals, and configure PayPal.
- **Mobile-Responsive**: Works perfectly on all devices.[2]
- **Freemium Upsell**: Pro version adds analytics, unlimited campaigns, and integrations.

## Installation
1. Upload the plugin files to `/wp-content/plugins/smart-donation-booster`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Donation Booster** to enter your PayPal email and set goals.
4. Add `[sdb_donation title="Support Our Work!"]` to any post/page.

## Setup
1. **Configure Settings**: Enter PayPal email (business account recommended), set goal (e.g., $1000), and current amount.
2. **Embed Shortcode**: Use `[sdb_donation]` or customize with `title` and `button_text` attributes.
3. **Test**: Visit the page, enter amount, and simulate donation.

## Usage
- Place shortcode in posts, pages, sidebars, or widgets.
- Progress auto-updates on donations (manual current amount adjustment in admin).
- Encourages tips/donations like 'buy me a coffee' models.[3]
- Track via admin panel; Pro unlocks detailed stats.
