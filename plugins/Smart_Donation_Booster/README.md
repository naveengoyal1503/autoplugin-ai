# Smart Donation Booster

## Features

- **Customizable Donation Forms**: Shortcode `[sdb_donate]` for easy embedding.
- **Progress Bars**: Visual goal tracking to encourage donations[1][3].
- **Suggested Amounts**: Pre-set buttons for quick selections (e.g., $5, $10)[1].
- **Exit-Intent Popups**: Capture leaving visitors for donations[2].
- **PayPal Integration**: One-click donations via PayPal[3].
- **Admin Dashboard**: Set goals, amounts, and PayPal email.
- **Freemium Model**: Pro upgrade for recurring payments, analytics, unlimited forms ($49/year)[1][5].

## Installation

1. Upload the plugin ZIP to your WordPress admin > Plugins > Add New.
2. Activate the plugin.
3. Go to **Settings > Donation Booster** to configure.
4. Add `[sdb_donate]` shortcode to any page/post.

## Setup

1. Enter your **PayPal email** (get a PayPal donate button ID for live mode).
2. Set **donation goal** (e.g., 1000) and **suggested amounts** (e.g., 5,10,25,50).
3. Enable **exit-intent popup** to boost conversions[2].
4. Test the form on a page.

## Usage

- Embed shortcode anywhere: `[sdb_donate]`.
- Progress bar auto-updates (Pro: real-time via donations).
- Visitors select amount and donate via PayPal.
- Track engagement; upgrade to Pro for full analytics and recurring options[5].
