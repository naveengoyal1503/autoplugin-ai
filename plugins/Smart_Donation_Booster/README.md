# Smart Donation Booster

A lightweight WordPress plugin to supercharge donations with animated progress bars, one-click PayPal integration, and goal tracking. Perfect for blogs, nonprofits, and creators.[1][3]

## Features
- **Animated Progress Bar**: Visual goal tracker updates in real-time.
- **PayPal One-Click Donations**: Custom amounts with secure redirects.
- **Admin Dashboard**: Set goals, track progress, choose currency.
- **Shortcode Ready**: `[donation_goal]` embeds anywhere.
- **Mobile Responsive**: Works on all devices.
- **Freemium Model**: Upgrade for multi-campaigns, analytics, custom branding ($29/year).

## Installation
1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Donation Booster** to configure goal and PayPal email.
4. Add `[donation_goal]` shortcode to any post/page.

## Setup
1. In admin settings:
   - Set **Goal Amount** (e.g., 1000).
   - Update **Current Amount** manually or via donations.
   - Select **Currency** (USD, EUR, GBP).
   - Replace `YOUR_PAYPAL_EMAIL` in plugin code with your PayPal email.
2. Save settings.

## Usage
- Embed shortcode in posts, pages, sidebars, or widgets.
- Visitors see progress bar and "Donate Now" button.
- Button reveals PayPal form for quick payments.
- Donations increase progress automatically (via AJAX in Pro).

**Pro Features**: Multiple goals, donation analytics, email thank-yous, WooCommerce integration.
