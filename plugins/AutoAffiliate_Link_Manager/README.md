# AutoAffiliate Link Manager

Automatically convert specified keywords within your WordPress posts and pages into affiliate links for effortless monetization.

## Features

- Easily specify keywords and their corresponding affiliate URLs.
- Automatic replacement of keywords with affiliate links in post content.
- Avoids replacing keywords already inside links.
- Open links in new tabs with proper nofollow and noopener attributes.
- Simple admin interface for managing keywords and links.
- Lightweight and performance-friendly.

## Installation

1. Upload the `autoaffiliate-link-manager.php` file to your `/wp-content/plugins/` directory.
2. Go to your WordPress admin dashboard and navigate to Plugins.
3. Activate the "AutoAffiliate Link Manager" plugin.

## Setup

1. In the WordPress dashboard, go to **Settings > AutoAffiliate Links**.
2. Enter your keywords and affiliate URLs in the format:
  
  keyword|https://your-affiliate-link.com
  another keyword|https://another-affiliate-link.com
  
3. Save changes.

## Usage

- The plugin will automatically replace the first occurrence of each keyword in your posts and pages with the matching affiliate link.
- Links open in a new tab for better user experience.
