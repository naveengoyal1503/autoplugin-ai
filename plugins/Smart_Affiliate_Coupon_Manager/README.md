## Installation

1. Download the plugin ZIP from your account.
2. In your WordPress dashboard, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate the plugin.
4. Navigate to **Settings > Affiliate Coupons** to add your first coupon.

## Setup

1. Go to **Settings > Affiliate Coupons** in your admin.
2. Enter coupons as JSON array (e.g., `{"name":"20% Off","code":"SAVE20","affiliate_link":"https://aff.link/?coupon=SAVE20","expires":"2026-12-31"}`).
3. Save settings.
4. Add `[sacm_coupons]` shortcode to any post, page, or widget.

## Usage

- **Display Coupons**: Place `[sacm_coupons limit="3"]` in content for a grid of top coupons.
- **Customization**: Style via CSS or upgrade to Pro for themes.
- **Tracking**: Clicks trigger GA4 events like `coupon_click`.
