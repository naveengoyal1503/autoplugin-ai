# Smart Affiliate Coupon Manager

## Features

- **Dynamic Coupon Display**: Shortcode `[affiliate_coupons]` to show coupons with click tracking.
- **Admin Dashboard**: Easily manage coupons, codes, discounts, and affiliate links.
- **Click Tracking**: Real-time click counts per coupon (free version limited to 10 coupons).
- **Categories Support**: Filter coupons by category (e.g., tech, fashion).
- **Pro Features**: Unlimited coupons, analytics dashboard, auto-coupon generation, conversion tracking ($49/year).

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Coupons** to add your affiliate coupons.
4. Use the shortcode `[affiliate_coupons]` in posts/pages.

## Setup

1. In the admin panel, navigate to **Settings > Coupons**.
2. Add coupon details: title, code, discount %, affiliate URL, category.
3. Save changes.
4. Embed shortcode: `[affiliate_coupons category="your-category" limit="5"]`.

## Usage

- **Shortcode Options**:
  - `category`: Filter by category (default: 'all').
  - `limit`: Max coupons to show (default: 5, free max: 10).
- View tracked clicks on coupon buttons.
- **Pro Upgrade**: Unlock unlimited coupons and advanced stats at [example.com/pro](https://example.com/pro).
