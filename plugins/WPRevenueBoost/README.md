# WPRevenueBoost

Automatically optimize ad placements, affiliate links, and content monetization for maximum revenue on any WordPress site.

## Features
- Easy-to-use admin settings
- Insert custom ad and affiliate codes
- Toggle monetization on/off
- Lightweight and non-intrusive

## Installation
1. Upload the `wprevenueboost.php` file to your WordPress plugins directory (`/wp-content/plugins/`).
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > WPRevenueBoost to configure your ad and affiliate codes.

## Setup
- Enter your ad code (e.g., Google AdSense) in the Ad Code field.
- Enter your affiliate code (e.g., Amazon Affiliate) in the Affiliate Code field.
- Enable monetization to start earning.

## Usage
- The plugin will automatically inject your codes into the footer of your site.
- Customize codes anytime from the settings page.
- Disable monetization if needed.
