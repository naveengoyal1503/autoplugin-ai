# WP Affiliate Booster

WP Affiliate Booster is a lightweight yet powerful WordPress plugin designed to help site owners optimize their affiliate marketing game by creating cloaked, trackable affiliate links with expiration and analytics.

## Features

- Add and manage cloaked affiliate links with user-friendly admin interface.
- Unique slugs for each affiliate link.
- Click tracking for every link.
- Optional expiration date for links to automatically disable outdated campaigns.
- Shortcode support to easily insert affiliate links in posts/pages.
- Clean, SEO-friendly cloaked URLs (e.g., yoursite.com/go/slug).
- Prevents direct exposure of long affiliate URLs, improving trust and click-through rates.

## Installation

1. Download the plugin PHP file.
2. Upload it to your WordPress plugin directory (`/wp-content/plugins/`).
3. Activate the plugin through the Plugins menu in WordPress.
4. Access the **Affiliate Booster** menu in your WordPress admin to start adding affiliate links.

## Setup

- Navigate to **Affiliate Booster** > **Add New Link**.
- Enter a unique slug (short identifier) for your affiliate link.
- Add the target affiliate URL.
- (Optional) Set an expiration date/time for this link.
- Save to create your cloaked affiliate link.

## Usage

- Use the generated cloaked URLs to share affiliate links.
- Place affiliate links in your content using the shortcode:
  
  [wpab_affiliate_link slug="your-slug" text="Click here to buy"]
  
- Track clicks directly from the admin panel.
