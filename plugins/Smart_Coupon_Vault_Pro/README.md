# Smart Coupon Vault Pro

## Features

- **Easy Coupon Display**: Use `[smart_coupon_vault limit="5"]` shortcode to show affiliate coupons.
- **Admin Dashboard**: Manage custom coupons via JSON in settings.
- **Affiliate Tracking**: Built-in links for commissions from networks like Amazon, CJ.
- **Pro Features** (paid upgrade): AI-powered coupon generation, click analytics, unlimited coupons, white-label, email capture.
- **Passive Income**: Attracts shoppers, boosts conversions with exclusive deals.[1][2]

## Installation

1. Upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > Coupon Vault** to add your affiliate API key and custom coupons.
4. Add `[smart_coupon_vault]` to any post/page.

## Setup

1. **Add Coupons**: In settings, paste JSON like:
   
   [
     {"code":"SAVE10","desc":"10% Off","afflink":"https://your-aff-link.com"}
   ]
   
2. **API Key**: Optional for Pro auto-fetching deals.
3. **Shortcode Options**: `[smart_coupon_vault limit="3"]`.
