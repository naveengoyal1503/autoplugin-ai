# WP Affiliate Sales Booster

## Description
WP Affiliate Sales Booster is a powerful WordPress plugin designed to help bloggers and niche website owners effortlessly increase their affiliate revenue. It allows you to create and manage dynamic affiliate coupon and deal pages that are fully customizable and optimized for conversions.

## Features

- Create unlimited affiliate coupons and deals with titles, descriptions, discounts, affiliate links, and categories.
- Display deals on any page or post using a simple shortcode `[affiliate_deals]`.
- Filter deals by category using shortcode attribute `[affiliate_deals category="categoryname"]`.
- Auto-insert random affiliate deal at the end of posts to boost clicks.
- Clean and responsive markup that fits any site style.
- Admin interface to easily add, edit, and manage coupons.
- Validates all input for security and reliability.

## Installation

1. Upload the `wp-affiliate-sales-booster.php` file to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the "Affiliate Deals" menu in the WordPress admin dashboard.
4. Add your affiliate coupons including title, description, discount, affiliate link, and category.

## Setup

- To display all deals, use the shortcode `[affiliate_deals]` on any post or page.
- To display deals filtered by a specific category, use `[affiliate_deals category="your-category"]`.
- The plugin will automatically insert a random deal to the end of each post to encourage visitor engagement.

## Usage

- Manage your coupons anytime from the WordPress admin menu "Affiliate Deals".
- Use the shortcode on posts, pages, or widgets that support shortcodes.
- Make sure your affiliate links are properly tracked with your affiliate program.

## FAQ

**Q: Can I add unlimited affiliate coupons?**  
A: Yes, there is no limit on the number of coupons you can add.

**Q: Is there an option for premium features?**  
A: This is a freemium-ready plugin concept; a premium version could add automatic coupon import, analytics, and advanced link cloaking.

**Q: Does it support all affiliate programs?**  
A: Yes, you just need to enter your affiliate URLs manually.
