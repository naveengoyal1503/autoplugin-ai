# Exclusive Affiliate Deals Manager

## Description
Create and display exclusive coupon deals to boost affiliate commissions. Generate unique promo codes, set discounts, and track performance. Perfect for bloggers and marketers.

## Features
- **Easy Deal Creation**: Add titles, affiliate links, coupon codes, discounts, and expiration dates via admin dashboard.
- **Beautiful Shortcodes**: Embed deals anywhere with `[exclusive_deal id="your-id"]`.
- **Responsive Design**: Eye-catching, mobile-friendly deal boxes.
- **Expiration Handling**: Automatically hides expired deals.
- **Affiliate Tracking**: Append referral params to links.
- **Freemium Model**: Free for up to 5 deals; Pro unlocks unlimited + analytics.

## Installation
1. Upload the `exclusive-affiliate-deals-manager` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Deals Manager** in your admin menu to create deals.

## Setup
1. Go to **Deals Manager**.
2. Fill in deal details (title, affiliate link, coupon code, discount %, expiration).
3. Save and copy the shortcode (e.g., `[exclusive_deal id="abc123"]`).
4. Paste shortcode in posts, pages, or widgets.

## Usage
- **Display Deals**: Use shortcode in Gutenberg blocks, Classic Editor, or sidebars.
- **Customization**: Deals auto-style; edit CSS in `style.css` for advanced tweaks.
- **Tracking**: Links append `?ref=exclusive`; upgrade to Pro for full stats.
