# Exclusive Affiliate Deals Manager

## Description
A powerful WordPress plugin that lets you create and display exclusive affiliate coupon deals to boost conversions and revenue. Perfect for bloggers and marketers.[1][2]

## Features
- **Easy Deal Management**: Add unlimited deals via simple admin interface (Pro: truly unlimited with tracking).
- **Shortcode Support**: Use `[exclusive_deals]` anywhere to display your coupons.
- **Responsive Design**: Mobile-friendly deal cards with copyable codes.
- **Affiliate Ready**: Direct links to your affiliate offers.
- **Freemium Model**: Free core features; Pro adds analytics, A/B testing, and API integrations.

## Installation
1. Upload the plugin files to `/wp-content/plugins/exclusive-deals-manager`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Deals Manager** to add your deals.
4. Use the shortcode `[exclusive_deals]` in posts/pages.

## Setup
1. In the admin page, enter deals in format: `Brand|Code|Affiliate URL|Description` (one per line).
2. Example:
   
   Amazon|SAVE20|https://your-affiliate-link|20% off electronics
   Shopify|DEAL10|https://shopify-aff-link|First month $1
   
3. Save and embed the shortcode.

## Usage
- **Frontend**: Deals display as styled cards with brand, description, coupon code, and affiliate button.
- **Customization**: Edit CSS/JS in `/assets/` folder.
- **Pro Upgrade**: Get click tracking, conversion stats, and more at example.com/pro.
