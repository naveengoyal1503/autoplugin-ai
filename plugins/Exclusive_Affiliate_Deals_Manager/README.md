# Exclusive Affiliate Deals Manager

A powerful WordPress plugin to create, manage, and display exclusive affiliate coupon codes and deals on your site. Boost conversions with unique offers from brands.[1][2]

## Features

- **Easy Deal Creation**: Add titles, custom codes, affiliate URLs, discounts, and expiry dates from the dashboard.
- **Trackable Links**: Monitor clicks on each deal for performance insights.
- **Shortcode Display**: Use `[exclusive_deal id="1"]` to embed beautiful, responsive deal boxes anywhere.
- **SEO-Friendly**: Optimized for search engines and AI crawlers.[2]
- **Freemium Model**: Free for basics; Pro adds unlimited deals, advanced analytics, email capture, and API support ($49/year).

## Installation

1. Upload the plugin ZIP to your WordPress admin > Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Navigate to **Deals Manager** in the admin menu to create your first deal.

## Setup

1. Go to **Deals Manager**.
2. Fill in deal details: title, unique code, affiliate URL, discount description, optional expiry.
3. Save. Note the deal ID.
4. Use shortcode `[exclusive_deal id="X"]` in posts/pages (replace X with ID).

## Usage

- **Embed Deals**: Place shortcodes in blog posts, sidebars, or dedicated pages for exclusive offers.[1][2]
- **Partner with Brands**: Request custom codes to make your site a go-to savings hub.[2]
- **Track Performance**: View clicks in the dashboard table.
- **Pro Upgrade**: For high-traffic sites, upgrade for detailed stats and more features.
