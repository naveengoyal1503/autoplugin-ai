# WP RevenueBoost

Automatically optimizes and manages multiple monetization streams for WordPress sites, increasing revenue through smart ad placement, affiliate link conversion, and dynamic content paywalls.

## Features
- Inject custom ad code into your site
- Automatically convert keywords to affiliate links
- Enable dynamic paywall for premium content
- Easy-to-use admin settings

## Installation
1. Upload the plugin file to your WordPress plugins directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the RevenueBoost settings page to configure your monetization options

## Setup
- Enter your ad code in the settings
- Add affiliate links as JSON (keyword => URL)
- Enable paywall and set the percentage of posts to protect

## Usage
- The plugin will automatically inject ads, convert affiliate links, and protect content based on your settings
- Premium features available in the pro version (subscription or one-time purchase)
