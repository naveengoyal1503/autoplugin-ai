# SmartAffiliateSuite

Comprehensive affiliate program management for WordPress sites with customizable commission structures, real-time tracking, and fraud detection.

## Features

- Custom post type for affiliates
- Add and manage affiliate profiles
- Real-time referral tracking
- Basic fraud detection (limits referrals per IP per day)
- Affiliate referral link shortcode
- Cookie-based affiliate tracking for 30 days
- AJAX-based referral counting

## Installation

1. Upload the `smartaffiliatesuite.php` file to your WordPress `/wp-content/plugins/` directory.
2. Activate the SmartAffiliateSuite plugin through the 'Plugins' menu in WordPress.

## Setup

1. Navigate to **SmartAffiliateSuite** in the WordPress admin menu.
2. Add new affiliates using the provided form.
3. Share affiliate links generated by `[sas_affiliate_link id="affiliate_id"]` shortcode. Replace `affiliate_id` with the correct affiliate post ID.

## Usage

- **Managing Affiliates:** View and manage affiliates from the plugin admin page.
- **Affiliate Links:** Use the shortcode on posts or pages to generate personalized affiliate links.
- **Tracking Referrals:** Referral visits are tracked automatically via cookies and AJAX calls.

## Monetization

SmartAffiliateSuite can be monetized with a freemium model where basic affiliate tracking is free, and subscription plans unlock advanced features such as:

- Detailed analytics dashboard
- Customizable commission tiers
- Priority customer support
- Integration with WooCommerce and other ecommerce platforms
