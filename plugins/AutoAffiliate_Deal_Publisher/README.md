# AutoAffiliate Deal Publisher

AutoAffiliate Deal Publisher helps affiliate marketers and deal bloggers automatically fetch and display fresh affiliate deals and coupons on their WordPress sites. This saves time on manual deal updating and increases affiliate sales through exclusive, dynamic coupon distribution.

## Features

- Automatically fetch and cache affiliate deals from a third-party deals API
- Display deals with coupon codes, descriptions, and expiry dates
- Use simple shortcode `[aafdp_deals]` to display deals anywhere
- Admin settings page to configure API key
- Regular hourly updates to keep deals fresh
- Clean, responsive deal display with affiliate tracking links

## Installation

1. Upload the plugin file to your WordPress `/wp-content/plugins/` directory or install via upload in the admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Settings > AutoAffiliate Deals** to enter your affiliate deals API key.

## Setup

- Obtain an API key from a supported affiliate deals provider or a custom affiliate deals API.
- Enter your API key in the plugin settings.
- The plugin will automatically fetch and cache deals hourly.

## Usage

- Insert the shortcode `[aafdp_deals]` in posts, pages, or widgets where you want to display affiliate deals.
- The deals will show with title, optional coupon codes, expiry date, and a link to the affiliate offer.
