# Smart Paywall Pro

**Effortlessly monetize your WordPress site with automatic paywalls on posts and pages.**

## Features

- **Simple Setup**: Add a price to any post/page via meta box; content locks behind payment.
- **One-Click Payments**: Simulated payments (pro: Stripe/PayPal integration).
- **User Tracking**: Purchases tied to user accounts; repeat access free.
- **Freemium Limits**: Free: 5 paywalled posts. **Pro**: Unlimited, subscriptions, analytics, tiers.
- **Mobile-Responsive**: Clean paywall UI works on all devices.
- **Shortcode Support**: `[paywall_unlock]` for custom placements.

## Installation

1. Upload the `smart-paywall-pro` folder to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Edit a post, set **Price** in the sidebar meta box.

## Setup

1. **Free Version**: Limited to 5 paywalled posts.
2. **Upgrade to Pro** ($29/year): Go to **Settings > Paywall Pro**. Enter license key for unlimited features, Stripe setup.
3. Customize teaser text per post.

## Usage

- **Enable Paywall**: In post editor, enter price (e.g., 4.99). Add teaser.
- **Test**: View post as logged-out user; paywall appears after teaser.
- **Pro Features**:
  - Recurring subscriptions.
  - Analytics dashboard: Views, conversions.
  - Tiered pricing.
- **Shortcode**: Use `[paywall_unlock post_id="123"]` anywhere.
