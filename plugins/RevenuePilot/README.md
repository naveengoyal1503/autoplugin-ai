# RevenuePilot

Automated revenue optimization plugin for WordPress sites.

## Features
- Auto-insert affiliate links in posts
- Easy ad code management
- Revenue tracking and analytics
- Simple admin interface

## Installation
1. Upload the plugin file to your WordPress plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

## Setup
1. Go to RevenuePilot in the admin menu.
2. Enter your affiliate ID and ad code.
3. Save settings.

## Usage
- Affiliate links will be added automatically to your posts.
- Ads will be displayed in the footer.
- Revenue tracking is enabled by default.
