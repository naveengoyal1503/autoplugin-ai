# Affiliate SmartLink Booster

## Description
Affiliate SmartLink Booster is a WordPress plugin designed to help bloggers and affiliate marketers increase their affiliate revenue by dynamically inserting affiliate links into post content based on keywords. It also tracks basic user interactions with affiliate links to enable smarter targeting in premium versions.

## Features
- Automatically inserts contextually relevant affiliate links into your posts.
- Supports customizable keyword-to-affiliate-URL mappings.
- Tracks user clicks and scrolls related to affiliate content for optimization.
- Lightweight and easy to use with zero configuration.
- Freemium-ready architecture for advanced behavior targeting.

## Installation
1. Upload the plugin file to your WordPress `/wp-content/plugins/` directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. The plugin will start inserting affiliate links automatically on single post pages.

## Setup
No setup required for the free version. To customize keywords or affiliate links, edit the plugin PHP code adding entries in the `$affiliate_links` array.

## Usage
- Write posts as usual. When your text contains predefined keywords (e.g., "WordPress", "hosting"), they will be automatically converted to affiliate links.
- Users clicking these links can be tracked for improved affiliate performance insights.

## Premium Version
The premium version will include:
- Admin interface for easy keyword and link management.
- User behavior analytics and heatmaps.
- Smart link insertion based on user interest.
- Integration with affiliate networks.
