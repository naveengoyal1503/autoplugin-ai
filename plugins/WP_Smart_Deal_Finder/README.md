# WP Smart Deal Finder

## Description
WP Smart Deal Finder is a WordPress plugin that automatically aggregates and displays personalized coupon codes and special offers tailored to your visitors based on their browsing behavior and your site content. It helps affiliate marketers, bloggers, and ecommerce sites increase affiliate income by showing relevant deals dynamically.

## Features
- Automatically detects user interests by analyzing page content and browsing patterns
- Displays personalized, affiliate-friendly coupon codes and discounts
- Easy shortcode usage: `[smart_deals]` to display deals anywhere
- Lightweight and self-contained single file plugin
- Extensible static deal mappings with plans for future API integration

## Installation
1. Download `wp-smart-deal-finder.php` and upload it to your `/wp-content/plugins/` directory.
2. Activate the plugin through the WordPress admin dashboard.
3. Insert the `[smart_deals]` shortcode into posts, pages, or widgets where you want deals to appear.

## Setup
No initial setup required. The plugin uses simple keyword extraction from page content to infer user interests and show relevant deals.

## Usage
- Place the shortcode `[smart_deals]` in any post or page content.
- Visitors will see a dynamic list of deals updated automatically based on their inferred interests.
- To customize deals, modify the `$all_deals` array in the plugin code or connect it to an external API/data source.
