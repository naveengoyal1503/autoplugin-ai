# ContentROI Tracker Pro

## Description

ContentROI Tracker Pro is a comprehensive analytics and monetization plugin for WordPress that helps content creators, bloggers, and site owners maximize revenue from affiliate marketing, display ads, and sponsored content. Track clicks, conversions, and revenue attribution in one centralized dashboard.

## Features

- **Link Tracking**: Create tracked affiliate links, ad links, and sponsored content links
- **Click Analytics**: Real-time tracking of clicks on all monetized links
- **Revenue Attribution**: Track which links generate the most revenue
- **Dashboard Analytics**: Visual charts and statistics for performance monitoring
- **Multiple Link Types**: Support for affiliate marketing, ads, and sponsored content
- **One-Click Link Generation**: Quickly generate trackable links with unique codes
- **Performance Insights**: Identify top-performing content and links
- **REST API**: Integrate tracking data with external tools
- **User-Level Isolation**: Each user can manage their own tracked links

## Installation

1. Upload the `contentroi-tracker` folder to `/wp-content/plugins/`
2. Activate the plugin from the WordPress Plugins menu
3. Navigate to ContentROI in the WordPress admin menu
4. Start creating tracked links from the "Manage Links" section

## Setup

### Initial Configuration

1. Go to **ContentROI > Settings**
2. Configure your API key (optional, for advanced integrations)
3. Set your preferred tracking parameters

### Creating Your First Tracked Link

1. Navigate to **ContentROI > Manage Links**
2. Click "Add New Link"
3. Enter the destination URL, link title, and select link type (Affiliate/Ad/Sponsored)
4. Click "Create" to generate a unique tracking code
5. Use the tracking code in your content

## Usage

### Adding Tracked Links to Your Posts

Add the tracking code to any link in your content:

html
<a href="https://your-product-link.com" data-contentroi-code="croi_abc123def456">Click Here</a>


### Viewing Analytics

1. Visit the **ContentROI Dashboard** to see overall statistics
2. Check individual link performance in **Manage Links**
3. Export data for deeper analysis

### Integration with Affiliate Networks

- Manually enter your affiliate URLs when creating links
- The plugin automatically wraps them with tracking code
- Monitor performance separately for each affiliate program
