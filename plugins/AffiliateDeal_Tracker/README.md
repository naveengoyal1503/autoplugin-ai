# AffiliateDeal Tracker

AffiliateDeal Tracker automatically collects affiliate deals and coupons from various affiliate networks, displaying them through customizable widgets to help WordPress site owners increase their affiliate revenue.

## Features

- Automatic aggregation of affiliate deals from multiple networks (simulated in free version).
- Hourly caching to reduce API calls and improve performance.
- Customizable widget to display latest deals in sidebar or any widget area.
- SEO-friendly and nofollow links for affiliate compliance.
- Easy setup with minimal configuration.

## Installation

1. Download the plugin file and upload it to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Appearance > Widgets and add the 'Affiliate Deal Tracker' widget to your sidebar or desired widget area.

## Setup

- Configure any API keys or network settings (future premium version).
- Customize widget title.
- The plugin will automatically fetch new deals every hour.

## Usage

- Place the widget in your sidebar or any widgetized area.
- Visitors will see fresh affiliate deals and coupons to click and purchase.
- Use the PRO version (coming soon) for advanced filtering, analytics, and integrations to maximize affiliate earnings.

*AffiliateDeal Tracker* aims to be a hands-off plugin to increase passive affiliate income by providing visitors with relevant deals automatically.
