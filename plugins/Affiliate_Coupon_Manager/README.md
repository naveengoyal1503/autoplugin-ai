# Affiliate Coupon Manager

## Description
Affiliate Coupon Manager allows you to create, manage, and display affiliate coupon codes easily on your WordPress site. Track usage of coupons and optimize your affiliate marketing revenue with simple analytics.

## Features
- Custom post type for managing coupons.
- Store affiliate URLs and coupon codes per coupon.
- Track coupon usage clicks.
- Shortcode `[affiliate_coupons]` to display all active coupons.
- Clean, minimal design for coupon display.
- Analytics on coupon usage count in admin.

## Installation
1. Upload the plugin PHP file to your `/wp-content/plugins/` directory.
2. Activate the plugin via the Plugins menu in WordPress.
3. You will see a new "Coupons" menu in the WordPress admin.

## Setup
1. In the WordPress admin, go to **Coupons > Add New**.
2. Enter the coupon title (e.g., "20% Off Sitewide").
3. Add details or description in the post content area.
4. Fill out the **Coupon Details** box below title with:
   - Affiliate Link URL: The URL for tracking clicks.
   - Coupon Code: The actual code users will use.
5. Publish the coupon.

## Usage
- To display coupons on any page or post, insert shortcode `[affiliate_coupons]`.
- Visitors clicking "Use Coupon" will be redirected to the affiliate link, and usage count will increment.

## Monetization & Upgrades
This plugin can be extended with premium features such as:
