# Affiliate Coupon Manager

Affiliate Coupon Manager is a WordPress plugin that lets you create, manage, and display exclusive affiliate-linked coupon codes to your visitors, helping you drive more affiliate sales and maximize revenue from your niche site.

## Features

- Custom post type for coupons with fields for affiliate URL, coupon code, expiry date.
- Easy-to-use shortcode `[acm_coupons]` to display active coupons on any page or post.
- Coupons automatically hide when expired.
- Responsive, simple coupon display with coupon code and call-to-action button.
- Lightweight, no external dependencies.

## Installation

1. Upload the plugin PHP file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the WordPress 'Plugins' menu.
3. Create new coupons under the "Coupons" menu.

## Setup

- Add a new coupon by entering the coupon title, description, affiliate link URL, coupon code, and optionally an expiry date.
- Save the coupon; it will be available for display.

## Usage

- Place the shortcode `[acm_coupons]` on any page or post where you want to show all active coupons.
- Visitors can click the "Use Coupon" button to go through your affiliate link and utilize the coupon code.

This plugin helps monetize your site by promoting affiliate deals with attractive coupons, increasing your chances of earning commissions from referred sales.
