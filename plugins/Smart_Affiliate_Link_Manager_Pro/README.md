# Smart Affiliate Link Manager Pro

## Description

**Smart Affiliate Link Manager Pro** is a powerful freemium WordPress plugin designed to help affiliate marketers cloak, track, and optimize their affiliate links effortlessly. The free version provides basic link cloaking and click tracking, while premium add-ons unlock advanced analytics, A/B testing, and integrations.

Boost your affiliate commissions with automated link management and conversion-boosting tools.

## Features

### Free Version
- **Automatic Link Cloaking**: Converts raw affiliate URLs in posts to trackable, cloaked links.
- **Click Tracking**: Logs clicks with IP, user agent, and referer data.
- **Shortcode Support**: Use `[affiliate_link url="https://affiliate.com"]` anywhere.
- **Dashboard Analytics**: View basic click stats in WordPress admin.

### Premium Add-ons (Coming Soon)
- **Advanced Analytics** ($49/year): Real-time reports, conversion tracking, top referrers.
- **A/B Testing** ($29/year): Test link variations to maximize clicks.
- **WooCommerce Integration** ($99/year bundle): Auto-track affiliate products in stores.

## Installation

1. Upload the `smart-affiliate-link-manager` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit **Settings > Affiliate Links** to configure options.

## Setup

1. **Enable Auto-Cloak**: Check the option in settings to automatically replace affiliate links in content.
2. **Manual Links**: Insert `[affiliate_link url="YOUR_AFFILIATE_URL"]` in posts/pages.
3. **View Stats**: Check your dashboard for click data.

## Usage

- **Posting Content**: Write normally with affiliate URLs; they auto-cloak on frontend.
- **Shortcodes**: Embed anywhere: `[affiliate_link url="https://amazon.com/product"]Your text[/affiliate_link]`.
- **Tracking**: All clicks redirect via tracker; stats appear in admin.
