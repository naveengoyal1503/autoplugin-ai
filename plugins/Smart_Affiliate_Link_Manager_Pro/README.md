# Smart Affiliate Link Manager Pro

## Features

**Free Version:**
- Create unlimited affiliate link aliases (e.g., `/go/amazon` redirects to your affiliate URL).
- Cloak links automatically in post content.
- Shortcode `[afflink id="amazon"]` for easy embedding.
- Click tracking with basic counters.
- Secure redirects with nonces to prevent abuse.

**Pro Version ($49/year):**
- Advanced analytics dashboard with click stats, geolocation, and device tracking.
- A/B testing for links.
- Auto-optimization: Rotate links based on performance.
- Link expiration and scheduling **bold**.
- Priority support and updates.

## Installation
1. Upload the plugin ZIP to your WordPress site via **Plugins > Add New > Upload**.
2. Activate the plugin.
3. Go to **Settings > Affiliate Links** to add your first link.

## Setup
1. In the admin panel (**Settings > Affiliate Links**), add an alias (e.g., `amazon`) and target affiliate URL.
2. Use the shortcode `[afflink id="amazon"]` in posts/pages, or type the alias directly (auto-cloaked).
3. Links redirect via `/go/amazon/` with tracking.

**Pro Upgrade:** Enter license key in settings for analytics.

## Usage
- **Embedding:** Use shortcode for control: `[afflink id="youralias"]`.
- **Auto-cloak:** Type `amazon` in content; it becomes clickable cloaked link.
- **Tracking:** View clicks in admin table (Pro: detailed reports).
- **Monetization:** Earn commissions faster with cloaked, trusted-looking links.
