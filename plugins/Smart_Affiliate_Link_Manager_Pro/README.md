# Smart Affiliate Link Manager Pro

## Description
Boost your affiliate earnings with automatic link cloaking, click tracking, and optimization tools. Free version for basics, Pro for advanced analytics and A/B testing.

## Features
- **Automatic Link Cloaking**: Converts affiliate URLs in posts to trackable shortcodes
- **Click Tracking**: Logs all clicks with timestamps (view in settings)
- **Shortcode Support**: `[afflink url="https://affiliate.com/ref=123"]`
- **Admin Dashboard**: Manage links and view basic stats
- **Pro Features** (paid add-ons):
  * A/B Testing ($49/year)
  * Detailed Analytics Dashboard ($99/year)
  * Unlimited Links & Integrations ($199/year)

## Installation
1. Upload the plugin files to `/wp-content/plugins/smart-affiliate-manager`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit **Settings > Affiliate Manager** to configure

## Setup
1. Go to **Settings > Affiliate Manager**
2. Add your affiliate links (one per line)
3. Insert `[afflink url="YOUR_AFFILIATE_URL"]` in posts or let auto-cloaking handle it
4. Upgrade to Pro for advanced features: [Get Pro](https://example.com/pro)
