# Smart Affiliate Link Manager Pro

## Description

**Smart Affiliate Link Manager Pro** is a powerful WordPress plugin that automatically detects, cloaks, tracks, and optimizes your affiliate links. Increase conversions with link shortening, click tracking, and pro features like A/B testing and analytics.

**Free Version:** Cloaks up to 10 affiliate links per post with basic click counting.

**Pro Version:** Unlimited links, advanced analytics, A/B testing, geo-targeting, and more. [$49/year](https://example.com/pro)

## Features

### Free Features
- **Auto-Link Cloaking:** Automatically shortens affiliate links in posts (e.g., yoursite.com/go/abc123 → amazon.com/product)
- **Click Tracking:** Counts clicks on cloaked links
- **Link Limit:** 10 links per site
- **Mobile Responsive**

### Pro Features (Add-on Purchase)
- **Unlimited Links**
- **Detailed Analytics:** Clicks by IP, time, device
- **A/B Testing:** Rotate between multiple affiliate links
- **Geo-Targeting:** Show links based on visitor location
- **Conversion Tracking:** Integrate with Google Analytics
- **Priority Support**

## Installation

1. Download the plugin ZIP file.
2. In your WordPress dashboard, go to **Plugins > Add New > Upload Plugin**.
3. Upload and activate the plugin.
4. Visit **Settings > Affiliate Manager** to configure.

## Setup

1. Go to **Settings > Affiliate Manager**.
2. Set your free link limit (default: 10).
3. (Pro) Enter your API key after purchase.
4. Add affiliate links to your posts—they auto-cloak!

## Usage

- Write posts with raw affiliate links (e.g., https://amazon.com/product123).
- Plugin auto-converts to `yoursite.com/go/abc123`.
- View stats in **Settings > Affiliate Manager**.
- Upgrade to Pro for full analytics dashboard.
