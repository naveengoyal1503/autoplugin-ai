# Smart Affiliate Link Manager Pro

## Overview

Smart Affiliate Link Manager Pro is a powerful WordPress plugin designed to help content creators and publishers monetize their websites through advanced affiliate marketing. With geolocation targeting, automatic link insertion, and comprehensive analytics, this plugin makes it easy to manage and optimize your affiliate links for maximum revenue.

## Features

### Core Features (Free Version)
- **Unlimited Affiliate Links**: Create and manage unlimited affiliate links
- **Link Slug Management**: Use custom, memorable slugs for easy link management
- **Basic Analytics**: Track clicks on all affiliate links
- **Geolocation Targeting**: Target specific countries with different affiliate links
- **Shortcode Support**: Easily embed affiliate links using `[affiliate_link]` shortcode
- **Click Tracking**: Automatic tracking of all link clicks with IP addresses and device types
- **Category Organization**: Organize links by category for better management

### Premium Features
- Advanced analytics dashboard with detailed reports
- Conversion tracking and ROI calculations
- A/B testing for link optimization
- Automatic link insertion in post content
- Priority email support
- White-label options

## Installation

1. Download the plugin file
2. Log in to your WordPress dashboard
3. Go to **Plugins > Add New > Upload Plugin**
4. Choose the plugin file and click **Install Now**
5. Activate the plugin from the Plugins menu
6. Navigate to **Affiliate Links** in the admin menu to get started

## Setup

### Initial Configuration

1. Go to **Affiliate Links > Settings**
2. Enable or disable click tracking as needed
3. Review your license type (Free or Premium)

### Creating Your First Link

1. Click **Affiliate Links > Create Link**
2. Fill in the following fields:
   - **Link Slug**: A unique identifier (e.g., `amazon-books`)
   - **Destination URL**: The affiliate URL you want to redirect to
   - **Affiliate Code**: Your unique affiliate/referral code
   - **Category**: Organize links by category (e.g., `Books`, `Electronics`)
   - **Geolocation Targeting**: Optional - target specific countries
3. Click **Create Link**

## Usage

### Displaying Affiliate Links

Use the shortcode to add affiliate links to your posts and pages:


[affiliate_link slug="amazon-books" text="Buy on Amazon"]


**Parameters:**
- `slug` (required): The link slug you created
- `text` (optional): The text to display for the link (default: "Click here")

### Viewing Analytics

1. Go to **Affiliate Links > Analytics**
2. View all your created links with click counts
3. Click **View** on any link to see detailed statistics
4. Monitor performance by country, device type, and time period

### Dashboard Overview

The main dashboard displays:
- Total number of affiliate links
- Total clicks across all links
- Your current license type
- Quick access to create new links
