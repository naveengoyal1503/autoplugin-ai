# Smart Affiliate Booster

## Description
**Smart Affiliate Booster** is a powerful WordPress plugin that automatically detects keywords in your posts and injects high-converting Amazon affiliate links. Boost your affiliate earnings by up to 30% with smart optimization and performance tracking. Perfect for bloggers and content creators.

## Features
- **Auto-Link Insertion**: Automatically adds Amazon affiliate links to matching keywords in posts.
- **Custom Keywords**: Define your own keywords for precise targeting.
- **Performance Tracking**: Monitor clicks and optimize (Pro).
- **Mobile-Responsive**: Links work perfectly on all devices.
- **Easy Setup**: Enter your Amazon tag and keywords in seconds.
- **Pro Add-ons**: AI keyword suggestions, analytics dashboard, WooCommerce integration, unlimited sites.

## Installation
1. Upload the `smart-affiliate-booster` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Affiliate Booster** to configure your Amazon tag and keywords.
4. Enable auto-insertion and publish posts—links appear automatically!

## Setup
1. Get your Amazon Associates tag from [affiliate-program.amazon.com](https://affiliate-program.amazon.com).
2. In plugin settings:
   - Paste your **Amazon Affiliate Tag**.
   - Add **keywords** (e.g., 'best laptop', 'wireless headphones') one per line.
   - Check **Enable Auto-Insertion**.
3. Save settings. New and existing posts will get links.

## Usage
- Write content naturally using your keywords.
- The plugin scans posts and wraps keywords in affiliate links (e.g., **best laptop** → Amazon search link with your tag).
- Links include `nofollow sponsored` for compliance.
- **Pro Tip**: Use Pro for click stats to refine keywords and maximize commissions.
