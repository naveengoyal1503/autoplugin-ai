# ContentMoat

## Automated Content Repurposing & Multi-Channel Monetization for WordPress

ContentMoat is a powerful WordPress plugin that transforms your blog posts into multiple content formats while providing integrated tools for managing affiliate links, sponsored content, and revenue tracking across all channels.

## Features

### Content Conversion
- **YouTube Scripts**: Automatically convert blog posts into structured YouTube video scripts with hooks and CTAs
- **Twitter Threads**: Transform articles into engaging Twitter thread formats
- **Newsletter Content**: Generate newsletter-optimized versions of your posts
- **Podcast Scripts**: Create podcast episode scripts with intro, main content, and outro sections

### Monetization Management
- **Affiliate Link Tracking**: Centralize and manage all affiliate links across converted content
- **Sponsored Content**: Track which brands sponsor your content across different formats
- **Donation Integration**: Enable "Buy Me a Coffee" style donations on posts
- **Revenue Dashboard**: Track total revenue generated from each post and channel

### Analytics & Insights
- Real-time conversion statistics
- Monthly usage tracking against your plan limits
- Revenue attribution by format and channel
- Content performance metrics

### Subscription Plans
- **Free**: 5 conversions per month
- **Premium**: Unlimited conversions, advanced analytics ($19/month)
- **Agency**: White-label options, API access, custom integrations ($99/month)

## Installation

1. Download the ContentMoat plugin zip file
2. Go to WordPress Admin Dashboard → Plugins → Add New
3. Click "Upload Plugin" and select the ContentMoat zip file
4. Click "Install Now" then "Activate Plugin"
5. Navigate to ContentMoat menu in the left sidebar

## Setup

1. Go to **ContentMoat → Settings**
2. Enter your ContentMoat API Key (get one at contentmoat.io)
3. (Optional) Add your OpenAI API Key for enhanced AI conversions
4. Select your subscription plan
5. Click "Save Settings"

## Usage

### Converting Posts

1. Edit any WordPress post
2. Scroll to the **ContentMoat Converter** meta box
3. Click any of the format buttons:
   - "Convert to YouTube Script"
   - "Convert to Twitter Thread"
   - "Convert to Newsletter"
   - "Convert to Podcast Script"
4. The conversion is automatically saved to your dashboard
5. View and download converted content from the ContentMoat Dashboard

### Managing Monetization

1. Edit a post and scroll to the **Monetization Settings** box on the right
2. Enable donations if desired
3. Paste your affiliate links in the "Affiliate Links" field
4. Add sponsored brand names in the "Sponsored Brands" field
5. Click "Save Monetization"
6. These settings apply to all converted versions of this post

### Dashboard Monitoring

Visit **ContentMoat → Dashboard** to see:
- Total conversions created
- Monthly usage vs. plan limit
- Total revenue tracked
- Current subscription plan

## FAQ

**Q: Can I edit conversions before publishing?**
A: Yes. All conversions are stored in the dashboard and can be edited before distributing to each platform.

**Q: Does ContentMoat support custom API integrations?**
A: Agency plan subscribers have full API access for custom integrations. Contact support for details.

**Q: How accurate are the AI-generated conversions?**
A: Conversions use template-based formatting. For best results, manually review and customize conversions for each platform.

**Q: Can I import conversions from other plugins?**
A: Not currently, but this is on our roadmap. Export your content and paste it into ContentMoat.

**Q: What payment methods do you accept?**
A: We accept all major credit cards, PayPal, and wire transfers for annual plans.
