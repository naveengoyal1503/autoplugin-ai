# ContentMoat - WordPress Membership & Paywalls Plugin

## Overview

ContentMoat is a powerful WordPress plugin designed to help content creators, educators, and publishers monetize their content through exclusive member-only sections, tiered subscription plans, and recurring revenue management.

## Features

- **Exclusive Content Management**: Create paywalled posts with subscription tier requirements
- **Tiered Membership Plans**: Set up multiple subscription levels with different access permissions
- **Member Dashboard**: Personal dashboard for subscribers to manage their subscriptions
- **Payment Integration**: Ready for Stripe, PayPal, and other payment gateways
- **Member Directory**: Admin panel to manage all members and their subscriptions
- **Revenue Tracking**: Dashboard to monitor recurring revenue and subscriber growth
- **Email Integration**: Works with Mailchimp, ConvertKit, and Constant Contact
- **Shortcodes**: Easy-to-use shortcodes for login forms, dashboards, and subscription plans
- **Customizable Tiers**: Create unlimited subscription tiers with custom pricing

## Installation

1. Download the ContentMoat plugin file
2. Upload to `/wp-content/plugins/contentmoat/`
3. Activate the plugin through WordPress admin
4. Navigate to ContentMoat > Settings to configure

## Setup

### Initial Configuration

1. Go to **ContentMoat > Settings**
2. Enter your payment provider credentials (Stripe, PayPal, etc.)
3. Create subscription tiers in **ContentMoat > Edit Tiers**
4. Add tier pricing and descriptions

### Creating Exclusive Content

1. Create a new post in **ContentMoat > Exclusive Content**
2. Write your content
3. Check "Make this content exclusive" in the ContentMoat Settings box
4. Select the required subscription tier
5. Publish

## Usage

### Shortcodes

**Login Form**

[contentmoat_login_form]


**Member Dashboard**

[contentmoat_member_dashboard]


**Subscription Plans**

[contentmoat_subscription_plans]


### Admin Panels

- **Members**: View all subscribers, their tiers, and manage access
- **Revenue**: Track monthly recurring revenue (MRR) and subscriber metrics
- **Exclusive Content**: Create and manage paywalled posts

## Monetization Models Supported

- Tiered subscriptions (Basic, Professional, Premium)
- Pay-per-article or one-time purchases
- Freemium model (free + premium content)
- Community access fees

## Requirements

- WordPress 5.0+
- PHP 7.2+
- Active payment gateway (Stripe, PayPal recommended)
