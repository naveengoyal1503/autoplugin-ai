# Affiliate Deal Spotlight

Affiliate Deal Spotlight is a lightweight WordPress plugin that automatically displays geo-targeted affiliate coupon deals and discount offers to your site visitors. Designed to boost your affiliate earnings by showing the most relevant deals based on visitor location with built-in click tracking.

## Features

- Automatically detects visitor country via IP and shows region-specific affiliate deals
- Supports multiple affiliate deals with custom titles, descriptions, and URLs
- Click tracking to measure performance of each deal link
- Simple shortcode `[affiliate_deals]` to display deals anywhere on your site
- Responsive and lightweight design with built-in styling
- Provides fallback messaging if no deals fit visitor location
- Easily extendable to integrate with affiliate APIs or add premium features

## Installation

1. Upload the `affiliate-deal-spotlight.php` file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place the shortcode `[affiliate_deals]` in posts, pages, or widgets where you want the deal spotlight to appear.

## Setup

- The plugin automatically detects visitor country and displays relevant deals. No additional setup is required.
- To customize deals, modify the sample deals array inside the plugin PHP file or extend with an admin settings page (future updates).

## Usage

- Insert `[affiliate_deals]` shortcode in your content.
- Visitors will see coupon deals tailored to their geo-location to maximize engagement and conversions.
- The plugin tracks clicks on each deal link for performance insights.
