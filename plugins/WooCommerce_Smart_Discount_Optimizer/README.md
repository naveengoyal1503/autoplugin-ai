# WooCommerce Smart Discount Optimizer

## Description
Automatically applies dynamic, personalized discounts based on your customersâ€™ purchase behavior to increase sales and encourage loyalty.

## Features

- Tiered discount system:
  - 5% discount for 1+ previous orders
  - 10% discount for 5+ previous orders
  - 15% discount for 10+ previous orders
- Seamlessly integrates with WooCommerce cart pricing
- Lightweight and self-contained in a single PHP file
- Ready for advanced features in future premium versions

## Installation

1. Go to your WordPress dashboard.
2. Navigate to Plugins > Add New.
3. Click 'Upload Plugin' and choose the plugin PHP file.
4. Install and activate the plugin.
5. Ensure WooCommerce plugin is active and configured.

## Setup

No additional setup required for basic functionality. Discounts are automatically applied at checkout based on user purchase history.

## Usage

- Customers will see discounted prices automatically if they have qualifying purchase history.
- Monitor your WooCommerce orders to see the impact.
