# Affiliate Deal Vault

Affiliate Deal Vault is a WordPress plugin designed to help bloggers, marketers, and affiliate websites easily create, manage, and display exclusive affiliate coupon deals and discounts to boost affiliate commission earnings and user engagement.

## Features

- Intuitive admin interface to add/edit affiliate coupon deals
- Set deal title, description, affiliate link, and optional expiry date
- Shortcode `[affiliate_deals]` to display active deals anywhere on your site
- Automatically hides expired deals
- Clean, simple frontend display with call-to-action buttons
- Secure, sanitized input and output
- Lightweight and self-contained plugin

## Installation

1. Upload the plugin PHP file to your WordPress `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

## Setup

1. In the WordPress admin menu, go to **Affiliate Deal Vault**.
2. Use the form to add your affiliate deals - provide title, description, affiliate URL, and (optionally) expiry date.
3. Click "Save Deals" to store your deals.

## Usage

- Place the shortcode `[affiliate_deals]` in any post, page, or widget area to show your active affiliate deals list.
- Expired deals will automatically be hidden.

You can customize the CSS by overriding styles in your theme if desired.

Enjoy increasing your affiliate revenue with a modern, easy coupon deal display on your WordPress site!
