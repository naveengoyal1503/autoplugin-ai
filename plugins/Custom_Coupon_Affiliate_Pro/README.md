# Custom Coupon Affiliate Pro

## Features

- **Easy Coupon Creation**: Add custom coupons with unique codes, descriptions, affiliate links, discounts, and expiration dates.[1][2]
- **Click Tracking**: Built-in analytics to monitor coupon usage and performance directly in WordPress admin.[3]
- **Shortcode Integration**: Display coupons anywhere with `[coupon_display id="YOURCODE"]`.[1]
- **SEO-Friendly URLs**: Trackable coupon links like `/coupon/CODE/track/` for better conversions.[2]
- **Freemium Upsell**: Premium features include unlimited coupons, advanced dashboard, and auto-emails ($49/year).[7]
- **Mobile-Responsive Design**: Beautiful, customizable coupon cards with hover effects.

## Installation

1. Download and upload the plugin ZIP to your WordPress admin > Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Go to **Settings > Coupon Pro** to add your first coupon via JSON or Pro interface.

## Setup

1. In **Settings > Coupon Pro**, enter coupons as JSON array (see plugin for example).
2. Save settings. Database table auto-creates on activation.
3. Copy the coupon code (e.g., "TEST123").
4. Add shortcode `[coupon_display id="TEST123"]` to any post/page.
5. Premium: Upgrade for bulk import, API integrations, and detailed reports.

## Usage

- **Display Coupons**: Use shortcode in posts, pages, or widgets.
- **Track Performance**: View clicks in admin dashboard table.
- **Affiliate Integration**: Set affiliate URLs; plugin redirects and logs clicks.[1][3]
- **Customization**: Style via CSS classes like `.coupon-card`, `.coupon-btn`.
- **Pro Tips**: Partner with brands for exclusive codes to boost engagement and revenue.[2]
