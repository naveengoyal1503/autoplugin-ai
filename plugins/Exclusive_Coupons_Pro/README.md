# Exclusive Coupons Pro

## Features
- **Easy Coupon Management**: Add unlimited exclusive coupon codes, affiliate links, and descriptions via simple textarea in WordPress admin.
- **Shortcode Integration**: Use `[exclusive_coupon]` for random or `[exclusive_coupon id="1"]` for specific coupons anywhere on your site.
- **Conversion-Boosting Design**: Eye-catching, mobile-responsive coupon boxes with styled buttons and codes.
- **Freemium Model**: Free for basics; Pro unlocks click tracking, analytics, custom branding, and API integrations.
- **SEO-Friendly**: Generates unique value for readers, improving engagement and affiliate earnings.[1][2]

## Installation
1. Upload the `exclusive-coupons-pro` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Coupons Pro** to configure your first coupons.

## Setup
1. In **Settings > Coupons Pro**, enter coupons in this format:
   
   Code: SAVE20
   Affiliate Link: https://your-affiliate-link.com
   Description: 20% off first purchase
   ---
   Code: WELCOME10
   Affiliate Link: https://another-link.com
   Description: $10 off your order
   
2. Save changes.
3. Add shortcode to any post/page: `[exclusive_coupon]`.

## Usage
- **Display Coupons**: Embed shortcodes in posts, pages, sidebars, or widgets.
- **Customization**: Style via CSS targeting `.ecp-coupon` class.
- **Monetization Tips**: Partner with brands for custom codes to boost conversions.[2] Track performance manually (Pro: automated).
- **Pro Upgrade**: For advanced features like usage limits, A/B testing, and integrations with WooCommerce/Affiliate plugins.
