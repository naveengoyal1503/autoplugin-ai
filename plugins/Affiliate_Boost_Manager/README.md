# Affiliate Boost Manager

## Description
Affiliate Boost Manager is a WordPress plugin designed to help bloggers and affiliate marketers optimize their affiliate marketing efforts by providing link cloaking, click tracking, and a simple personalized coupon code generation feature.

## Features
- Add and manage cloaked affiliate links
- Automatic click tracking per affiliate link
- Display affiliate links with coupons using a shortcode
- AJAX-based coupon code generator

## Installation
1. Upload `affiliate-boost-manager.php` to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the Affiliate Boost Manager menu in your WordPress admin.

## Setup
1. Navigate to the Affiliate Boost Manager admin panel.
2. Add your affiliate links, optionally adding coupon codes.
3. Use the shortcode `[abm_affiliate_links]` in any post or page to display your affiliate links with coupons.

## Usage
- Add new affiliate links via the admin page.
- Embed your affiliate links on posts/pages using the shortcode.
- Track clicks through the admin panel.
- Use the 'Generate Coupon' button to create random coupon codes for promotions.

## Monetization
This plugin uses a freemium model:
- Basic features are free.
- Premium features such as detailed analytics, multi-user affiliate management, and automated advanced coupon generation available via subscription add-ons.
