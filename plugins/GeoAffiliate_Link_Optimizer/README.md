# GeoAffiliate Link Optimizer

## Description
GeoAffiliate Link Optimizer automatically manages your affiliate links by geolocation, scheduling, and categories to maximize conversions and revenue. It replaces keywords in your post content with affiliate links tailored to visitors' countries and allows easy management via the admin panel.

## Features
- Automatic replacement of keywords with affiliate links in post/page content
- Geolocation targeting to show region-specific affiliate URLs
- Scheduled link activation dates to run limited-time promotions
- Link categorization for organization and filtering
- Admin UI for managing affiliate keywords and links
- Shortcode `[geoaffiliate_links category=""]` to display affiliate links by category

## Installation
1. Upload the `geoaffiliate-link-optimizer.php` file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'GeoAffiliate Links' admin page to add and manage your affiliate links.

## Setup
- Add affiliate keywords and corresponding URLs.
- Optionally set geolocation country codes (e.g., US,CA) to target visitors from those countries.
- Use schedule fields to define active periods for each link.
- Save the settings.

## Usage
- The plugin will automatically replace the first occurrence of each keyword in your content with the configured affiliate link, considering geolocation and schedules.
- Use shortcode `[geoaffiliate_links]` anywhere to list all affiliate links.
- Filter by category with `[geoaffiliate_links category="categoryname"]`.

## Monetization
The base plugin is free and included features support common affiliate marketing tactics.
Advanced functionality can be added via paid upgrades, such as enhanced analytics, multiple replacement per post, and priority geolocation logic.
