# WP Course Affiliate Bridge

## Features
- Create and manage courses as custom post types
- Manage affiliates as custom post types
- Shortcode to display course list
- Shortcode for affiliate sign-up
- Admin dashboard for plugin management

## Installation
1. Upload the plugin file to your WordPress plugins directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the plugin settings page to manage courses and affiliates

## Setup
- Use the admin dashboard to add courses and affiliates
- Use the provided shortcodes to display course lists and affiliate sign-up forms on your site

## Usage
- Add `[course_list]` to any page or post to display all courses
- Add `[affiliate_signup]` to any page or post to display the affiliate sign-up form
