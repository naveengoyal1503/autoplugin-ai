# Affiliate Smart Coupons

Affiliate Smart Coupons is a WordPress plugin designed to help bloggers and affiliate marketers increase their revenue by automatically aggregating and displaying affiliate coupons on their website.

## Features

- Add and manage affiliate coupons with title, description, affiliate link, and optional expiry date.
- Automatically exclude expired coupons from public display.
- Display coupons anywhere on your site via shortcode `[affiliate_coupons]`.
- Clean, simple coupon layout optimized for conversions.
- Admin interface to add, view, and manage coupons.
- Scheduled infrastructure ready for future automated coupon imports.

## Installation

1. Upload the `affiliate-smart-coupons.php` file to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'Affiliate Coupons' menu in the WordPress admin panel.

## Setup

- Go to the 'Affiliate Coupons' admin page.
- Add new coupons by filling out the form with the coupon title, description, affiliate link, and (optionally) expiry date.
- Save the coupons.

## Usage

- Insert the shortcode `[affiliate_coupons]` into any page or post where you want to display the list of active coupons.
- Visitors will see current coupon deals with a clickable 'Get Deal' button linking to your affiliate URL.

## Monetization
