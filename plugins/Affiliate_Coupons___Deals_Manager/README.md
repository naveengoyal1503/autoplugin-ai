# Affiliate Coupons & Deals Manager

## Description
Affiliate Coupons & Deals Manager lets you create, manage, and display affiliate coupons and deals easily. Boost your affiliate marketing conversions by showcasing active and compelling discount offers with optional discount codes.

## Features
- Custom post type to add coupons and deals with affiliate URLs
- Optional discount code field
- Expiry date support to automatically hide expired offers
- Active/inactive toggle for each coupon
- Shortcode `[affiliate_coupons count="x"]` to display coupons anywhere
- Widget to display latest active coupons in sidebars
- Responsive styling for coupon display

## Installation
1. Upload the plugin file to your `/wp-content/plugins/` directory.
2. Activate the plugin through the WordPress admin dashboard.
3. Add new coupons under the 'Affiliate Coupons' menu.

## Setup
- When adding a coupon, enter the title, description, affiliate URL, optional discount code, and optional expiry date.
- Set the coupon as active to display on the front end.

## Usage
- Use the shortcode `[affiliate_coupons count="5"]` in any post or page to show up to 5 active coupons.
- Add the Affiliate Coupons widget in Appearance > Widgets to show coupons in sidebar or footer areas.
