# ContentMorph Pro

Automatically convert your high-performing WordPress blog posts into multiple content formats with AI-powered optimization and one-click publishing.

## Features

- **Multi-Format Conversion**: Transform posts into video scripts, social media carousels, email sequences, and infographic briefs
- **One-Click Publishing**: Easily copy and paste converted content to your WordPress editor
- **Freemium Model**: 5 free conversions per month, upgrade for unlimited access
- **Post Meta Integration**: Access conversion tools directly from the post editor
- **Conversion History**: Track all conversions with timestamps and formats used
- **Tiered Subscriptions**: Free, Premium, and Enterprise plans for different needs
- **Dashboard Analytics**: Monitor monthly conversion usage

## Installation

1. Download the plugin files
2. Upload the plugin folder to `/wp-content/plugins/`
3. Activate the plugin through the WordPress admin panel
4. Navigate to **ContentMorph Pro** in the left admin menu

## Setup

1. After activation, the plugin creates a database table for tracking conversions
2. Go to **ContentMorph Pro > Settings** to view upgrade options
3. The plugin defaults to the Free tier (5 conversions/month)
4. To upgrade, click the upgrade button and complete payment via Stripe integration

## FAQ

**Q: Can I upgrade anytime?**
A: Yes, you can upgrade your plan at any time from the Settings page.

**Q: What happens if I exceed my monthly limit on Free plan?**
A: You'll receive a notification to upgrade. Your conversions are saved and accessible.

**Q: Can I use converted content commercially?**
A: Yes, all converted content is yours to use for commercial purposes.

**Q: Is there a limit to post length?**
A: Posts up to 10,000 words are supported. Longer posts are automatically summarized.
