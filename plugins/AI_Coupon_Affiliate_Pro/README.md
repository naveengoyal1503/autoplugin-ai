# AI Coupon Affiliate Pro

## Features

- **AI-Powered Coupon Generation**: Automatically creates personalized coupons based on user input or niche (OpenAI integration in Pro).
- **Affiliate Link Tracking**: Monitors clicks and conversions with unique tracking IDs for monetization analytics[1][2].
- **Shortcode Integration**: Easy `[ai_coupon_generator niche="amazon"]` embedding in posts/pages.
- **Admin Dashboard**: Configure API keys, affiliate links (JSON), and settings.
- **Conversion Boost**: Displays dynamic deals to increase affiliate commissions and site engagement[1][3].
- **Freemium Ready**: Basic free version; Pro unlocks unlimited coupons, analytics, and custom branding.

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Use shortcode `[ai_coupon_generator]` in any post/page.
4. Configure settings in **Settings > AI Coupon Pro**.

## Setup

1. **Add Affiliate Links**: In settings, enter JSON like `{"amazon":"https://amzn.to/abc","shopify":"https://shopify.com/xyz"}`[2].
2. **Pro API Key**: Input OpenAI key for advanced AI generation (Pro feature).
3. **Embed Shortcode**: Place in sidebar, posts, or dedicated deals page for SEO traffic[1].

## Usage

- Visitors enter a product/niche (e.g., "shoes"), click **Generate Deal**.
- Plugin outputs coupon code + tracked affiliate link.
- Track performance via transients/logs (Pro: dashboard charts).
- **Monetize**: Earn commissions from clicks/sales; upsell Pro for more features.
