# AI Coupon Affiliate Pro

## Features

- **AI-Powered Coupon Generation** (Pro): Automatically creates personalized, niche-specific coupons using OpenAI integration.
- **Affiliate Tracking**: Seamlessly appends your affiliate IDs to all coupon links.
- **Shortcode Support**: Easy `[ai_coupon niche="tech" amount="3"]` integration.
- **Responsive Design**: Mobile-friendly coupon widget.
- **Admin Dashboard**: Manage API keys and affiliate IDs.
- **Freemium Model**: Free demo coupons; Pro unlocks unlimited AI features.

## Installation

1. Upload the plugin ZIP to your WordPress admin → Plugins → Add New → Upload Plugin.
2. Activate the plugin.
3. Use shortcode `[ai_coupon]` on any page/post.

## Setup

1. Go to **Settings → AI Coupon Pro**.
2. Enter your affiliate IDs (e.g., `amazon123,shopify456`).
3. **Pro Only**: Add OpenAI API key for real-time generation.
4. Save settings.

## Usage

- **Basic**: Add `[ai_coupon niche="fashion" amount="5"]` to display coupons.
- **Customization**: Style via CSS classes like `.ai-coupon-pro`.
- **Monetization**: Earn commissions via tracked affiliate links. Upgrade to Pro for AI automation.
