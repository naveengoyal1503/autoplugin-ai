- **Shortcode Support**: Easy embedding with `[ai_coupon niche="fashion" count="5"]`.
- **Responsive Design**: Mobile-friendly coupon cards that boost click-through rates.
- **Pro Features**: Unlimited coupons, OpenAI integration, click analytics, custom branding ($49/year).
- **Cache Optimized**: Transients for fast loading and performance.

## Installation

1. Upload the plugin files to `/wp-content/plugins/ai-coupon-affiliate-pro`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > AI Coupon Pro** to configure.

## Setup

1. **Free Version**: Use shortcodes immediately with sample coupons.
2. **Pro Setup**:
   - Enter your OpenAI API key.
   - Add affiliate IDs in JSON format (e.g., `{"amazon": "your-id"}`).
   - Upgrade to Pro for AI generation.
3. Add shortcode to any post/page: `[ai_coupon niche="tech" count="3"]`.

## Usage

- **Basic**: Paste shortcode in Gutenberg, Classic Editor, or widgets.
- **Customization**: Edit niche parameter for targeted coupons (e.g., "travel", "wellness").[2]
- **Monetization**: Earn commissions on clicks. Track performance in Pro dashboard.
- **Best Practices**: Place on sidebar, dedicated deals page, or post footers for max conversions.[1][4]
