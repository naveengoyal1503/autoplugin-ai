# AI Coupon Affiliate Pro

## Features

- **AI-Powered Coupon Generation**: Automatically creates unique, personalized coupon codes for your audience.
- **Affiliate Tracking**: Seamlessly integrates affiliate links to track commissions from coupon redemptions.[1][2]
- **Shortcode Support**: Easy `[ai_coupon_box store="Amazon" discount="20%"]` embedding in posts/pages.
- **Customizable Dashboard**: Manage affiliate IDs, stores, and settings from WordPress admin.
- **Pro Upgrade**: Unlimited coupons, analytics, email capture, and premium store integrations ($49/year).
- **Mobile-Responsive**: Clean, conversion-optimized design boosts click-through rates.[3]

## Installation

1. Download and upload the plugin ZIP to your WordPress admin > Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Go to **Settings > AI Coupon Pro** to configure your affiliate ID and stores.
4. Add `[ai_coupon_box]` shortcode to any post or page.

## Setup

1. **Configure Settings**: Enter your affiliate ID (e.g., Amazon Associates) in the plugin settings page.
2. **Add Stores**: List default stores like Amazon, Shopify in the textarea.
3. **Embed Shortcode**: Use `[ai_coupon_box store="Walmart" discount="15%"]` for custom deals.
4. **Pro Features**: Upgrade for AI-generated codes based on user behavior and conversion analytics.

## Usage

- **On Frontend**: Visitors click "Generate Coupon" to reveal unique codes linked to your affiliate program.
- **Monetization**: Earn commissions on every redemption. Ideal for blogs in shopping, tech, or deals niches.[1][4]
- **Examples**:
  - `[ai_coupon_box store="Amazon" discount="20%"]`
  - Place in sidebars, posts, or dedicated coupon pages for maximum exposure.
