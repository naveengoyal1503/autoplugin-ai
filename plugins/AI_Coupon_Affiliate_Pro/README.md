# AI Coupon Affiliate Pro

## Features

- **AI-Powered Coupon Generation**: Automatically creates personalized, niche-specific coupons using AI (OpenAI integration in Pro).
- **Affiliate Link Tracking**: Seamlessly appends your affiliate IDs to coupon links for commissions.[1][2]
- **Shortcode Integration**: Easy `[ai_coupon_box niche="tech" count="3"]` for any page/post.
- **Free Tier Limits**: 5 coupons per page; unlimited in Pro.
- **Customizable Designs**: Responsive, mobile-friendly coupon boxes.
- **Analytics Ready**: Pro tracks clicks and conversions.
- **SEO Optimized**: Generates schema-friendly deal markup.

**Pro Features ($49/year)**: Unlimited coupons, real AI generation, custom templates, export data, priority support.

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Go to **Settings > AI Coupon Pro** to configure API key and affiliate IDs.
4. Add shortcode `[ai_coupon_box]` to any post/page.

## Setup

1. **Free Setup**: No API needed; uses demo coupons.
2. **Pro Setup**:
   - Get OpenAI API key from openai.com.
   - Enter affiliate IDs (e.g., Amazon, CJ) in settings.
   - Optional: Add niches like 'fashion', 'tech'.
3. Test with `[ai_coupon_box niche="general" count="2"]`.

## Usage

- **Basic**: Paste shortcode in Gutenberg/Classic editor.
- **Advanced**: Customize via `niche` (e.g., 'travel') and `count` attributes.
- **Monetization Tips**: Place on high-traffic pages, blogs, or sidebars for passive affiliate income. Track performance in Pro dashboard.[1][3][4]
- **Upgrade**: Visit example.com/pro for premium license.
