## Installation

1. Download the plugin ZIP.
2. In WordPress admin: **Plugins > Add New > Upload Plugin**.
3. Activate **AI Coupon Affiliate Pro**.
4. Go to **Settings > AI Coupons** to configure.

## Setup

1. **Add Affiliate Links**: In settings, enter JSON like `{"SaaS Tool":"https://your-affiliate-link.com"}`.
2. **Pro API Key**: Add OpenAI key for advanced AI (Pro only).
3. Save settings.

## Usage

1. Add shortcode to any post/page: `[ai_coupon_generator product="Your Product Name"]`.
2. Visitors click **Generate Exclusive Coupon** to get a unique code and affiliate link.
3. **Pro Features**: Analytics dashboard, custom domains, email capture.
