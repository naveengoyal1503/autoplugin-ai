# AI Coupon Affiliate Pro

## Features

- **AI-Powered Coupon Generation**: Automatically creates unique, personalized coupons for any niche using mock AI (real OpenAI integration in Pro).
- **Affiliate Link Integration**: Embeds your affiliate IDs for Amazon, etc., tracking clicks and boosting commissions.[1][2]
- **Shortcode Ready**: Use `[ai_coupon_generator niche="tech" affiliate="amazon"]` anywhere.
- **Conversion Optimized**: Eye-catching designs with expiry dates and grab buttons.
- **Freemium Model**: Free core, Pro unlocks real AI, analytics, unlimited coupons ($49/year).
- **Admin Dashboard**: Easy setup for API keys and affiliate IDs.

**Pro Exclusive**: Real-time AI via OpenAI, click analytics, custom branding, priority support.

## Installation

1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via **Plugins > Installed Plugins**.
3. Configure in **Settings > AI Coupons** (add affiliate IDs).
4. Add shortcode to any post/page: `[ai_coupon_generator]`. Upgrade to Pro for full AI.

## Setup

1. Go to **Settings > AI Coupons**.
2. Enter your affiliate IDs as JSON (e.g., `{"amazon": "your-id"}`).
3. (Pro) Add OpenAI API key for real generation.
4. Click **Activate Pro** (demo) or purchase at example.com/pricing.

## Usage

- **Basic**: Insert shortcode `[ai_coupon_generator niche="fashion"]`.
- **Customize**: `niche="tech" affiliate="amazon"`.
- **Monetize**: Earn commissions on clicks; track in Pro dashboard.
- **Examples**:
  - Blog post: "Best Deals" with auto-coupons.[1]
  - Sidebar widget for ongoing revenue.[3]
