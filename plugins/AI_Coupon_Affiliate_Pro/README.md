## Installation

1. Download and upload the plugin ZIP to your WordPress admin > Plugins > Add New > Upload Plugin.
2. Activate the plugin.
3. Go to **Settings > AI Coupons** to configure affiliate links and (Pro) API key.

## Setup

1. **Admin Settings**: Enter affiliate links in JSON format (e.g., `{"amazon":"https://amzn.to/xxx"}`).
2. **Add OpenAI Key (Pro)**: Get from openai.com for AI coupon titles/descriptions.
3. **Embed on Site**: Use shortcode in posts/pages: `[ai_coupon_generator niche="tech"]`. Supports `niche` and `count` attributes.
4. **Pro Upgrade**: Visit example.com/pro for full AI features.
