# AI Coupon Affiliate Pro

## Features

- **AI-Powered Coupon Generation**: Automatically creates personalized coupons using OpenAI (Pro feature)[1][2].
- **Affiliate Link Integration**: Easily add affiliate links to coupons for commissions[3][5].
- **Customizable Shortcodes**: Use `[ai_coupon_section niche="fashion"]` for dynamic sections[1].
- **Responsive Design**: Mobile-friendly coupon grids.
- **Admin Dashboard**: Manage API keys and affiliate JSON.
- **Freemium Upsell**: Built-in pro prompts for upgrades.
- **SEO Optimized**: Schema-ready for better search visibility[2].

**Pro Features ($49/year)**: Unlimited coupons, real AI integration, analytics dashboard, premium templates, email capture.

## Installation

1. Upload the `ai-coupon-affiliate-pro` folder to `/wp-content/plugins/`.
2. Activate the plugin via **Plugins > Installed Plugins**.
3. Go to **Settings > AI Coupon Pro** to configure affiliates and API key (Pro).

## Setup

1. In settings, add affiliate links as JSON: `{"amazon":"https://amzn.to/yourlink", "shopify":"https://shop.link"}`.
2. For Pro: Enter your OpenAI API key.
3. Add shortcode to any post/page: `[ai_coupon_section niche="tech"]`. 

## Usage

- **Basic**: Drop shortcode for instant coupon sections with your affiliates.
- **Advanced**: Use multiple shortcodes on deal pages for high conversion[1][2].
- **Monetization Tips**: Target niches like fashion, tech; promote via SEO and social[3]. Track clicks with pro analytics.
- **Customization**: Edit CSS in shortcode output or child theme.
