# AI Coupon Affiliate Pro

## Features

- **AI-Powered Coupon Generation**: Automatically creates unique, personalized coupon codes using AI (Pro feature).[1][2]
- **Affiliate Link Integration**: Seamlessly embeds trackable affiliate links with custom discounts.[3]
- **Shortcode Support**: Easy `[ai_coupon_generator merchant="Amazon"]` for any page or post.[1]
- **Admin Dashboard**: Manage API keys, affiliates, and settings.
- **Pro Features**: Unlimited coupons, analytics, custom branding, real-time AI via OpenAI ($49/year).
- **SEO-Friendly**: Optimized for search engines with schema markup.[2]
- **Mobile-Responsive**: Beautiful card designs for all devices.

## Installation

1. Upload the `ai-coupon-affiliate-pro` folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > AI Coupon Pro** to configure.

## Setup

1. **Free Version**: Add affiliate links in Settings (format: `Merchant|Affiliate URL|Discount %`).
2. **Pro Upgrade**: Enter OpenAI API key and license for AI generation.
3. Use shortcode: `[ai_coupon_generator merchant="YourBrand"]` or `[ai_coupon_generator category="tech"]`.
4. Customize via CSS classes.
