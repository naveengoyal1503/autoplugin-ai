# AI Coupon Affiliate Pro

**AI-powered plugin to generate unique coupons and affiliate links directly in your WordPress posts.**

Boost your affiliate revenue by creating personalized, trackable coupon deals that drive clicks and conversions.[1][2]

## Features

- **One-click coupon shortcodes**: `[ai_coupon store="amazon" discount="20%"]` generates unique codes with affiliate links.[1]
- **Admin dashboard**: Manage affiliate links (JSON format) and pro license.[2]
- **Pro version** ($49/year): AI coupon generation, click analytics, unlimited stores, premium integrations.[3][4]
- **Mobile-responsive design**: Eye-catching coupon boxes encourage clicks.
- **Shortcode customization**: Set discount, expiry, store.
- **GDPR compliant**: No external trackers in free version.

## Installation

1. Upload the plugin ZIP to **Plugins > Add New > Upload**.
2. Activate the plugin.
3. Go to **Settings > AI Coupon Pro** to add your affiliate links (e.g., `{"amazon":"https://amazon.com/?tag=yourtag"}`).
4. Use shortcode `[ai_coupon]` in posts/pages.

## Setup

1. **Configure links**: In settings, enter JSON of store:affiliate_url pairs.
2. **Optional Pro upgrade**: Enter license key for advanced features.
3. **Embed shortcodes**: Add to Gutenberg blocks, Classic Editor, or widgets.

Example:

[ai_coupon store="amazon" discount="50%" expires="2026-02-01"]


## Usage

- **In posts**: Insert shortcode for instant coupon boxes with unique codes.
- **Customization**: Style via CSS or pro themes.
- **Tracking (Pro)**: View clicks per store in dashboard.
- **Monetization tips**: Target niches like software, travel, e-commerce. Pair with SEO plugins for traffic.[1][3]
