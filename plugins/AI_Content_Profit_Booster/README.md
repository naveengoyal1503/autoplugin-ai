# AI Content Profit Booster

## Features
- **AI-Powered Content Generation**: Uses OpenAI to create SEO-optimized blog posts with embedded affiliate links and custom coupon codes[1][2].
- **Affiliate Integration**: Easily add your affiliate links via JSON in settings for automatic insertion.
- **One-Click Generation**: Enter a topic and generate profitable content instantly.
- **Freemium Model Ready**: Pro version (not included) adds unlimited generations and analytics.
- **SEO-Friendly**: Outputs include headings, lists, and CTAs for better rankings.

## Installation
1. Download and upload the plugin ZIP to `/wp-content/plugins/`.
2. Activate via WordPress Admin > Plugins.
3. Go to Settings > Profit Booster to configure.

## Setup
1. **Get OpenAI API Key**: Sign up at openai.com/api and paste your key.
2. **Add Affiliate Links**: Edit JSON array, e.g., `[{ "keyword": "hosting", "link": "https://aff.link" }]`. Include coupon ideas like 'SITE10OFF'.[1][3]
3. Save settings.

## Usage
1. Navigate to Settings > Profit Booster.
2. Enter topic (e.g., "Best WordPress Plugins 2026").
3. Click "Generate Profitable Content".
4. Copy the output and paste into a new post/page.
5. Publish to start earning commissions![2][4]

**Pro Tip**: Target niches like hosting, tools, or e-commerce for high conversions[1][3].
